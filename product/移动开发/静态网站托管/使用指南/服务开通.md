本文为您介绍如何开通静态网站托管以及部署应用等操作。




## 环境开通

1. 登录 [云开发控制台](https://console.cloud.tencent.com/tcb)，单击【新建】。
2. 选择 [应用模板](https://www.cloudbase.net/marketplace.html) 或连接 Github/Gitlab/Gitee 公开仓库，进行应用部署准备，单击【下一步】。
3. 选择环境所属地域，选择【按量计费】计费方式，填写环境名称，单击【下一步】。
4. （可选）如果选择非空白模板，按照提示填写应用配置。
5. 确认订单后开通，我们会在1 - 5分钟左右为您发货。
6. 环境开通成功后，进入环境，使用 [静态网站托管](https://console.cloud.tencent.com/tcb/hosting/index) 服务。


静态网站托管支持一键部署环境和通过导入仓库文件的方式创建环境。




<dx-tabs>
::: 一键部署
云开发提供各类热门应用，支持将项目一键部署至云开发中，并支持二次开发。

1. 登录 [云开发控制台](https://console.cloud.tencent.com/tcb)，单击【新建】。
2. 应用来源选择【模板仓库】，并选择 [应用模板](https://www.cloudbase.net/marketplace.html) 进行应用部署准备，单击【下一步】。
![](https://main.qcloudimg.com/raw/ff7c743f6dc222b440489171a6e644c3.jpg)
3. 选择环境所属地域，选择【按量计费】计费方式，填写环境名称后单击【下一步】。
4. 按照提示单击【立即开通】完成应用配置。
5. 进入 [我的应用](https://console.cloud.tencent.com/tcb/apps/index)，查看并管理部署完成的应用。
:::
::: 仓库文件导入
云开发支持将 Github/Gitlab/Gitee 仓库中的项目导入至云开发中，便于您将项目部署上云。

1. 登录 [云开发控制台](https://console.cloud.tencent.com/tcb)，单击【新建】。
2. 应用来源选择【[代码导入](https://www.cloudbase.net/marketplace.html)】，填写代码来源地址及部署分支，单击【下一步】。
![](https://main.qcloudimg.com/raw/fa73a847f17a3f0b6b432dbbb28be861.jpg)
3. 选择环境所属地域，选择【按量计费】计费方式，填写环境名称；单击【下一步】。
4. 按照提示单击【立即开通】完成应用配置。
5. 进入 [我的应用](https://console.cloud.tencent.com/tcb/apps/index)，查看并管理部署完成的应用。
:::
