1. 中小企业客户登录 [腾讯云官网](https://cloud.tencent.com/login?s_url=https%3A%2F%2Fcloud.tencent.com%2F)，进入 [融资产品选择](http://finance.cloud.tencent.com/ttr/web/page/product-recommend.html) 页面。
2. 在融资产品选择页面中，中小企业客户可根据自身实际情况，在对应的融资产品中，单击【立即申请】。
3. 在弹框中，完成手机号及动态验证码的验证，即可进入问卷调查页面。
4. 在问卷中，完成融资需求的填写后，单击【确定提交】。
5. 企业金融服务平台根据产品智能匹配规则，帮助中小企业匹配合适的银行贷款产品并进行展现，中小企业客户可选择具体产品进行申请操作。

## 对接模式
企业金融服务平台提供 H5 嵌入和 API 对接两种模式供商业银行选择：
### H5 对接模式
中小企业客户在匹配到银行产品后，可通过腾讯云入口直接跳转到银行 H5 页面完成后续的申请流程及操作。
### API 对接模式
中小企业客户在企业金融服务平台填写融资需求后，企业金融服务平台通过 API 方式传输给合作银行，由合作银行负责后续审批及服务。

>? 为保障客户的二次登录体验，两种模式均需要向腾讯云回传审批/逾期详细信息。
