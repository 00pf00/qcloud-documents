| DescribeInstanceInternetBandwidthConfigs | 查询实例带宽配置 |
| ModifyInstancesAttribute                 | 修改实例属性     |
| ModifyInstancesProject                   | 修改实例所属项目 |
| ModifyInstancesRenewFlag                 | 修改实例续费标识 |
| RebootInstances                          | 重启实例         |
| RenewInstances                           | 续费实例         |
| ResetInstancesPassword                   | 重置实例密码     |
| ResetInstancesType                       | 调整实例配置     |
| ResizeInstanceDisks                      | 扩容实例磁盘     |
| RunInstances                             | 创建实例         |
| StartInstances                           | 启动实例         |
| TerminateInstances                       | 退还实例         |
| CreateKeyPair                            | 创建密钥对       |
| DeleteKeyPairs                           | 删除密钥对       |
| DescribeKeyPairs                         | 查询密钥列表     |
| AssociasteInstancesKeyPairs              | 绑定密钥对       |
| CreateImage                              | 创建自定义镜像   |
| DeleteImages                             | 删除镜像         |
| DescribeImages                           | 查看镜像列表     |
| DescribeImagesAttribute                  | 查看镜像属性     |
| ModifyImageSharePermission               | 修改镜像分享信息 |
| SyncImages                               | 同步镜像         |

### <span id="cloudaudit">云审计</span>

| API 接口              | API 描述                           |
| --------------------- | ---------------------------------- |
| ListAudits            | 拉取 CloudAudit 列表               |
| CreateAudit           | 创建 CloudAudit                    |
| LookupEvents          | 检索日志                           |
| DescribeAudits        | 根据 AuditName 获取 Audit 详细信息 |
| DeleteAudit           | 删除 Audit                         |
| StartLogging          | 开启数据采集                       |
| StopLogging           | 停止数据采集                       |
| UpdateAudit           | 更新 Audit 信息                    |
| GetAuditServiceStatus | 查询是否开通 Audit 服务            |
| OpenAuditService      | 开通 Audit 服务                    |
