本文将为您介绍如何操作 JVM 。

## 操作步骤

1. 登录 [应用性能监控-应用](https://console.cloud.tencent.com/monitor/tapm/application/list)，进入应用列表页。
2. 单击应用名称，进入应用详情页。
3. 单击【JVM】。
当选择了具体的实例后，**JVM** 页签会显示在上方。该页面针对 Java 应用提供 JVM 运行环境的各类统计信息和图表，包括：内存曲线、线程曲线、HTTP 会话曲线。用户可通过单击上方的各个页签进行查看。
