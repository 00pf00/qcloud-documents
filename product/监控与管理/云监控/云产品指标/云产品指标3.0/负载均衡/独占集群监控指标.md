## 命名空间

Namespace=QCE/TGW_SET

## 监控指标

| 指标英文名      | 指标中文名         | 单位  | 统计周期 |
| --------------- | ------------------ | ----- | -------- |
| ConcurConn      | 全量连接数         | 个    | 60s、300s  |
| ConcurConnRatio | 并发连接数利用率   | %     | 60s、300s  |
| ConNum          | 活跃连接数         | 个    | 60s、300s  |
| DropTotalConns  | 丢弃连接数         | 个/秒 | 60s、300s |
| InactiveConn    | 非活跃连接数       | 个    | 60s、300s  |
| InPkg           | 公网入包量         | 个/秒 | 60s、300s  |
| InpkgRatio      | 公网入包量利用率   | %     | 60s、300s  |
| InTraffic       | 公网入带宽         | Mbps  | 60s、300s  |
| IntrafficRatio  | 公网入带宽利用率   | %     | 60s、300s  |
| InDropBits      | 丢弃入流量         | Mbps  | 60s、300s  |
| InDropPkts      | 丢弃流入数据包     | 个/秒 | 60s、300s |
| NewActiveConn   | 新建活跃连接数     | 个/秒 | 60s、300s  |
| NewConn         | 新增连接数         | 个/秒 | 60s、300s  |
| NewConnRatio    | 新增连接利用率     | %     | 60s、300s |
| OutPkg          | 公网出包量         | 个/秒 | 60s、300s  |
| OutpkgRatio     | 公网出包量利用率   | %     | 60s、300s |
| OutTraffic      | 公网出带宽         | Mbps  | 60s、300s  |
| OuttrafficRatio | 出带宽利用率       | %     | 60s、300s |
| OutDropBits     | 公网丢弃出流量     | Mbps  | 60s、300s |
| OutDropPkts     | 公网丢弃流出数据包 | 个/秒 | 60s、300s  |

> ?每个指标对应的统计粒度（Period）可取值不一定相同，可通过 [DescribeBaseMetrics](https://cloud.tencent.com/document/product/248/30351) 接口获取每个指标支持的统计粒度。

## 各维度对应参数总览

| 参数名称                       | 维度名称  | 维度解释                   | 格式                                |
| ------------------------------ | --------- | -------------------------- | ----------------------------------- |
| Instances.N.Dimensions.0.name  | clusterId | 负载均衡集群 ID 的维度名称 | 输入 String 类型维度名称：clusterId |
| Instances.N.Dimensions.0.value | clusterId | 负载均衡具体集群 ID        | 输入具体集群 ID，例如：tgw-12345678 |

## 入参说明

**负载均衡-独占集群入参维度如下：**

&Namespace=QCE/TGW_SET
&Instances.N.Dimensions.0.Name=clusterId
&Instances.N.Dimensions.0.Value=集群 ID
