## 操作场景
当您不再需要某个协作者时，可以删除该协助者。删除协作者后，协作者将不再拥有该主账号下资源的管理权限。


## 操作步骤
>?
>- 如果协作者下有 API 密钥，在删除用户前，您需要禁用且完全删除完所属 API 密钥。
>- API 密钥删除后无法恢复，请您确认清楚再进行删除。用户被删除后，该用户无法登录腾讯云以及接收消息通知，同时会解除关联权限。


### 删除单个协作者
1. 使用主账号或具有 **QcloudCamFullAccess 策略** 权限的子账号登录 [访问管理控制台](https://console.cloud.tencent.com/cam)。
2. 在【用户列表】页面，找到需要删除的协作者，单击操作列的【更多操作】>【删除】。
![](https://main.qcloudimg.com/raw/cca4fde3b58c075d9070e2ae961adaef.png)
3. 在删除用户的窗口中，根据账号下是否有 API 密钥的页面提示，执行删除操作。[](id:删除)
<dx-tabs>
::: 账号下无\sAPI\s密钥
单击【删除】，即可删除协作者。
![](https://main.qcloudimg.com/raw/7ddce3b6dcbb0ed3a87cf5f646a2f031.png)
:::
::: 账号下有\sAPI\s密钥
3.1. 在删除用户的窗口中，单击操作列的【前往API密钥】。
![](https://main.qcloudimg.com/raw/f304503f01587c664541b951c4d8e128.png)
3.2. 在 API 密钥模块，单击操作列的【禁用】，并根据页面提示完成禁用操作。
![](https://main.qcloudimg.com/raw/b7e7bf0f34c557231148467173274725.png)
3.3. 在 API 密钥模块，单击操作列的【删除】，删除协作者的密钥。
>?有多个 API 密钥时，重复执行步骤3.2和3.3，禁用且完全删除完所属 API 密钥。
>
3.4. 返回3.1中删除用户的窗口，单击【刷新】，刷新后单击【删除】即可删除协作者。
![](https://main.qcloudimg.com/raw/283c86d3f6ede326c3356f7d52066ab7.png)
:::
</dx-tabs>


### 删除多个协作者
1. 使用主账号或具有 **QcloudCamFullAccess 策略** 权限的子账号登录 [访问管理控制台](https://console.cloud.tencent.com/cam)。
2. 在【用户列表】页面，勾选需要删除的协作者，单击用户列表上方的【更多操作】>【删除】。
![](https://main.qcloudimg.com/raw/6cc393069368cbaf2057d3d838c41df7.png)
3. 在删除用户的窗口中，根据账号下是否有 API 密钥的页面提示，执行删除操作。
操作步骤同 [删除单个协作者](#删除)。
