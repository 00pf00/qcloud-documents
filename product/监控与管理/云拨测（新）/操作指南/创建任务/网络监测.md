本文将为您介绍如何新建网络监测任务场景的拨测任务。

## 操作场景

“网络监测”专注于基础网络质量，您只需要输入您的网页地址，分布全国的拨测点将自动对您的地址发起探测，并记录拨测情况。根据高频的探测，获取基础网络时延、丢包、路由等状态，获取网络数据，支持实时告警，并提供详细数据指标供您了解网络情况。 网络监测帮助您快速了解基础网络的运行情况，快速发现并定位问题。

> ?网络监测任务场景没有免费体验额度，网络监测按拨测点进行收费，详情请参考 [购买指南](#)。

## 操作步骤

1. 登录 [云拨测控制台](https://console.cloud.tencent.com/catpro)。
2. 单击左侧菜单栏中的【场景化监控】，进入场景化监控页面。
3. 单击【新建任务】，进入新建任务页，选择“网络监测”任务场景。其它配置请参考下表。

- **基本信息**：

| 配置项   | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 地区     | 根据网站所在区域选择，不同的地区收费不同，详情请参考 [价格总览](#) |
| 任务名称 | 填写拨测任务名称，建议以监测对象来命名                       |
| 拨测地址 | 请填写以 http:// 开头的拨测地址                              |

- **拨测点配置**：

| 配置项     | 说明                                                         |
| ---------- | ------------------------------------------------------------ |
| 拨测点类型 | 不同的拨测点类型收费规则不同，详情请参考 [价格总览](#) 骨干IDC：部署在骨干线路中IDC机房的拨测点，代表骨干线路终端网民 LastMail：部署在终端用户PC电脑上的拨测点，代表终端PC用户体验手机端：部署在终端移动手机上的拨测点，代表终端移动用户体验 |
| 拨测点选择 | 您可以选择系统推荐的节点组，也可以自定义节点组。网络监测按拨测点收费，拨测点越多收费越高，详情请参考 [价格总览](#)。您可以根据需求勾选拨测点组，勾选后单击【![img](https://main.qcloudimg.com/raw/bfc5ea3194db6c218feb1c1011bb2584.png)】按钮即可。如需取消拨测点选择请点击【![img](https://main.qcloudimg.com/raw/bfc5ea3194db6c218feb1c1011bb2584.png)】即可。 |

- **拨测参数**：

| 配置项                | 说明                                                         |
| --------------------- | ------------------------------------------------------------ |
| IP类型                | 支持自动、IPV4、IPV6三种类型                                 |
| 拨测配置—DNS监测      | 监测超时：定义监测超时时间，取值范围为0-45查询方式：支持递归和迭代两种查询方式。指定 NS 服务器：指定用于DNS解析的服务器，填写您的服务器域名dig命令：是否开启dig命令格式的监测结果DNS服务器类型：支持自动、IPV4、IPV6三种类型 |
| 拨测配置—Tracert 监测 | 监测超时：定义监测超时时间，取值范围为：0-300最大跃点数（个）：填写跃点数，一个路由位一个跃点数 |
| 拨测配置—Ping 监测    | 协议类型：支持ICPM、TCP两种协议类型监测超时：定义监测超时时间，取值范围为：0-60执行间隔：定义Ping监测任务执行时间间隔Package数量（个）：填写软件包数量Package大小（KB）：填写软件包大小切分ackage：是否切分软件包 |

- **拨测周期配置**：

| 配置项   | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 拨测时长 | 选择您需要的拨测时长，拨测时长越长收费越高，详情请参考 [价格总览](#) |
| 拨测频率 | 支持5分钟、10分钟、20分钟、30分钟、60分钟、120分钟的拨测频率 |
