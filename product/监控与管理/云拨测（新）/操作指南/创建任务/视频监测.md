本文将为您介绍如何新建视频体验任务场景的拨测任务。

## 操作场景

“视频任务”专注于视频流网页的用户体验 ，您只需要输入您的视频地址，分布全国的拨测点将自动对您的视频地址发起探测，并记录拨测情况。根据高频的探测，获取用户在观看视频过程中所出现的卡顿、缓存、延迟等状态，并生成视频体验数据。云拨测还支持实时告警，并提供详细指标监控数据供您了解视频情况。视频体验监控帮助您快速了解视频的播放运行情况，快速发现并定位问题。

> ?视频体验有免费体验版，仅限体验一次，如需购买请参考 [购买指南](#)。

## 操作步骤

1. 登录 [云拨测控制台](https://console.cloud.tencent.com/catpro)。
2. 单击左侧菜单栏中的【场景化监控】，进入场景化监控页面。
3. 单击【新建任务】，进入新建任务页，选择“视频体验”任务场景。其它配置请参考下表。

- **基本信息**：

| 配置项   | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 功能     | 免费体验：拨测点不支持选择，仅限体验一次。高级定制：高级定制版需要收费。详情请参考 [价格总览](#) |
| 地区     | 根据网站所在区域选择，不同的地区收费不同，详情请参考 [价格总览](#) |
| 任务名称 | 填写拨测任务名称，建议以监测对象来命名                       |
| 拨测地址 | 填写您需要拨测的网站URL，http类型地址填写以 http:// 开头的URL；MP4 类型 RTMP 流地址，须标明 mp4:xxx，如rtmp://host/server/mp4:res |

- **拨测点选择**：

| 配置项     | 说明                                                         |
| ---------- | ------------------------------------------------------------ |
| 拨测点类型 | 不同的拨测点类型收费规则不同，详情请参考 [价格总览](#) 骨干IDC：部署在骨干线路中IDC机房的拨测点，代表骨干线路终端网民 LastMail：部署在终端用户PC电脑上的拨测点，代表终端PC用户体验手机端：部署在终端移动手机上的拨测点，代表终端移动用户体验 |
| 拨测点选择 | 您可以选择系统推荐的节点组，也可以自定义节点组。网络监测按拨测点收费，拨测点越多收费越高，详情请参考 [价格总览](#)。您可以根据需求勾选拨测点组，勾选后单击【![img](https://main.qcloudimg.com/raw/bfc5ea3194db6c218feb1c1011bb2584.png)】按钮即可。如需取消拨测点选择请点击【![img](https://main.qcloudimg.com/raw/bfc5ea3194db6c218feb1c1011bb2584.png)】即可。 |

- **拨测参数**：

| 配置项         | 说明                                                         |
| -------------- | ------------------------------------------------------------ |
| IP类型         | 支持自动、IPV4、IPV6三种类型                                 |
| 音视频标示     | 支持视频和音频两种类型                                       |
| 拨测时长（秒） | 定义每次拨测的时长                                           |
| 地址类型       | 选择视频地址类型，资源地址和页面地址                         |
| 自定义 Host    | 支持按 IP 地址轮询或随机监测，多个IP请用半角逗号分隔符，如：IPv4：192.168.2.1,192.168.2.5:img.a.com\|192.168.2.1󠉀]:img.a.com\|ipv6:[0:0:0:0:0:0:0:1][8080],[0:0:0:0:0:0:0:2][8081]:[www.a.com](http://www.a.com)\|] |

- **拨测周期配置**：

| 配置项   | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 拨测时长 | 支持7天和30天拨测时长，拨测时长不同收费规则不同，详情请参考 [价格总览](#) |
| 拨测频率 | 默认只支持30分钟的拨测频率                                   |
| 分析报告 | 支持性能分析报告、专项性能优化报告、竞品对比分析报告，购买后需填写邮箱信息，系统会定期给您的邮箱发送报告。不同报告类型收费不一致，详情请参考 [价格总览](#) |
