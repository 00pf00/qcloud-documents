本文为您详细介绍任务的管理与使用。

## 登录
1. 登录 CODING，单击页面右上角的 <img src ="https://main.qcloudimg.com/raw/7531b01c25014beb2754277107fc4ab1.png" style ="margin:0">，进入项目列表页面，单击项目图标进入目标项目。
2. 单击左侧菜单栏中的【任务管理】，进入任务管理页面。





## 任务处理
任务处理包含了对具体任务的创建、查看、更新、关联需求、删除、评论、记录日志等操作。

### 创建任务
1. 在任务管理页面，单击【创建任务】。
2. 填写任务标题、任务描述，按需设置任务各属性。
3. 单击【创建】按钮即可创建成功。

>?
- 任务可以设置优先级、截止日期等基本信息，并指定处理人员处理。
- 任务必须处于“未开始”、“进行中”和“已完成”其中一个阶段。

### 查看任务列表
任务列表是该项目的所有任务的列表视图。用户可以迅速对任务进行筛选和操作（尤其当存在大量任务时）。通过搜索与筛选配合使用，可快速筛选出用户想要查找的任务。了解更多筛选器的使用方法请参见 [使用筛选器](https://cloud.tencent.com/document/product/1113/36837) 。

### 查看任务详情
任务详情页展示了任务具体内容和属性。
1. 在任务管理页面，单击目标任务标题或任务 ID 即可查看任务详情页。
2. 右侧将会出现任务详情页，在任务详情页可进行任务属性修改。

### 关联需求
任务是为了实现需求中特性或功能，关联需求是为了能在任务中查看到关联的需求列表，方便明确任务目标和内容。任务可关联多个需求。
1. 在任务详情页，单击 <img src ="https://main.qcloudimg.com/raw/46e500af40b74ec4d3501e7b394583bb.png" style = "margin:0"> ，输入需求名称，选择需求后即可完成关联。
2. 若要取消关联需求或删除关联需求。单击已关联需求的最末端的 <img src ="https://main.qcloudimg.com/raw/f000f0cee5005373b46f1d348ee3053f.png" style = "margin:0"> ，在下拉菜单中即可选择取消关联需求或删除关联需求。

### 删除任务
删除任务有两种途径：
- 在任务详情页右上角点击 <img src ="https://main.qcloudimg.com/raw/f000f0cee5005373b46f1d348ee3053f.png" style = "margin:0">，出现的下拉菜单中单击【删除】。
- 在任务列表页，每个任务前可点击勾选，然后在下方可选择执行批量删除。


## 筛选器管理
列表页支持使用筛选器筛选任务，筛选器是一组搜索条件和排序方式。筛选器分为系统筛选器和自定义筛选器，其中系统筛选器不可修改和删除。

### 设置默认筛选器

在任务列表页左上角下拉菜单中选择所需筛选器，并设置为默认筛选器。
 - 当筛选器设置为默认筛选器后，当前登录成员每次进入列表页则默认使用该筛选器。且默认筛选器不支持删除操作。
 
### 创建自定义筛选器
1. 在任务列表页选择任意筛选器，修改搜索条件和排序方式后筛选器会标记为“已修改”。
	![](https://main.qcloudimg.com/raw/fbd2ae0685080054b1e01be1fded8a4f.png)
2. 单击【另存为】，在弹出窗口中输入筛选器名称。
3. 单击【保存】完成筛选器的创建。
4. 可在筛选器列表中查看到已创建成功的自定义筛选器。且该自定义筛选器的搜索条件和排序方式默认为创建时所填写的选项。

### 修改自定义筛选器
1. 在任务列表页选择任意筛选器，修改搜索条件和排序方式后筛选器会标记为“已修改”。
![](https://main.qcloudimg.com/raw/b4f903c859748ca1805efe62fca8cb04.png)
2. 单击【保存】按钮，即可完成自定义筛选器修改。

### 删除/重命名自定义筛选器
1. 在任务列表页，打开筛选器列表，选择任意一个筛选器单击【...】。 
![](https://main.qcloudimg.com/raw/f24e6657c15c628fd977df80c3a01f61.png)
2. 选择【删除】或【重命名】选项，即可完成删除或重命名操作。
