## 操作场景

本文为您介绍腾讯云主账号如何开通并使用 CODING DevOps 服务，您可以按照以下几种情况有选择性操作。

## 主账号未开通服务

访问 [CODING 控制台](https://console.cloud.tencent.com/coding)

![](https://help-assets.codehub.cn/enterprise/20211102153203.png)

### 未注册过 CODING

若您未注册过 CODING 服务，请点击「没有 CODING 账号，去开通」按钮。

![](https://help-assets.codehub.cn/enterprise/20211102153348.png)

按照提示填写邮箱、验证码、密码进入下一步，输入团队名称和域名后完成团队创建。

### 曾注册过 CODING

若已有 CODING 账号，点击控制台中的「已有 CODING 账号，去关联」按钮。

![](https://help-assets.codehub.cn/enterprise/20211102154758.png)

填写邮箱密码后完成账号关联。

## 主账号已开通服务

点击[控制台](https://console.cloud.tencent.com/coding)中的立即使用或团队域名，跳转至 CODING 团队页。
