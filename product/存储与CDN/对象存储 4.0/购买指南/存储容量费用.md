| 标准存储（多 AZ）容量费用      | 指用户选择标准存储（多 AZ）类型将数据上传到 COS  进行保存所产生的存储容量费用。          <br><br>说明：<br>     按数据的实际存储时间和实际存储量计算。 | 北京、广州             | 按量计费：<br><li>存储容量费用 = 存储容量单价 x 月存储容量<br><li>月存储容量 = 当月“日存储容量”之和 / 当月天数<br><li>日存储容量 = 当日“每5分钟存储容量”之和 / 288（采样点数） |
| 低频存储（多 AZ）容量费用      | 指用户选择低频存储（多 AZ）类型将数据上传到 COS  进行保存所产生的存储容量费用。          <br><br>说明：<br>     1. 对于低频存储（多  AZ）类型的数据，存储时间不足30天，按30天计算；单个存储文件不足64KB，按64KB计算，大于或等于64KB，按实际大小计算。     <br>2. 如果您在未开启版本控制功能的情况下，上传一份同名的低频存储（多 AZ）类型的对象，COS 将会在您上传成功后删除原来的同名对象，此时同样会收取因提前删除对象而产生的存储费用。 | 北京、广州             | 按量计费：<br><li>存储容量费用 = 存储容量单价 x 月存储容量<br><li>月存储容量 = 当月“日存储容量”之和 / 当月天数<br><li>日存储容量 = 当日“每5分钟存储容量”之和 / 288（采样点数） |
