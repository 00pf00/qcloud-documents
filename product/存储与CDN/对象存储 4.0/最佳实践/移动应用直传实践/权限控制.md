下面基于 [搭建临时密钥服务](https://cloud.tencent.com/document/product/436/19134) 一文中提到的通过设置 STS SDK 的 Policy 参数，来实现不同的权限控制。


## 常用的 Policy

### 完全权限

拥有该账号下 COS 资源的所有权限，**不推荐使用**。

```
 {
    "version": "2.0",
    "statement":
     {
         "effect": "allow",
         "action": "cos:*",
         "resource": "*"
     }
}
```

### 拥有所有 COS 资源的读权限

```
{
    "version": "2.0",
    "statement":
     {
         "effect": "allow",
         "action":  [
                    "cos:List*",
                    "cos:Get*",
                    "cos:Head*",
                    "cos:OptionsObject"
                ],
         "resource": "*"
     }
}
```

### 指定前缀的文件的读写权限：

假设您的 appid 为 8000001，资源地域为 ap-shanghai，Bucket 为 Bucket1，文件前缀为 /dir1/test*

```
{
    "version": "2.0",
    "statement":
     {
         "effect": "allow",
         "action": "cos:*",
         "resource": "qcs::cos:ap-shanghai:uid/8000001:prefix//8000001/Bucket1/dir1/test*"
     }
}
```

更多 Policy 设置方式请参考 [权限设置相关案例](https://cloud.tencent.com/document/product/436/12514) 以及 [CAM 服务](https://cloud.tencent.com/document/product/598/10583)。

## 请求具有权限控制的临时密钥 

### Python SDK

```
from sts import Sts

config = {
	# 临时密钥有效时长，单位是秒，如果没有设置，默认是30分钟
	'duration_in_seconds': 1800,
	# 您的secret id
	'secret_id': 'xxx',
	# 您的secret key
	'secret_key': 'xxx',
	
	# 您配置的 policy
	'policy': 'your policy',
}

sts = Sts(config)
response = sts.get_credential()
json_content = response.content
```


### Java SDK

```
TreeMap<String, Object> config = new TreeMap<String, Object>();

// 您的 SecretID
config.put("SecretId", "xxx");
// 您的 SecretKey
config.put("SecretKey", "xxx");
// 临时密钥有效时长，单位是秒，如果没有设置，默认是30分钟
config.put("durationInSeconds", 1800);


// 配置的 policy
config.put("policy", "your policy");

JSONObject credential = StorageSts.getCredential(config);
```

###  NodeJS SDK

```
ar sts = require('tac-storage-sts');

var options = {
	// 您的 secretId
 	secretId: 'xxx',
	// 您的 secretKey
	secretKey: 'xxx',
	// 临时密钥有效时长，单位是秒
	durationInSeconds: 1800,
	
	
	// policy
	policy: 'your policy',
};


sts.getCredential(options, function(data) {
    console.info(data)
});
```
