## 操作场景
以微信开发者工具为例，本文介绍使用小游戏联机对战引擎 MGOBE API 快速接入对战平台的方法，实现一个 MGOBE 的 Hello World。


## 前提条件
使用 SDK 前需要在小游戏联机对战引擎控制台创建小游戏实例并开通联机对战服务，获取游戏 gameId 和 secretKey。SDK 需要对这两个参数进行校验。

## 操作步骤
### 创建游戏
1. 打开微信开发者工具，创建一个小游戏项目（AppID 可以先使用测试号）。
2. 进入编辑界面。由于微信会限制请求域名，开发者可以在开发阶段先关闭域名校验。
