## 操作场景
该文档指导您在使用 WeTest 压测大师产品时，如何使用压测大师 URL 测试的检查点功能判断 URL 返回 body、header 内容是否符合业务预期。

## 前提条件
已创建一个 WeTest [压测大师项目](https://console.qcloud.com/wetest/master)。

## 操作步骤
1. 进入腾讯云 [测试服务 WeTest](https://console.qcloud.com/wetest) 控制台，选择左侧菜单【压测大师】>【用例列表】，单击【[新建 URL 测试](https://console.qcloud.com/wetest/master/testcase)】。
![](https://main.qcloudimg.com/raw/8eb66eba5503f4691f5b401ecad55fc6.png)
2. 进入 URL 用例配置页后，在客户端请求填写区，勾选【添加检查点】，并单击【添加】。
![](https://main.qcloudimg.com/raw/a24d9d3c04e48ed4d9b883c329261544.png)
3. 我们提供多种提取方式，包括键值对、[正则表达式](https://baike.baidu.com/item/%E6%AD%A3%E5%88%99%E8%A1%A8%E8%BE%BE%E5%BC%8F/1700215?fr=aladdin)、Json 等 N8，请根据实际选择。
![](https://main.qcloudimg.com/raw/70251318019feb1c8f46e01d24e7574d.png)
4. 填写示例，某个 URL 返回的结果 body 内容是如下的一个 json。
```
[{
“data”: {
“hello”: “world”
},
“type”: “1234”,
“date”: “2012-10-30 17:06:19”,
“user”: “000000000000000”,
“time_stamp”: 1351587969902
}, {
“data”: {
“hello”: “world”
},
“type”: “1234”,
“date”: “2012-10-30 17:12:53”,
“user”: “000000000000000”,
“time_stamp”: 1351588373519
}]
```

5. 我们如果想检查倾斜的那个字段是否为”world”，我们可以这样配置：
![](https://main.qcloudimg.com/raw/93c9a8f1c6c5ecc5fdc8e35923fe62f5.png)
其中0是 json 的第一个数组元素，通过[0][“data”][“hello”]这样的路径就可以定位到”world”了。

>!
- 中括号以及双引号是英文下的，同时如果检查的字段是字符串的话需要加上双引号，如果是数字就不需要加。
- 另外检查值的长度不要超过256字符。

6. 测试结束之后可以在事务统计这边看到测试过程中该检查点的统计情况。
![](https://main.qcloudimg.com/raw/0d8c2e3072809db2c94efe47d8fc7fdc.png)

>!如果是正则表达式的话直接在检查规则中填入正则表达式即可。
