### 黑石弹性公网 IP 是否可以绑定 CVM？
不可以，黑石弹性公网 IP 只可以绑定黑石物理服务器、托管服务器、NAT 网关以及黑石虚拟 IP 等资源。

### 已释放的黑石弹性公网 IP，再次申请会分配相同的 EIP 地址吗？
无法保证，黑石弹性公网 IP 被释放后进入地址池重新随机分配，有可能会被其他用户申请掉，所以如果只是短暂停用黑石弹性公网 IP，后续还希望继续使用该 EIP，建议不要释放，当前黑石弹性公网 IP 未绑定期间会有0.2元/小时的IP地址闲置费用

### NAT 网关可以绑定多少个 EIP 地址？
最多可以绑定4个。

### 为什么按照公网流量计费也有带宽峰值？
这个带宽峰值不是计费带宽，而是为了避免在用户不知情情况下产生过多的出向流量而设置的限速带宽，特别是在客户机器被黑变成肉鸡向外打流量的时候，可以有效避免用户产生不必要的费用。
