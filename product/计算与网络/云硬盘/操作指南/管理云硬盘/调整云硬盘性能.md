云硬盘性能通常情况下与云硬盘容量相关，您可以在未达到产品性能最大值时，通过调整云硬盘容量以获得更高的性能。特别的，**增强型SSD云硬盘**支持在性能达到**基准性能**的最大值后，通过配置**额外性能**以突破基准性能限制，详细内容请参见 [增强型 SSD 云硬盘性能说明](https://cloud.tencent.com/document/product/362/51896)。

在满足条件时，您可根据需求进行额外性能的配置，并且支持随时调整额外性能。

## 前提条件

- 当前仅**增强型SSD云硬盘**和**极速型SSD云硬盘**支持性能独立调整。
- 必须满足 [基准性能](https://cloud.tencent.com/document/product/362/51896) 已经达到最大值，才可独立调整额外性能。

## 性能升级

### 使用控制台进行性能升级

在满足前提条件时，您可通过以下方式进行性能升级：

1. 登录 [云硬盘控制台](https://console.cloud.tencent.com/cvm/cbs)。
2. 选择地域，选择您需要调整性能的云硬盘。
3. 选择目标云硬盘的【更多】>【调整性能】。
4. 在弹出的“调整性能”窗口，选择您需要调整的目标配置。
5. 勾选说明，开始调整。

### 使用 API 进行性能升级

请参考 [调整云硬盘额外性能](https://cloud.tencent.com/document/product/362/51483) 接口。

您可以使用 ModifyDiskExtraPerformance 接口对指定云盘进行性能升级，具体操作请参考 [调整云硬盘额外性能](https://cloud.tencent.com/document/product/362/51483)。

## 云硬盘性能调整费用说明

### 性能升级

- 对于包年包月云硬盘：性能升级按照生命周期的剩余时间补齐新配置与旧配置的差价。具体计算按照实际情况，您可以在付款页面查看。
- 对于按量计费云硬盘：立即生效，并开始按新配置的价格进行计费。

### 性能降级

- 对于包年包月云硬盘：性能降级将通过计算生命周期内剩余时间的价值与新购新配置价值的差价进行退费。具体计算按照实际情况，您可以在退款页面查看。
- 对于按量计费云硬盘：立即生效，并开始按新配置的价格进行计费。
