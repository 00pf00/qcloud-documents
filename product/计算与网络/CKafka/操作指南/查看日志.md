## 操作场景

CKafka 专业版实例支持在控制台上查看日志详情，包含消费分组协调者日志和 controller日志，本文介绍在 CKafka 控制台上查看日志详情的操作步骤。

> ?该功能仅专业版实例支持，标准版不支持。

**日志类型**

| 日志类型           | 事件                                                         |
| ------------------ | ------------------------------------------------------------ |
| 消费分组协调者日志 | <li>消费分组成员心跳超时</li><li>触发 rebalance 异常事件</li><li>消费分组成员提交消费进度</li><li>消费分组 rebalance</li> |
| controller日志     | <li>leader 切换</li><li>集群节点上线</li><li>集群节点下线</li> |

## 操作步骤

1. 登录 [CKafka 控制台](https://console.cloud.tencent.com/ckafka) 。

2. 在【实例列表】页，单击目标实例的“ID/名称”，进入实例详情页。

3. 在实例详情页，单击页面顶部的【日志】，选择好日志类型，可查看日志情况。

   ![](https://main.qcloudimg.com/raw/efbaf5c06355e124bbc7d6b4b2f783dc.png)

   | 参数     | 说明                                                         |
   | -------- | ------------------------------------------------------------ |
   | 事件名称 | 事件名称。                                                   |
   | 事件类型 | 事件根据是否会对服务造成影响分为**异常事件**和**状态变更**。 |
   | 影响对象 | 事件发生影响的对象，如Topic、消费组等。                      |
   | 开始时间 | 事件发生开始的时间。                                         |
   | 操作     | 可以查看事件详情或者配置告警策略。                           |

4. 单击操作栏的**详情**，可以查看对应的具体日志。

   ![](https://main.qcloudimg.com/raw/580dfdd9917b9197b01eb682be3c7f35.png)

5. 单击操作栏的**配置告警**，可以为已经接入告警的指标配置告警策略。详细说明参考[配置告警](https://cloud.tencent.com/document/product/597/57244)。
