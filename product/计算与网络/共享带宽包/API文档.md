对等连接 API 2.0 ， 相关接口如下表所示：

| 接口功能        | Action ID                                | 功能描述         |
| ----------- | ---------------------------------------- | ------------ |
| 创建共享带宽包   | [CreateBandwidthPackage](https://cloud.tencent.com/doc/api/245/2107) | 创建设备带宽包和 IP 带宽包。|
| 添加带宽包资源 | [AddBandwidthPackageResources](https://cloud.tencent.com/document/product/215/19213) | 添加带宽包资源，包括弹性公网 IP 和负载均衡等。|
| 删除共享带宽包 |[DeleteBandwidthPackage](https://cloud.tencent.com/document/product/215/19211) | 删除共享带宽包，包括设备带宽包和IP带宽包。|
|  删除带宽包资源  | [RemoveBandwidthPackageResources](https://cloud.tencent.com/document/product/215/19207)  |  删除带宽包资源，包括弹性公网IP和负载均衡等 。|
| 修改带宽包属性 | [ModifyBandwidthPackageAttribute](https://cloud.tencent.com/document/product/215/19208) |  修改带宽包属性，包括带宽包名字等。|
| 查询带宽包资源 | [DescribeBandwidthPackages](https://cloud.tencent.com/document/product/215/19209) |  查询带宽包详细信息，包括带宽包唯一标识ID，类型，计费模式，名称，资源信息等。|
| 查询带宽包配额 | [DescribeBandwidthPackageQuota](https://cloud.tencent.com/document/product/215/19210) |  查询账户在当前地域的带宽包上限数量以及使用数量。|
