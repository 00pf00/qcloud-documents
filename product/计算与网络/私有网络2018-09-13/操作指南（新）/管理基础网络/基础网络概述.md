基础网络和私有网络均为腾讯云上的网络空间：
+ 基础网络是腾讯云上所有用户的公共网络资源池（如下图右所示）。所有云服务器的内网 IP 地址都由腾讯云统一分配，无法自定义网段划分、IP 地址。
+ 私有网络是用户在腾讯云上建立的一块逻辑隔离的网络空间（如下图左所示）。与基础网络相比，在私有网络内，用户可以自由定义网段划分、IP 地址和路由策略，更适合有网络自定义配置需求的场景。

## 注意事项
+ 由于基础网络资源日益紧缺且无法扩增，2017年6月13日后，新注册的账号已不支持基础网络，即无法在基础网络内创建实例（云服务器、云数据库、负载均衡等），仅支持私有网络。
+ 如果您的业务必须使用基础网络服务，请 [提交申请](https://cloud.tencent.com/apply/p/qnm7krv9glo)。
## 网络互通
腾讯云提供基础网络和私有网络的互通功能，满足您各种场景的业务互联：
+ 基础网络云服务器访问私有网络云资源：您可以通过在私有网络控制台，配置**基础网络互通**功能，使得基础网络云服务器可以与私有网络内的云资源（云服务器、负载均衡等）互访通信，具体配置请参考 [基础网络互通](https://cloud.tencent.com/document/product/215/44696 )  。
+ 私有网络云资源访问基础网络非云服务器资源：您可以通过**终端连接**实现私有网络内实例通过内网与基础网络内非云服务器实例通信的功能。终端连接的原理是：将基础网络实例 IP 与私有网络内 IP 建立映射，访问该私有网络 IP 即访问基础网络实例。目前终端连接支持的基础网络产品包括：CLB、MySQL、Memcached、Redis、MongoDB。
>? 终端连接不支持跨地域、跨账号，如您有建立终端连接的需要，请提交 [工单申请](https://console.cloud.tencent.com/workorder/category)。
>
+ 基础网络切换到私有网络：私有网络具有可定制化，网络环境更安全等优点，我们推荐您使用私有网络部署云资源，腾讯云提供基础网络切换至私有网络的服务，您可参考 [切换私有网络服务](https://cloud.tencent.com/document/product/215/44696 ) 了解更多内容。
