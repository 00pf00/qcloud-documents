#概念类
####什么是私有网络？和基础网络有什么区别？如何选择？
私有网络是用户在腾讯云上建立的一块逻辑隔离的网络空间，具有更多功能，区别和选择请参见[基础网络和私有网络](https://cloud.tencent.com/document/product/215/532#.E9.80.89.E6.8B.A9.E5.9F.BA.E7.A1.80.E7.BD.91.E7.BB.9C-or-.E7.A7.81.E6.9C.89.E7.BD.91.E7.BB.9C.EF.BC.9F)。
####私有网络不同子网内如何通信？
每个私有网络默认内网互通，您可以在对应路由表中查看到有一条默认路由，该路由即表示该私有网络下所有资源内网互通。
如果是不同私有网络的子网（同账号/不同账号），则内网不互通，需要使用[对等连接](https://cloud.tencent.com/document/product/553)或[云联网](https://cloud.tencent.com/document/product/877)才能实现通信。

####什么是公网网关？
公网网关是开启了转发功能的云服务器，无外网 IP 但需进行 Internet访问的云服务器可通过位于非相同子网公网网关转发来访问 Internet。
更多信息请参见[公网网关概述](https://cloud.tencent.com/document/product/215/11119#.E7.AE.80.E4.BB.8B)
注意（本段蓝底）：公网网关有一定使用限制，网关子网和普通子网不能关联同一张路由表，需要新建一张独立的网关路由表关联网关子网。
####什么是地域和可用区？
地域是指物理的数据中心的地理区域，例如广州、北京、上海等。

- 不同地域之间网络完全隔离，不同地域之间的云产品 默认不能通过内网通信。
- 不同地域之间云产品可以通过 公网 IP 进行 Internet 访问；处于不同地域的私有网络中的云产品无法直接内网互联，需要建立 [对等连接](https://cloud.tencent.com/document/product/553) 以实现通信，以获得比 Internet 访问更稳定高速的互联。

可用区（Zone）是指腾讯云在同一地域内电力和网络互相独立的物理数据中心。目标是能够保证可用区间故障相互隔离（大型灾害或者大型电力故障除外），不出现故障扩散，使得用户的业务持续在线服务。
- 处于相同地域不同可用区，但在同一个 VPC 下的云产品之间均通过内网互通，可以直接使用 内网 IP 访问。
- 上述内网互通是指同一账户下的资源互通，不同账户的资源内网完全隔离。
####同一 VPC 网络下我是否可以将主机部署在不同可用区？
可以。VPC具有地域属性（如：广州、北京、韩国），VPC内子网具有可用区属性（如广州一区、广州二区），同一VPC内的子网可以在同一地域的不同可用区。而云主机的可用区属性继承自其所属子网，在可用区的子网下购买主机即可实现将主机部署在不同可用区。
####不同可用区间的云服务器和数据库如何通信？

- 相同 VPC：默认互通，如果不通，请优先排查 [安全组](https://cloud.tencent.com/document/product/213/124520) 及 [网络 ACL](https://cloud.tencent.com/document/product/215/5132) 等防火墙策略。
- 不同 VPC：默认隔离，如需互通，您可以通过 [对等连接](https://cloud.tencent.com/document/product/553) 或 [云联网](https://cloud.tencent.com/document/product/877)，实现两个 VPC 的内网互通。
