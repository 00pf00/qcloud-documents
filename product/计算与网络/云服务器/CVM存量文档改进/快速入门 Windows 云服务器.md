您只需要完成以下四个步骤，即可快速配置一台 Windows 腾讯云服务器。

-----



## **步骤一：** 注册腾讯云账号。
如果已在腾讯云注册，可忽略此步骤。

[点击注册腾讯云账号](https://cloud.tencent.com/register?s_url=https%3A%2F%2Fcloud.tencent.com%2Fdocument%2Fproduct)


## **步骤二：** 购买云服务器。

[点击进入快速购买页面](https://cloud.tencent.com/document/product/213/6090)

- 点击 **快速配置** 页签。 **地域** 选择与您最近的一个地区，比如我在“深圳”，地域选择“广州”。
- 
- **机型** 选择 “入门设置（1核1GB）”。
- **镜像** 选择您需要的云服务器操作系统。这里我们配置 Windows云服务器，选择 “Windows Server 2012 R2 数据中心版 64位中文版”。
- **公网带宽** 勾选后会为您分配公网IP，选择“1 Mbps”。
- **购买数量** 选择默认为 “1台”。**购买时长** 选择默认为 “1个月”。

>! 使用快速配置方式购买的云服务器，系统将为您自动分配云服务器登录密码并发送到您的站内信中，请您务必注意查收初始密码。

[点击查看初始登录密码](https://cloud.tencent.com/)

## **步骤三**：登录云服务器。

[点击进入云服务器控制台](https://cloud.tencent.com/login?s_url=https%3A%2F%2Fconsole.cloud.tencent.com%2F)

在实例列表中找到刚生成的实例，在右侧菜单中选择**“登录”。**
