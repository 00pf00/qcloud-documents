#示例说明

本示例主要演示：

1. 在 SCF 控制台上快速创建 API 网关触发器，由云函数实现 Web 后台
2. 在云函数中，通过调用 AI 的接口，实现图像转文字处理

## 实现概要

### 演示流程

1. 在本地 Web 页面上传带有文字的图片，图片会先做 base64 编码，然后通过 API 接口 post 到云函数
2. 云函数在收到请求后，会自动运行，从 event 参数里获取文件编码并做 base64 解码，还原图片
3. 把还原后的图片上传到 COS bucket 中留存（可选）
4. 通过 AI 的接口对图片进行识别，把识别后的结果返回到前端 Web 页面

### 实现步骤

1. 环境准备：本地Python2.7 运行环境、pip工具
2. 在云端部署云函数，实现 AI 接口的调用，并配置 API 网关触发器
3. 在本地创建 html 静态页面，实现本地图片的上传和处理结果的展示
