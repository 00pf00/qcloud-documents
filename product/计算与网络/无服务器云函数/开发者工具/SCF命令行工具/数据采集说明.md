## 数据采集内容
为了提升工具使用体验，了解用户使用情况以及提供部分功能的取舍决策，SCF CLI 会采集关于以下命令的相关数据：

- scf init
- scf deploy
- scf function
- scf eventdata
- scf local
- scf logs
- scf native
- scf state

所采集的数据只有命令和其参数的使用频次，不包括参数值等信息。而获取到的统计数据，仅会用于 SCF 团队对 CLI 模块进行功能升级和性能优化。

## 关闭数据采集

您可以执行 `scf configure set -ar n` 命令关闭数据采集。
