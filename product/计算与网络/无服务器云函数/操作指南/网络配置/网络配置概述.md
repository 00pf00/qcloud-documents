云函数创建成功后，默认只有公网访问权限。即云函数在运行时，只能访问例如腾讯云官网等暴露在公共网络环境中的资源。

云函数的网络配置项支持用户进行以下设置：
- 可设置仅保持公网访问。
- 同时设置公网访问和公网出口 IP 固定，此时云函数使用一个固定的 IP 地址访问公共网络资源。
- 仅内网访问，云函数可以访问到已配置该私有网络中的数据库、Redis 等资源。
- 同时开启内网访问和公网访问，请注意：
 - 该模式下云函数会强制开启固定公网出口 IP 功能。
 - 云函数访问公网所使用的流量，将会基于该固定公网 IP 进行转发。

>?云函数获取固定公网出口 IP 配置，请参考 [固定公网出口 IP](https://cloud.tencent.com/document/product/583/38198)。
>
## 网络配置方法
1. 登录 [云函数控制台](https://console.cloud.tencent.com/scf/index)，单击左侧导航栏中的【函数服务】。
2. 在页面上方选择地域，单击需要进行网络配置的函数名。
3. 在“函数配置”页面，选择右上角的【编辑】。
4. 您可根据实际需求，参考 网络配置 相关文档进行配置。


## 查看网络配置
1. 登录 [云函数控制台](https://console.cloud.tencent.com/scf/index)，单击左侧导航栏中的【函数服务】。
2. 在页面上方选择地域，并单击函数名，即可在“函数配置”中查看当前网络配置。
