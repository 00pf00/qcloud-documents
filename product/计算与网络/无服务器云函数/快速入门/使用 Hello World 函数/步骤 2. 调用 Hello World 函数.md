用户可以按照以下步骤使用控制台提供的测试模版来调用刚刚创建的Hello World函数。


1. 在【函数代码】页里，在函数编辑框底部，找到【测试】按钮，保持默认的函数测试模板即可。
![](https://main.qcloudimg.com/raw/0c3491fc828411697aa7ea4dd77b586c.png)

2. 如果想要更换测试模板或者模板中的内容，分别可以单击`更换`和`配置`按钮。需要注意的是，不同的测试模板分别模拟不同的触发器消息源。同时，不同的触发器和云函数之间传递消息，均为约定好的数据结构，详情可参考[触发器介绍](https://cloud.tencent.com/document/product/583/9705)。
3. 在本次测试过程中，我们使用的是测试模版中的`Hello World事件模版`，该模板会把如下数据结构发送给云函数。云函数会在`main_handler`的`event`参数中，获取该数据结构。
```
{
  "key1": "test value 1",
  "key2": "test value 2"
}
```

4. 单击【测试】按钮，代码开始运行并将显示测试结果。同时，也可前往【运行日志】选项卡查看运行结果。
![](https://main.qcloudimg.com/raw/72d13043d1e565973c62293441c1b071.png)

5. 您可以多运行几次，并单击【监控信息】选项卡,来查看监控信息。需要注意的是，监控统计的粒度最小为 1 分钟，需要 1 分钟后，方能查看当次的监控记录。
