1. 登录腾讯云控制台，选择[无服务器云函数](https://console.cloud.tencent.com/scf/list?rid=1)，进入【函数服务】页面。
 
2. 选择`广州`地域，单击【创建】按钮，进入新建函数页面。

3. 创建方式保持为`空白函数`，填写函数名称`hello-world`，然后选择期望使用的语言，这里可以选择 Python2.7。

4. 单击【下一步】按钮，进入函数代码编辑页，保持默认配置即可，单击【完成】按钮，完成函数的创建，如下图所示：
![](https://main.qcloudimg.com/raw/c96dc28b16c023f6a9688c09f6e285a1.png)

 - 执行方法显示`index.main_handler`。表示无服务器云函数控制台会将此段代码自动保存为`index.py`文件，并压缩该文件上传至 SCF 平台以创建云函数。
 - 函数代码显示以下代码片段：

```
# -*- coding: utf8 -*-
print('Start Hello World function')
def main_handler(event, context):
    print('Hello World')
    if 'key1' in event.keys():
        print("value1 = " + event['key1'])
    if 'key2' in event.keys():
        print("value2 = " + event['key2'])
    return "hello from scf"  #return
```
 - 该示例代码的`main_handler`为入口函数，主要有两个入参，`event`参数中可以获取触发源的消息，`context`参数中可以获取本函数的环境及配置信息。

5. 函数创建完成后，您可以在【函数配置】里查看函数配置等信息，也可以在【函数代码】里在线编辑代码。
