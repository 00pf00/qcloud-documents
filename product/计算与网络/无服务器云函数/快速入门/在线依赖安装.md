云函数提供部署时在线安装依赖的功能。

## 前提条件
 - 已开通云函数在线安装依赖功能。
线安装依赖目前为内测发布功能，可通过 [内测申请](https://cloud.tencent.com/apply/p/kd8np1at8r) 获取此功能。
 - 暂时只支持北京和广州地域。


## 启用在线安装依赖
如果在函数配置页面中启用了“在线安装依赖”，每次上传代码后，云函数后台会自动安装依赖。

![](https://main.qcloudimg.com/raw/08be78c4d91228f41b53cfe4f326942c.png)

## Node.js 在线安装依赖
针对 Node.js 的运行时，如果在函数配置中启用了“在线安装依赖”，每次上传代码后，云函数后台将检查代码包根目录的 package.json文件，并使用 npm 工具尝试安装 package.json 中有提及但代码包中不存在依赖包。

在此示例中，package.json 文件中列出了一个依赖包:
```
{
    "dependencies": {
    "lodash": "4.17.15"
  }
}
```
此依赖包随后被导入函数中：
```
const _ = require('lodash');
exports.handle = (event, context, callback) => {
    _.chunk(['a', 'b', 'c', 'd'], 2);
    // => [['a', 'b'], ['c', 'd']]
};
```

### 使用 npm 安装 Node.js 依赖包
安装 Node.js 依赖包最简单的方法是在您的云函数文件夹中使用 npm install 命令。例如，以下命令将添加 lodash 依赖包：
```
npm install lodash
```
