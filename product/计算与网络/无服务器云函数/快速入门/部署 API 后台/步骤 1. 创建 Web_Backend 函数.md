1. 登录腾讯云控制台，选择[无服务器云函数](https://console.cloud.tencent.com/scf/list?rid=1)，进入【函数服务】页面。
 
2. 选择`广州`地域，单击【创建】按钮，进入新建函数页面。

3. 选择`模板函数`，填写函数名称`web_backend`，然后选择期望使用的语言，这里可以选择 Python2.7,模板选择“API网关触发器基础范例”。
  ![](https://main.qcloudimg.com/raw/5749b6161b101eff5009649e08c29da8.png)

4. 单击【下一步】按钮，进入函数代码编辑页，保持默认配置即可，单击【完成】按钮，完成函数的创建。切换到【函数代码】选项页：
 - 执行方法显示`api_gw_basic_demo.main_handler`。表示无服务器云函数控制台会将此段代码自动保存为`api_gw_basic_demo.py`文件，并压缩该文件上传至 SCF 平台以创建云函数。
 - 函数代码显示以下代码片段：

```
# -*- coding: utf-8 -*-
import sys
import logging

print('Loading function')
logger = logging.getLogger()
def main_handler(event, context):
    logger.info("start main handler")
    print(event)
    body = 'API GW Test Success'
    response = {
        "isBase64": False,
        "statusCode": 200,
        "headers": {"Content-Type": "text", "Access-Control-Allow-Origin": "*"},
        "body": body
    }
    return response
```
 - 该示例代码的`main_handler`为入口函数，主要有两个入参，`event`参数中可以获取触发源的消息，`context`参数中可以获取本函数的环境及配置信息。

5. 函数创建完成后，您可以在【函数配置】里查看函数配置等信息，也可以在【函数代码】里在线编辑代码。
6. 切换至【触发方式】选项页，单击【添加触发方式】，为云函数添加 API 网关触发器。触发方式选择“API网关触发器”，其余保存默认配置即可，单击【保存】完成创建。如下图所示：
