1. 复制 API 网关触发器生成的`访问路径`，即 https 链接。

2. 在浏览器里访问该链接，并查看浏览器收到的返回值。
![](https://main.qcloudimg.com/raw/10739b9bcab1b4ae9f79cac5428fad1f.png)

3. 返回云函数控制台，切换至【运行日志】选项卡，检查运行结果及运行日志。
![](https://main.qcloudimg.com/raw/19a4315da90d7ab8937ba19fd19053d0.png)

4. 您可以多运行几次，并单击【监控信息】选项卡,来查看监控信息。需要注意的是，监控统计的粒度最小为 1 分钟，需要 1 分钟后，方能查看当次的监控记录。
