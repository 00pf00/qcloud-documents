## 公网无固定 IP 型 CLB 升级公告
尊敬的腾讯云用户您好，

为了给您提供更加优质的负载均衡服务，腾讯云公网无固定 IP 型负载均衡将于`2018年11月15日`前进行全网升级。升级后的负载均衡性能更强、特性更多、服务更健壮。

公网无固定 IP 型负载均衡服务由于推出时间较长，性能方面逐渐无法满足用户的业务需求，即将在`2018年11月15日`下线。升级后的负载均衡价格为 0.02元/小时，支持四层（TCP/UDP）和七层（HTTP/HTTPS）转发规则，并且具备监控告警、安全组、日志管理等功能。

升级过程平滑、对业务无影响，腾讯云会将有服务（有转发规则且绑定云主机）的公网无固定 IP 型负载均衡升级为传统型公网负载均衡，对其余无服务（没有转发规则或者未绑定云主机）的公网无固定 IP 型负载均衡则进行回收。作为补偿，我们将发放一定量的负载均衡专用代金券到您的账户，您可以使用这些代金券购买负载均衡服务。

**升级计划**
1. 升级时间：`2018年10月15日 – 2018年11月15日`
  - 升级对象为全网有服务（有转发规则且绑定云主机）的公网无固定 IP 型负载均衡。
  - 腾讯云将分用户、分地域逐步平滑升级。
  - 升级前需保证您的账户余额 **大于或等于0（即非欠费状态）** ，腾讯云将优先使用代金券为您扣费。
  - 此次升级为平滑升级，业务无感知，相关域名、转发规则、实例名称等均会保留，升级后价格为0.02元/小时，负载均衡实例ID会改变。

2. 回收时间：`2018年11月15日 15:00 - 17:00`
  - 回收对象为全网无服务（没有转发规则或者未绑定云主机）的公网无固定 IP 型负载均衡。
  - 腾讯云将对全部地域统一进行回收操作。
注：若您对升级实例、升级时间等有特殊要求，或者在此过程中有任何问题，请登录工单系统提单或者拨打4009-100-100沟通，我们将尽快为您核实处理。

感谢您对腾讯云的支持和理解，谢谢！
