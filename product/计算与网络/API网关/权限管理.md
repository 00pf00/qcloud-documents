### API网关上的子账号协作者权限配置

通过腾讯云上统一的权限管理平台，主账号可以配置子账号对于API网关上常见操作的操作权限和允许操作的服务以及 API 资源。

#### 配置子账号和协作者对资源的操作权限 - 无标签

1. 主账号登陆选择[访问管理控制台](https://console.cloud.tencent.com/cam/overview)。

2. 选择策略选项并新建自定义策略。![](https://main.qcloudimg.com/raw/43fe8e95955a7bad38532032d1b64b03.png)

3. 填写自定义策略

   ![](https://main.qcloudimg.com/raw/3991649e2974195b188751e21471bb61.png)

   - 选择策略效果
   - 选择服务为API网关
   - 选择涉及到的操作
   - 填写资源。当填写内容为所有资源时可以写做 * ，如果仅仅涉及到部分资源，可以参考使用指南填写
   - 点击添加声明并点击下一步

4. 编辑策略，填写策略名称并创建策略

5. 将创建好的策略绑定在用户或者用户列表上。

#### 配置子账号或协作者对标签资源的操作权限

 标签能力是腾讯云上的统一能力，您可以对腾讯云上的不同产品的不同资源设置相同的标签，对一套标签设置相同的操作权限。

在API网关中，可以设置 API 的标签，设置入口位于：

![](https://main.qcloudimg.com/raw/3017ce7453b65abaff1e50f284c3e122.png)

点击笔状按钮即可编辑标签。

配置基于标签的管理权限与配置非标签的管理权限入口相同，在访问管理 - 策略 - 新建自定义策略，选择按照标签授权，填写标签以及标签值，并添加用户以及用户组信息即可。
