## API网关上的子账号协作者权限配置
## 操作场景
在腾讯云访问管理控制台中，主账号可以配置子账号在 API 网关上常见操作的权限、允许操作的服务及 API 资源。

## 操作步骤
### 配置子账号和协作者对资源的操作权限 - 无标签
1. 主账号登录【[访问管理控制台](https://console.cloud.tencent.com/cam/overview)】>【[策略](https://console.cloud.tencent.com/cam/policy)】。
2. 在策略列表页，单击左上角【新建自定义策略】。
3. 在新建策略弹窗中，选择【按策略生成器创建】。
![](https://main.qcloudimg.com/raw/2bbada93f591392ae94f2cf21e07c324.png)
4. 填写自定义策略信息，完成后单击【添加声明并】和【下一步】进入编辑策略页面。
![](https://main.qcloudimg.com/raw/eaf8e57dda41f749606b81951190402e.png)
   - 效果：允许
   - 服务：API 网关
   - 操作：选择设计到的操作
   - 资源：当填写内容为所有资源时可以写做`*` ，如果仅涉及到部分资源，可以参考 [资源描述方式](https://cloud.tencent.com/document/product/598/10606) 填写。
5. 填写策略名称，并对策略内容进行编辑（详情请参考 [策略语法](https://cloud.tencent.com/document/product/598/10604)）。
6. 单击【创建策略】，完成策略创建。
7. 将创建好的策略绑定在用户或者用户组上（详情请参考 [通过策略关联用户/用户组](https://cloud.tencent.com/document/product/598/10602#.E9.80.9A.E8.BF.87.E7.AD.96.E7.95.A5.E5.85.B3.E8.81.94.E7.94.A8.E6.88.B7.2F.E7.94.A8.E6.88.B7.E7.BB.84.EF.BC.9A)）。

### 配置子账号或协作者对标签资源的操作权限
标签能力是腾讯云上的统一能力，您可以对腾讯云上的不同产品的不同资源设置相同的标签，对一套标签设置相同的操作权限。

1.登录【[API 网关](https://console.cloud.tencent.com/apigateway/index?rid=8) 】>【[服务](https://console.cloud.tencent.com/apigateway/service?rid=8)】。
2. 在服务列表页，单击服务名进入【API 管理】。
3. 单击 API 的 ID，进入API 详情页面。 单击笔状按钮即可编辑标签。
![](https://main.qcloudimg.com/raw/5f416a9f57c2b576d443c8b41ca4cde2.png)
配置基于标签的管理权限与配置非标签的管理权限入口相同，在访问管理 - 策略 - 新建自定义策略，选择按照标签授权，填写标签以及标签值，并添加用户以及用户组信息即可。
