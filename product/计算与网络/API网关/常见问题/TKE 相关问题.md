## 开启 TKE 集群内网访问功能

1. 登录【容器服务】控制台 ，选择左侧导航栏中的**集群**，进入集群管理界面。
2. 单击需要连接的集群“ID/名称”，进入集群详情页。
3. 选择左侧导航栏中的基本信息，即可在基本信息页面中查看**集群APIServer信息**模块中该集群的访问地址、外网/内网访问状态、Kubeconfig 访问凭证内容等信息。如下图所示：

​                 ![img](https://docimg9.docs.qq.com/image/kIIYKlq9mZZTrMOAN573WA.png?w=830&h=409)        

4. 点击**开启内网访问**按钮。开启内网访问时，需配置一个子网，开启成功后将在已配置的子网中分配 IP 地址。注意开启内网访问后，不能关闭，否则 API 网关会访问不到集群的 APIServer。



## 获取 TKE 集群 admin 角色

1. 登录【容器服务】控制台 ，选择左侧导航栏中的**集群**，进入集群管理界面。
2. 单击需要连接的集群"ID/名称"，进入集群详情页。
3. 选择左侧导航栏中的**授权管理**，然后在授权管理中选择**ClusterRole**，接着在主界面就看到**获取集群admin角色**的按钮。点击即可获取集群 admin 角色。

​                 ![img](https://docimg10.docs.qq.com/image/cKOkcp0m27JGPkbPCq7zHA.png?w=634&h=162)        

4. 如果获取集群admin角色失败，按照提示解决即可。一般是由于子账号没有 TKE 集群的 Cam 权限导致的。
