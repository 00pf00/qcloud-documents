产品限制

单个租户下API网关使用限制如下：

| 类别                           | 限制  |
| ------------------------------ | ----- |
| 单租户下服务数量               | 50个  |
| 每个服务中API数量              | 200个 |
| 每个服务的自定义独立域名数     | 5个   |
| 单租户下使用计划数             | 200   |
| 单组户下密钥对数               | 400   |
| 每个使用计划可绑定密钥数       | 50    |
| 单个密钥对可被绑定的使用计划数 | 10    |
| 单个使用计划和配置最高 QPS     | 2000  |
| 使用计划可绑定的API数量        | 10000 |
| 单个服务可支持被访问的最大 QPS | 5000  |

其中，对于API网关共享集群，单个服务可支持被访问的最大 QPS 5000，API网关共享集群不能提升数额。对于 QPS 需求更高的客户，欢迎通过工单咨询开通 API 网关性能优化型集群服务。
