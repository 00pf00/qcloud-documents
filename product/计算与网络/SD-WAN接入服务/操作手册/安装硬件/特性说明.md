### 指示灯说明
![](https://main.qcloudimg.com/raw/b3b13212feca34c77345b4da282880a0.png)
>?所有面板灯为绿色，慢闪表示1s亮1s灭，快闪表示0.5s亮0.5s灭。

| **指示灯名称** | **功能定义**                                                 |
| -------------- | ------------------------------------------------------------ |
| PWR            | <ul><li>长亮：电源接通</li><li>熄灭：无电源   </li></ul>                                |
| SYS            | <ul><li>慢闪：正常运行，Edge 控制器可以 ping 通</li><li>快闪：起始状态，控制器未通</li><li>长亮：Agent 挂起</li><li>熄灭：硬件自检故障 </li></ul>  |
| WIFI           | 表示 Edge 设备的 WiFi 连接状态<ul><li>长亮：WLAN 启动</li><li>闪烁：数据传输</li><li>熄灭：WLAN 未启动</li></ul> |
| LTE/5G         | <ul><li>长亮：已注册网络</li><li>闪烁：数据传输</li><li>熄灭：未注册网络 </li></ul>                |
| CLOUD          | 表示 Edge 设备是否连接到腾讯云<ul><li>慢闪：正常工作，已经连接到腾讯云联网</li><li>快闪：云网络部分隧道断开或者正在建立连接</li><li>长亮：隧道完全断开（已经建立，但后来断开）</li><li>熄灭：未开始建立隧道</li></ul> |
| WAN/LAN        | 表示 Edge 设备的以太网使用状态<ul><li>长亮：以太网已连接</li><li>闪烁：数据传输</li><li>熄灭：以太网未连接 </li></ul> |
