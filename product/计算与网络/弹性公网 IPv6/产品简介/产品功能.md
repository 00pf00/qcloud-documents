云服务器或者弹性网卡申请了 IPv6 地址后，默认关闭了公网访问能力，仅支持 VPC 内的 IPv6 通信。通过弹性公网 IPv6，支持单个 IPv6 或者多个 IPv6 开启公网或者关闭公网。

### VPC 内通信
同一 VPC 下不同的云服务器或弹性网卡获取到 IPv6 地址，并在云服务器启用该 IPv6 地址后，即默认支持 VPC 内的 IPv6 相互通信。
### 开通/关闭公网
每一个弹性网卡获取到 IPv6 地址后，会在弹性公网 IPv6 控制台生成一个表项，包含了该公网 IP 所在的弹性网卡、VPC、子网信息。通过设置带宽来为每个 IPv6 开通公网和设置公网带宽峰值。如果已经开启了公网，也可以设置带宽为0，来关闭公网。
### 开通公网（批量）
您可以通过开通公网来为多个 IPv6 地址开启公网并设置统一的公网带宽。
### 关闭公网（批量）
您可以通过关闭公网来为多个 IPv6 地址关闭公网访问。
