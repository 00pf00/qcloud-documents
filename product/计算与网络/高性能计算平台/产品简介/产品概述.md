## 产品简介
腾讯云高性能计算平台（TencentCloud High Performance Computing，THPC）是一款腾讯云自研的高性能计算资源管理平台服务，集成 IaaS 资源，提供弹性、可靠及高性能的计算服务。可以帮助您高效地管理云上高性能计算资源，实现弹性使用云上高性能计算资源的需求。

高性能计算平台支持常用作业调度软件（sge 等）的自动化部署，并按您的业务需求使用计算资源。您可使用平台即时获取海量弹性计算资源，轻松地构建和部署业务，且仅需为所使用的资源付费。


## 产品功能

| 功能     | 说明                                                 |
| -------- | ---------------------------------------------------- |
| 集群管理功能 | 对集群进行启动、查询、删除等操作。             |
| 节点管理功能 | 对节点进行查询、启动、停止、重启、删除等操作。 |
