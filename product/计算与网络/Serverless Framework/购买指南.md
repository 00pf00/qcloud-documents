## 关联产品免费额度

当前，Serverless Framework 服务用到云函数SCF、API 网关、对象存储 COS 等产品，均提供免费额度，详细说明如下：

| 服务 |  免费额度 |
| ---- | ---- |
| 云函数 SCF  | 开通云函数 SCF 服务的用户，每月可享受 40万GBs 的免费资源使用量及 100万次 调用次数。详情参考[云函数 SCF 免费额度说明](https://cloud.tencent.com/document/product/583/12282) |
| API 网关   | 开通服务后，第一年每月（自然月）前一百万次调用免费，当月超过部分按照调用次数费用表阶梯计费。<br/>
开通服务后，第一年每月（自然月）前1GB流量免费，当月超过部分按照流量费用表按地域不同计费。详情参考[API 网关免费额度说明](https://cloud.tencent.com/document/product/628/39300#.E5.85.8D.E8.B4.B9.E9.A2.9D.E5.BA.A6) |
| 对象存储 COS  | 新开通对象存储 COS 服务的个人用户将享受6个月 50GB 标准存储容量的免费额度；<br/>企业用户将享受6个月 1TB 标准存储容量的免费额度；详情参考[对象存储 COS免费额度说明](https://cloud.tencent.com/document/product/436/6240) |
