## 产品简介
弹性容器服务（ElasticKubernetesService，EKS）是腾讯云容器服务推出的无须用户购买节点即可部署工作负载的服务模式。弹性容器服务完全兼容原生Kubernetes，支持使用原生方式购买、管理资源，并扩展支持腾讯云的存储、网络等产品，开箱即用。弹性容器服务按容器真实使用的资源量计费，腾讯云保证用户容器的安全隔离。

## 相关概念

**容器和镜像**

容器技术通过把应用程序及其依赖全部打包成镜像，再使用镜像生成资源隔离的环境来运行程序，简单高效的实现了应用程序运行时的独立性和环境一致性。

容器是操作系统级别的轻量虚拟化技术，把系统资源进行隔离和控制，让原来全局的资源变得只能容器内进程使用。

容器镜像类似虚拟机的快照，但更轻量，可以理解是容器的静态形式。镜像定义了容器运行的一切文件和依赖关系，保证了容器运行的一致性。

**Kubernetes**

Kubernetes 是 Google 基于 Borg 开源的容器编排调度引擎，是 CNCF（Cloud Native
Computing Foundation）最重要的组件之一。其提供了生产级别的应用编排、容器调度、服务发现、自动扩缩容等能力。详情请参阅 [Kubernetes 官方文档](https://kubernetes.io/docs/home)。

## 产品优势

**原生支持**

弹性容器服务紧跟社区，支持最新的 Kubernetes 版本，支持原生的 Kubernetes 集群管理方式。以插件形式扩展支持了腾讯云系列存储、网络、负载均衡服务，开箱即用。

**无服务器**

弹性容器服务是一种全托管的 Kubernetes 服务，意味着用户无须管理任何计算节点。EKS 以 Pod 的形式交付计算资源，支持用户使用 Kubernetes 原生的方式购买、退还、管理云资源。

**安全可靠**

弹性容器服务基于腾讯云成熟的虚拟化技术和网络架构，提供99.95 %的服务可用性。腾讯云保证用户间 EKS 集群的虚拟化隔离和网络隔离，支持用户通过安全组、网络ACL等产品为具体服务配置网络策略。

**秒级伸缩**

弹性容器服务通过腾讯云自研的轻量虚拟化技术保证更快的资源创建效率，您可以在几秒内创建或删除您的容器服务。EKS 支持通过设置 Kubernetes 原生 HPA 的方式让服务根据实际负载进行自动伸缩。

**降低成本**

无服务器的形态决定了 EKS 能为用户带来更高的资源利用率和更低的运维成本，灵活高效的弹性伸缩保证容器服务总是只会使用当前负载需要的资源量。

**服务集成**

EKS 与腾讯云的大部分业务都可以做到高度集成，比如存储产品 CBS、CFS、COS等，云数据库 CDB 系列产品，私有网络 VPC 系列产品等，提供满足各类业务需求的解决方案。

## 产品定价

弹性容器服务是全托管的无服务器 Kubernetes 服务，和 TKE 托管集群一样，不会收取托管的 Master、Etcd 等管理面资源的费用。

弹性集群内运行的工作负载会采用后付费的按量计费模式。费用根据实际配置的资源量，按使用时间计算，具体请参考 [价格总览](https://cloud.tencent.com/document/product/xxx待定)。

## 应用场景

**微服务场景**

使用 EKS 来运行微服务免除您对计算节点的运维工作。服务可根据负载情况自动伸缩，让您永远用最合理的资源量来承载应用，降低资源使用成本。

**离线计算场景**

使用 EKS 运行离线计算任务，只须准备好容器镜像，即可快速部署任务负载。另外，EKS 只会收取任务真实运行时间所使用算力的费用，任务结束 Pod 自动释放即结束计费。

**在线推理场景**

EKS 支持使用 CPU、GPU 以及 vGPU 来运行在线推理服务，丰富的资源规格和弹性伸缩的负载，让您运行服务更高效、更经济。
