1. 登录 [弹性公网 IP 控制台](https://console.cloud.tencent.com/cvm/eip)。
2. 在弹性公网 IP 管理页面，选择需要解绑云资源的弹性公网 IP，单击【更多】>【解绑】。
3. 在弹出的 “解绑弹性公网IP” 窗口中，确认解绑信息，单击【确定】。
>? 若与弹性公网 IP 解绑的云资源为 CVM 实例，且该弹性公网 IP 为主网卡的主 IP 绑定的公网 IP，可在解绑时勾选【解绑后分配免费公网IP】，解绑后将被分配新的普通公网 IP，被分配新的普通公网 IP 可能与绑定前的普通公网 IP 不一致。
>
![](https://main.qcloudimg.com/raw/98aa3b6a8fc2b735959e45eebc744b6b.png)
4. 在弹出的 “解绑弹性公网IP” 提示框中，单击【确定】，即可完成与云资源的解绑。
