
## 购买指南


腾讯云在线教育互动课堂（Tencent Interact Class，TIC）是集实时音视频、交互式白板涂鸦、IM聊天室、PPT课件共享、屏幕分享、录制回放等功能于一体的一站式在线教育互动课堂解决方案。在使用互动课堂解决方案前，您需要开通音视频服务、云通讯服务以及互动白板服务。


### 开通音视频

开通实时音视频服务，请您移步到[腾讯云实时音视频](!https://cloud.tencent.com/document/product/647/17195)开通服务并创建应用。应用创建成功会生成与之一一对应的`SDKAppID`。

> 新用户创建应用赠送1000分钟的体验套餐，如需单独的技术支持群，帮助您更快速稳定接入，则需购买73888的的预付费套餐包。

### 开通云通讯

创建实时音视频应用成功后，我们将为您生成和音视频应用一致的`SDKAppID`。请您登陆[腾讯云官网](https://cloud.tencent.com/)后移步到[云通讯应用列表](!https://console.cloud.tencent.com/avc)中查看。

> 音视频体验套餐自动生成的云通信`SDKAppID`的业务版本为`TRTC体验版`，音视频聊天室数量上限10，群人数上限20上线，创建和加入群数量上限50。

### 开通互动白板

申请试用或正式的互动课堂服务，请您按以下格式发送邮件至`ticsupport@qq.com`，使用互动课堂所需的相关凭证，我们将会以邮件形式回复。

* 试用服务申请
```
标题：***公司为***（SDKAppID）申请互动课堂试用服务

公司名：
账号ID：
SDKAppID：
联系人姓名：
联系电话：
行业类型：（学前教育，K12，高等教育，语言类，职业教育，O2O）
```

1. `SDKAppID`是您在`实时音视频`中`创建应用`后得到的应用唯一标识。
2. `账号ID`为腾讯云账号唯一标识，再腾讯云官网注册登陆后可从[账号中心](!https://console.cloud.tencent.com/developer)查看。
3. 申请成功后，我们将为您开通除`云端课后录制`以外的`互动白板`、`PPT动画转码`、`云端实时录制`功能。
4. 试用有效期为30天，为不影响您的正式业务，请于有效期内申请正式服务。

* 正式服务申请

```
标题：***公司为***（SDKAppID）申请互动课堂正式服务

基本信息：
    公司名：
    账号ID：
    SDKAppID：
    联系人姓名：
    联系电话：
    行业类型：（学前教育，K12，高等教育，语言类，职业教育，O2O）

其他信息：
    管理员UserID：
    管理员UserSig：
```

1. `SDKAppID`是您在`实时音视频`中`创建应用`后得到的应用唯一标识。
2. `账号ID`为腾讯云账号唯一标识，再[腾讯云官网](https://cloud.tencent.com/)注册登陆后可从[账号中心](!https://console.cloud.tencent.com/developer)查看。
3. 如果您需要开通`云端课后录制`，请提供在`云通讯`对应`SDKAppID`下`基础配置`中的`管理员UserID`，并使用`开发辅助工具`生成`管理员UserSig`，为避免经常更换，建议有效期1年以上。
