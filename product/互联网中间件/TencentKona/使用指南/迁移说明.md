腾讯 Kona 基于 OpenJDK，同时提供了更多的功能拓展及维护。通过腾讯 Kona，用户可以获得更为先进的功能及性能优化，提高用户和开发者的使用体验。

KonaJDK与商业版JDK对比如下：

| 特性              | 商业版JDK                                  | Kona JDK                                                     |
| ----------------- | ------------------------------------------ | ------------------------------------------------------------ |
| 更新策略          | 定期更新                                   | 定期更新(3个月，高级别安全漏洞即时更新)                      |
| 组件与替代建议    | Java Web Start                             | 可单独安装IcedTea-Web                                        |
|                   | JavaFX                                     | 可单独单独安装OpenJFX                                        |
|                   | T2K font rendering engine                  | KonaJDK内置FreeType                                          |
|                   | Ductus 2D renderer                         | KonaJDK内置Pisces                                            |
|                   | Kodac Color Matching System (KCMS) library | KonaJDK内置LCMS                                              |
|                   | SNMP                                       | 可单独安装SNMP4J                                             |
|                   | Sound drivers                              | 可单独安装 Windows sound driver                              |
|                   | Java Flight Recorder (JFR)                 | KonaJDK内置稳定高效JFR                                       |
|                   | Java Mission Control (JMC)                 | 可单独安装 JDK Misson Control                                |
| JVM核心与分析工具 | GC（PS，CMS，G1等算法）                    | GC线程调度优化（默认开启）：GC并行线程通过Work Stealing进行负载均衡，减少GC停顿时间<br>物理内存释放优化（-XX:+FreeHeapPhysicalMemory开启） |
|                   | JDK工具                                    | jstack，jmap等工具优化                                       |
| JDK 基础库        | 基本密码算法                               | 内置JCA国密算法sm2/sm3/sm4支持，在KonaJDK中的API符合JCE CipherAPI的标准 |
