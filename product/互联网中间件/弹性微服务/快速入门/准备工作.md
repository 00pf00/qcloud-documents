# 准备工作

![](https://main.qcloudimg.com/raw/008e7403f8d0e9b36c88c95824860eba.png)

跟随快速入门教程引导，您将可以快速开始您的弹性微服务体验，尝试部署一个我们提供Demo应用或者您开发的应用。

本文将向您介绍，在使用弹性微服务托管您的应用之前需要完成的准备工作：包括新建账号、申请内测权限、新建环境和私有网络。



#### **步骤一：账号注册和申请内测权限**

在开始使用弹性微服务（TEM）前，您将需要获得内测资格。内测申请成功后，在内测期间，本产品使用不收取计算资源费用，您自行选择的相关云产品则按需收费。计费方式的详细说明，请见 [购买指南](https://cloud.tencent.com/document/product/1371/52885)。

（1）在开始使用腾讯云服务前，您需要先 [注册一个腾讯云账号](https://cloud.tencent.com/document/product/378/17985)。

（2）申请 [弹性微服务内测](https://console.cloud.tencent.com/tem) 资格。获取内测资格后，您可访问控制台。



#### **步骤二：新建环境和私有网络**

1. 登陆 [弹性微服务控制台](https://console.cloud.tencent.com/tem)
2. 在左侧导航菜单中选择**环境**，在**环境**页面上方选择**部署地域**，之后点击**新建**。
3. 在**新建**面板中，配置**环境**相关信息。

![img](https://docimg5.docs.qq.com/image/F3YnJZ5poqjQtzzPEmGKIQ?w=978&h=693)

- 名称：不超过40个字符
- 私有网络（VPC）：选择已有的VPC。如果您现有网络不合适或还未创建网络，可单击 [新建私有网络跳](https://console.cloud.tencent.com/vpc/vpc?rid=4) 转进行配置（地域选择需与环境一致），完成后回到本页面刷新后进行选择。
- 子网：选择已有的子网，建议选择多可用区部署，以提升容灾能力。如果您现有的子网不合适或还未创建子网，可以单击 [新建子网](https://console.cloud.tencent.com/vpc/subnet?rid=4&unVpcId=) 跳转进行配置，完成后回到本页面刷新后进行选择。
- 自动为您部署 CoreDNS 支持环境内应用发现。自动在 kubernetes 集群命名空间 kube-ststem 中部署2副本的 Deployment:coredns，该应用默认不收取费用，同时不建议进行修改。                  

4. 单击**确定**，环境进入**初始化状态**，等待几分钟后，**环境创建完成**。
