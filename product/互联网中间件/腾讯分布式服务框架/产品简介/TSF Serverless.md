## Serverless微服务平台 TSF Serverless 简介

TSF Serverless （Tencent Service Framework Serverless）是面向应用和微服务的高性能Serverless平台，提供按需使用、按量计费、免运维的使用体验。支持南北向 Web 应用场景和东西向的 Spring Cloud 和 Service Mesh 微服务场景。

TSF Serverless 目前处于内测阶段，请先填写 [使用申请](https://cloud.tencent.com/apply/p/om62iz2gqx)。申请审核通过后，即可获取内测资格。

内测期间，您可以免费使用。内测结束之后，本产品将进行公测。届时我们将在 [产品发布台](https://cloud.tencent.com/product/events) 发布公告，如有定价方面的变动也将提前通知您。

### 功能特性

* 通用的应用托管平台
  支持南北向 Web 框架（如Express、Koa）和东西向微服务框架（ 如Spring Cloud 和 Service Mesh ）。

* 精益成本，不为闲置资源付费
  无需提前为业务峰值准备资源，按需使用、按量计费，无需为闲置资源付费。

* 完善的微服务中台能力
  提供应用全生命周期管理（支持创建、部署、回滚、扩容、下线、启动和停止应用）、细粒度微服务治理（支持服务路由、服务限流、服务鉴权规则，支持分布式配置管理）、分布式事务等能力。

### 应用场景

* Web服务
  TSF Serverless和其他腾讯云云服务紧密结合，开发者能够轻松构建可弹性扩展的移动或 Web 应用程序、小程序应用、BFF（Backend For Frontend）应用。

![](https://main.qcloudimg.com/raw/dc7f4f7eba7307f3159f75f10dfca5cf.png)

* 分布式系统
  TSF Serverless帮助用户屏蔽底层资源购买和运维细节，低门槛部署微服务应用，让企业聚焦核心业务本身。
