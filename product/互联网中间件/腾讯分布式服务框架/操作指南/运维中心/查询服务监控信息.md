## 操作场景

该任务指导您通过 TSF 控制台，在已部署服务的详情中，查看服务、实例、接口维度的请求监控信息。

## 操作步骤

### 查看服务监控信息

1. 登录 [TSF 控制台](https://console.cloud.tencent.com/tsf/index)。
2. 在左侧菜单栏中，找到【应用中心】分类，单击【服务治理】，进入服务列表页。
3. 选择您感兴趣的服务，单击其 ID，进入该服务详情页。
   ![](https://main.qcloudimg.com/raw/7e90a4900c66f301c126ee4a0fc1cf52.png)
4. 您可在【服务概览】tab页中，查看到
   - 该服务的相关的依赖拓扑图（服务及其上下游服务）
     ![](https://main.qcloudimg.com/raw/ce878d040b8bc0fa6674bbfb18f12adc.png)
   - 服务的请求监控信息
     - 请求量
     - 错误率
     - 响应耗时
     - HTTP 状态码
       ![](https://main.qcloudimg.com/raw/b081c2e2339222cfc39fa2f87a195b6a.png)

### 查看实例监控信息

1. 进入【服务实例列表】页面。您可通过以下任一种方式，进入服务实例列表页

- 方式一：【[部署组](https://console.cloud.tencent.com/tsf/group)】>【部署组详情】>【服务实例列表】
  ![](https://main.qcloudimg.com/raw/aa79d63cdc9992a7b022ff99cf379ba0.png)
  ![](https://main.qcloudimg.com/raw/2fa070218362d122b913740d36456674.png)
- 方式二：【[服务治理](https://console.cloud.tencent.com/tsf/service)】>【服务详情】>【服务实例列表】
  ![](https://main.qcloudimg.com/raw/b6c92349bd53c0c2ba5b6ee35aedcdd9.png)

2. 选择您感兴趣的实例，单击其 ID，进入该实例的详情页面。

#### 实例监控概览

您可在实例详情中的【实例概览】tab 中，查看到该实例的以下信息：

- 基本信息
- 经过该实例的请求监控信息
- 该实例的 JVM 监控概览（如果该实例为 Java 实例）

![](https://main.qcloudimg.com/raw/3f2b2a1f39a14feff8cd5f75cf3b342d.png)
![](https://main.qcloudimg.com/raw/0f018976ddcadfe74a5bea116a3a3efe.png)

#### 内存

如果实例为 Java 实例，您可在实例详情的【内存】tab 页中，查看到该实例的内存监控详情：
![](https://main.qcloudimg.com/raw/4336c4e415aad5835e6dc65817035a26.png)
![](https://main.qcloudimg.com/raw/3bce78368b01aaf84b462092cc201033.png)
![](https://main.qcloudimg.com/raw/74d95e9efc9ec688be1320f605f986b0.png)

#### 线程

如果实例为 Java 实例，您可在实例详情的【线程】tab 页中，查看到该实例的线程数变化情况、线程列表及线程详情，并进行死锁检测，了解实例的死锁情况：
![](https://main.qcloudimg.com/raw/38c1d0f9224c21111cffdd239943f6d9.png)
![](https://main.qcloudimg.com/raw/23069bfe9da2e4e1301f10b98ef44d87.png)

#### 火焰图

如果实例为 Java 实例，您还可以在【火焰图】tab 页中，采集并查看指定时段内程序执行过程所形成的“火焰图”，用以进行性能分析。
![](https://main.qcloudimg.com/raw/ce25295b4d755538f954affe190c1bde.png)

#### 请求详情

您可在【请求详情】tab页中，查询经过该实例的请求的调用链详情。

> ?列表页中的“耗时”和“状态”，为服务处理请求的本地耗时和状态。

![](https://main.qcloudimg.com/raw/d76892052da10b36bd85ba960e9a79a4.png)

#### 日志

您可在【日志】tab页中，查看该实例上的日志，包括业务日志及 JVM 的 GC 日志。
![](https://main.qcloudimg.com/raw/c01359796f4bba48c081d6e2a613ec57.png)

### 查看接口监控信息

1. 单击【[服务治理](https://console.cloud.tencent.com/tsf/service)】>【服务详情】>【接口列表】，进入接口列表页。
2. 选择您感兴趣的接口，单击其路径，进入该接口的详情页。
   ![](https://main.qcloudimg.com/raw/86d0449541296af81e57966eeb33af29.png)

#### 接口监控概览

您可在实例详情中的【实例概览】tab 中，查看到该接口的请求监控信息。
![](https://main.qcloudimg.com/raw/7ac8fe9b07269e975e1f7138dc8aaecc.png)
![](https://main.qcloudimg.com/raw/c306a8cacb8f86d3009207980b5ba5a3.png)

#### 请求详情

您可在【请求详情】tab页中，查询经过该接口的请求的调用链详情。

> ?列表页中的“耗时”和“状态”，为服务处理请求的本地耗时和状态。

![](https://main.qcloudimg.com/raw/0be6bbead78db9414d31641d7c79042d.png)
