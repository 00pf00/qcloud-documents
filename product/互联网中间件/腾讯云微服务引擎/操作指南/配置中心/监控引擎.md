## 操作场景

该任务指导您通过 TSE 控制台查看Apollo配置中心的监控指标。


## 操作步骤

1. 登录 [TSE 控制台](https://console.cloud.tencent.com/tse)。
2. 在左侧导航栏单击【配置中心】，单击目标实例的“ID/名称”，进入【运行监控】页面，即可查看监控指标信息。
3. 可查看的监控指标如下所示：

| 指标类型 | 指标名                            |
| -------- | --------------------------------- |
| 业务指标 | 客户端长轮询请求数                |
| 业务指标 | 获取配置失败数                    |
| 业务指标 | 获取配置成功数                    |
| 业务指标 | 客户端获取服务地址失败数          |
| 业务指标 | 客户端获取服务地址成功数          |
| 系统指标 | ConfigService进程打开的文件句柄数 |
| 系统指标 | ConfigService CPU 使用率          |
| 系统指标 | ConfigService Load1 负载          |
| JVM指标  | JVM 老年代可用内存                |
| JVM指标  | JVM 老年代已用内存                |
| JVM指标  | JVM Old GC 数                     |
| JVM指标  | JVM Old GC 耗时总数               |
| JVM指标  | JVM 活跃线程数                    |
| JVM指标  | JVM Young GC 数                   |
| JVM指标  | JVM Young GC 耗时数               |
