
弹性防护峰值指大禹高防服务可提供抵御攻击流量的能力范围。若攻击流量超过最高防护峰值，则被攻击 IP 将触发封堵。

## 前提条件
设置防护对象 IP，您需要成功 [购买 DDoS 高防 IP ](https://cloud.tencent.com/document/product/1021/31479) 。

## 操作步骤
1. 登录 [DDoS 高防 IP 管理控制台](https://console.cloud.tencent.com/antiddos-native/package)，在左侧导航中，选择【高防 IP 】。
2. 单击目标 DDoS 高防 IP 实例所在行的防护规格的弹性防护后面的铅笔图标。
![](https://main.qcloudimg.com/raw/dcaf67a5ee130a4823bff1b1ea0702df.png)
3. 在【设置弹性防护】对话框，根据实际防护需求选择弹性防护峰值。
![](https://main.qcloudimg.com/raw/08dcb5f593e30e51c430fbebeb37825e.png)
4. 单击【确定】。
