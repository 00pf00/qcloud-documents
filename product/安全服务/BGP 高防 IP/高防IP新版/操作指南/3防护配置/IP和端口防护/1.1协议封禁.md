

配置高防IP的协议封禁

## 前提条件
您需要成功 [购买 DDoS 高防IP](https://cloud.tencent.com/document/product/1021/31479) ，并设置防护对象。

## 操作步骤
1. 登录[DDoS 高防IP管理控制台](https://console.cloud.tencent.com/ddos/antiddos-advanced/package) ，在左侧导航中，选择【防护配置】。
2. 在左边的列表选中高防IP的ID或端口，如"212.64.63.71 bgpip-000002je" 或 "212.64.63.71 bgpip-000002je"-> "tcp:8000"。
![](https://main.qcloudimg.com/raw/acb5ef2fffe8c4efe1c9dbfcee9705fd.png)
3. 右侧卡片中选择【协议封禁】卡片的【设置】，进入协议封禁列表。
![](https://main.qcloudimg.com/raw/8d20c3c4bdab3119faa6fcb4952cca38.png)
4. 点击【新建】按钮，创建协议封禁规则
![](https://main.qcloudimg.com/raw/ff395f0e5b5c2079a1cc6491a5eb9d14.png)
5. 新建完成后协议封禁列表多了一条协议封禁规则，可以点击操作列的【配置】按钮，可以修改协议封禁规则。
![](https://main.qcloudimg.com/raw/148b19f21a39db46cf8f06cd28008ae6.png)
