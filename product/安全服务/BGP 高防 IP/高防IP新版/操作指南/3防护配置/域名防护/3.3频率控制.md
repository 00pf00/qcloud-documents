
配置高防IP的频率控制

## 前提条件
您需要成功 [购买 DDoS 高防IP](https://cloud.tencent.com/document/product/1021/31479) ，并设置防护对象。

## 操作步骤
1. 登录 [DDoS 高防IP管理控制台](https://console.cloud.tencent.com/ddos/antiddos-advanced/package) ，在左侧导航中，选择【防护配置】。
2. 在左边的列表选中高防IP的ID，如"bgp-000000co"，在右边的顶部选中【域名防护】。
![](https://main.qcloudimg.com/raw/a433e85210749b132e523c2b7f1e95e8.png)
3. 右侧卡片中选择【频率控制】卡片中的【设置】，进入频率控制规则列表。
![](https://main.qcloudimg.com/raw/651934af145ba171f396dcce20a3701b.png)
4. 点击【新建】按钮，创建频率控制规则，点击开关按钮开启AI。
![](https://main.qcloudimg.com/raw/4e09d4d9f59f9175159c04e311eafc7d.png)
5. 新建完成后频率控制列表多了一条频率控制规则，可以点击操作列的【配置】按钮，可以修改频率控制规则。
![](https://main.qcloudimg.com/raw/93542cacd5f1bc7e2235ab2a71d916e7.png)
