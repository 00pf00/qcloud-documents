配置高防IP的域名防护CC等级与阈值

## 前提条件
您需要成功 [购买 DDoS 高防IP](https://cloud.tencent.com/document/product/1021/31479) ，并设置防护对象。

## 操作步骤
1. 登录 [DDoS 高防IP管理控制台](https://console.cloud.tencent.com/ddos/antiddos-advanced/package) ，在左侧导航中，选择【防护配置】。
2. 在左边的列表选中高防IP的ID下面的域名，如"212.64.63.71 bgpip-000002je" -> "http:80" -> "www.qq.com"。
![](https://main.qcloudimg.com/raw/a433e85210749b132e523c2b7f1e95e8.png)
3. 右侧卡片中选择【CC防护设置】卡片中设置防护等级，用户可以根据自己的业务选择"严格"、"适中"、"宽松"三种等级和CC防护清洗阈值。
