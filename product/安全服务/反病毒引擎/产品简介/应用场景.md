## 联网场景
- **可疑文件公有云查**
对可疑文件计算 Hash，通过 TAV 公有云查询服务查询文件黑白属性。
- **可疑文件云端扫描**
将可疑文件通过 API 接口上传至云端扫描服务器进行样本扫描，服务器将返回样本黑白属性。
- 可疑文件本地黑白鉴定
可直接使用 TAV 或集成有 TAV SDK 的软件进行本地文件扫描，直接判断样本黑白属性。
- **被感染文件修复**
可直接使用 TAV或集成有 TAV SDK 的软件对被病毒感染过的重要文件进行修复，还原母本文件，保护文件数据。
