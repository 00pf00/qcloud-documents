### 接口描述
根据用户 id 获取用户详细信息。

### 请求路径
```
/scim/api/v2/Users/{id}
```

### HTTP 请求方式
```
GET
```

### 请求参数
| 参数名称     | 必选 | 类型   | 参数类型 | 描述         |
| ------------ | ---- | ------ | -------- | ------------ |
| id           | 是   | string | url      | 用户 id      |
| access_token | 是   | string | header   | access_token |

### 响应状态码
| code | reason | description |
| ---- | ------ | ----------- |
| 200  | ok     | 请求成功    |

### 返回字段

| 参数名称                                                   | 类型          | 描述                                                         |
| ---------------------------------------------------------- | ------------- | ------------------------------------------------------------ |
| schemas                                                    | array，string | scim 的 schema，返回字符串数组固定值"urn:ietf:params:scim:schemas:core:2.0:User","urn:ietf:params:scim:schemas:extension:enterprise:2.0:User"。 |
| id                                                         | string        | 用户 id                                                      |
| userName                                                   | string        | 用户名                                                       |
| emails                                                     | array,object  | 用户邮箱                                                     |
| - value                                                    | string        | 邮箱                                                         |
| phoneNumbers                                               | array,object  | 用户电话                                                     |
| - value                                                    | string        | 电话，格式：+86-1xxxxxxxxxx                                  |
| urn:ietf:params:scim:schemas:extension:enterprise:2.0:User | object        | 其他属性（包括自定义属性）                                   |
| meta                                                       | object        | scim 的标准，用户的元数据。                                  |
| -  resourceType                                            | string        | 资源类型，固定值"User"                                       |
| - created                                                  | string        | 创建时间，格式为 "2010-01-23T04:56:22Z"                      |
| -  lastModified                                            | string        | 最后修改时间，格式为  "2010-01-23T04:56:22Z"                 |
| - version                                                  | string        | 版本号                                                       |
