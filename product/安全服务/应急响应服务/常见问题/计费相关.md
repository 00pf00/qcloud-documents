### 如何购买应急响应服务？
进入 [购买应急响应服务购买页]() ，根据当前受灾资产数量，选择输入响应的资产数字，单击**立即支付**即可。

### 应急响应服务收费标准是？
应急响应服务 CIRS 收费标准，主要根据受影响的资产数量来测算。服务详细价格请参考如下表格：

| **受影响及其数量（台）** | **总体价格（元/次）**                                        |
| ------------------------ | ------------------------------------------------------------ |
| 1-20                     | 20,000                                                       |
| 21-50                    | 50,000                                                       |
| 51-100                   | 80,000                                                       |
| 101-200                  | 100,000                                                      |
| 201-500                  | 150,000                                                      |
| >500                     | 请 [联系我们](https://cloud.tencent.com/online-service) 进行咨询 |
