## 操作步骤
如果您需要在业务系统中配置“呼出”按钮并呼出电话，您可在业务系统呼出功能中调用 ccc-sdk，调用云呼叫中心呼出拨号盘。调用方法如下所示：
```
window.parent.postMessage({
      source: 'ccc-sdk',
      payload: {
        event: 'callOut',
        message: '15889xxxxxx'
      }
    }, '*')
```
后续我们将支持呼出时获取通信相关参数，届时您可以将呼出时通信相关参数用于您的业务系统。
