# 云工作站

云工作站将分**新建**和**使用**两部分做快速入门使用介绍。

## 新建云工作站

**新建云工作站**在用户完成软件选择、节点配置和确认提交三个步骤后，会创建云上仿真工作站供客户使用。

1.  进入[仿真云控制台](https://console.cloud.tencent.com/cloudsim/desktop)页面，进入【云工作站】，点击【新建】按钮；

![新建云工作站](../images/30-0030.png)

2. 按照页面内容提示，选择所需软件，并在右侧的**版本**中选择所需的软件版本；

![软件版本](../images/30-0031.png)

3. 在“节点配置”页面，定义云工作站名称和标签；

   ![节点配置](../images/30-0032.png)

4. 选择合适的机型、系统镜像和系统盘类型和容量；

   ![节点配置](../images/30-0033.png)

5. 按照页面内容提示，选择或配置公网IP、公网带宽、登陆方式、私有网络、子网、文件系统CFS和挂载点；

   ![节点配置](../images/30-0034.png)

6. 点击【下一步：确认配置信息】，进入“确认配置信息”页面；

7. 检查确认相关信息，勾选“同意《腾讯云服务协议》”，点击【开通】完成创建云工作站。

## 使用云工作站
云工作站创建完毕后，可通过多种方式登录云工作站，这里介绍使用web方式和使用windows远程桌面的方式。

### 使用web方式
1.  进入[仿真云控制台](https://console.cloud.tencent.com/cloudsim/desktop)页面，进入【云工作站】即可看到刚创建好的云工作站实例。
![已建实例](../images/30-0035.png)

2. 点击登录按钮，选择标准登录方式，点击立即登录按钮。
![实例登录](../images/30-0036.png)

3. 在登录页面输入端口号、用户名、密码等相关信息即可登录。
![标准登录](../images/30-0037.png)

4. 用户名密码校验正确后，即可通过网页进入实例桌面，实例已经安装预选软件。
![实例桌面](../images/30-0038.png)


### 使用Windows远程桌面

1.  在本地 Windows 计算机上，按下windows+r键，启动命令，然后在命令框中输入mstsc，然后点击回车即可启动windows远程桌面连接。
![远程连接](../images/30-0039.png)

2. 在“计算机”后输入 Windows 服务器的公网 IP，单击连接。您可参考[获取公网IP地址](https://cloud.tencent.com/document/product/213/17940)获取服务器公网IP。

3. 在弹出的“Windows 安全” 窗口中，输入实例的管理员帐号和密码，如下图所示：
![Windows安全](../images/30-0040.png)

4. 用户名密码校验正确后，即可通过远程连接进入实例桌面，实例已经安装预选软件。
![实例桌面](../images/30-0038.png)
