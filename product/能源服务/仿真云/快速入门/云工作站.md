云工作站适用于仿真任务需要GPU资源或者高性能主机即可满足需求的业务场景，提供多种高性能单节点主机给用户选择。
对于云工作站的简要介绍，将按**新建**和**使用**两部分进行。

## 新建云工作站
**新建云工作站**在用户完成**软件选择**、**节点配置**和**确认提交**三个步骤后，会创建云上仿真工作站供客户使用。
1. 进入[仿真云控制台](https://console.cloud.tencent.com/cloudsim/desktop)页面，进入**云工作站**，单击**新建**。
![](https://qcloudimg.tencent-cloud.cn/raw/cc1888ac7612cf6f4dfe22a96ac9b499.png)
2. 按照页面内容提示，选择所需软件，并在右侧的**版本**中选择所需的软件版本。
![](https://qcloudimg.tencent-cloud.cn/raw/89800fc1939d2c8c6a9b43ae11636012.png)
3. 在**节点配置**页面，定义云工作站名称和标签。
![](https://qcloudimg.tencent-cloud.cn/raw/0b127d569b5cfdf1bc8fa5651329d88d.png)
4. 选择合适的机型、系统镜像和系统盘类型和容量。
![](https://qcloudimg.tencent-cloud.cn/raw/ca86cb70dbf2dd48c809bc60e07c30a2.png)
5. 按照页面内容提示，选择或配置公网IP、公网带宽、登陆方式、私有网络、子网、文件系统 CFS 和挂载点。
![](https://qcloudimg.tencent-cloud.cn/raw/08940dd35209ed7990f4c0a14315f0c5.png)
6. 单击**下一步：确认配置信息**，进入**确认配置信息**页面；
![](https://qcloudimg.tencent-cloud.cn/raw/212e0ea3ea4dac8c08702d46e95072f2.png)
7. 检查确认相关信息，勾选**同意《腾讯云服务协议》**，单击**开通**完成创建云工作站。

## 使用云工作站
云工作站创建完毕后，可通过多种方式登录云工作站，这里介绍使用 Web 方式和使用 Windows 远程桌面的方式。

### 使用web方式
1.  进入[仿真云控制台](https://console.cloud.tencent.com/cloudsim/desktop)页面，进入【**云工作站**】即可看到刚创建好的云工作站实例。
![](https://qcloudimg.tencent-cloud.cn/raw/a6949009dbc734579d9d3930deead849.jpg)
2. 单击**登录**，选择标准登录方式，单击**立即登录**。
![](https://qcloudimg.tencent-cloud.cn/raw/8e59eaf10a910f6921e00404254805a8.png)
3. 在登录页面输入端口号、用户名、密码等相关信息即可登录。
![](https://qcloudimg.tencent-cloud.cn/raw/a689458282b5df3145557fb273c68e99.jpg)
4. 用户名密码校验正确后，即可通过网页进入实例桌面，实例已经安装预选软件。
![](https://qcloudimg.tencent-cloud.cn/raw/7f8ada54be0e187f6e46dc03d4a04d5c.png)


### 使用Windows远程桌面
1.  在本地 Windows 计算机上，按下 Windows + r 键，启动命令，然后在命令框中输入 mstsc，然后单击 Enter 即可启动 Windows 远程桌面连接。
![](https://qcloudimg.tencent-cloud.cn/raw/65e5adab1f7948a4f4d46483af0e9290.png)
2. 在**计算机**后输入 Windows 服务器的公网 IP，单击连接。您可参考[ 获取公网IP地址 ](https://cloud.tencent.com/document/product/213/17940)获取服务器公网 IP。
3. 在弹出的**Windows 安全**窗口中，输入实例的管理员帐号和密码，如下图所示：
![](https://qcloudimg.tencent-cloud.cn/raw/e7044d23363ad36cf5a345632bd04221.png)
4. 用户名密码校验正确后，即可通过远程连接进入实例桌面，实例已经安装预选软件。
![](https://qcloudimg.tencent-cloud.cn/raw/43f98fc42dfad9effc4ff2980a72c52b.png)
