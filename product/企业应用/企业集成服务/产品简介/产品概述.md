企业集成服务（Enterprise Integration Service，EIS）作为一种新型的云集成服务，充分利用云平台固有的动态缩放、容器、资源管理和自助服务等能力，将企业内外部间不同的系统或业务连接到一个统一的的平台中，通过复用集成最佳实践范例、快速搭建系统集成模型实现各个系统间的资源整合、数据编排和传递、业务衔接等功能，满足企业轻量级、全方位、高灵活度的一体化系统集成需求，有效降低企业系统集成周期和难度。

## 产品功能

#### 通信协议连接器

EIS 将会提供业界主流的通信协议连接器，比如 FTP、HTTP、高级消息队列协议（AMQP）和适用性声明1（AS1）/ 2/3/4。
一期产品将提供基于HTTP通信协议的连接器来支持连接数据源系统和数据目的地系统。 

#### SaaS 应用程序连接器

随着 SaaS 应用日益流行，开箱即用的应用程序连接能力也愈加重要。通过 SaaS 应用程序连接器，我们可以将这些不同服务简单的结合在一起。我们在产品一期将发布企业微信、腾讯会议、企点、乐享四大应用连接器供企业使用。 同时我们也开放应用连接器生态，以赋能更多的 SaaS 厂商能基于 SDK 推出自己应用的连接器组件。

#### 数据映射和转换

数据转换是 EIS 最核心的功能，它消除了异构应用之间的技术差异，让不同的应用服务协调运作，实现了不同服务之间的通讯。

数据接入 EIS 平台后，用户可以根据目标系统的数据格式和数据结构要求将输入的数据进行针对性的编排和转换，例如：

- 数据格式转换：从 xml 格式转换成 JSON
- 数据映射：数据结构的转换和字段的映射
- 内容修改：参数的修改、变量的赋值等

#### 消息路由

控制消息的流向有很多不同的场景，数据的路由，数据的拆分和组合，数据的排序，数据的分发等。平台提供多个组件支持响应功能，比如 Choice 组件，类比程序设计中的 if-else 分支逻辑，优先匹配前序的 if 条件判断，如果命中则该条件后的节点进行处理，否则继续匹配后面的 if 条件判断。

#### 集成流程开发和生命周期管理

用户在控制台配置界面通过拖拽组件并设置参数即可完成复杂的集成业务逻辑。此外脚本语言的支持，赋能广大的业务人员和技术人员通过开发模式来满足复杂集成业务场景的需求。通过开发、部署、执行、管理来完成以及集成应用的全生命周期管理。

#### 集成应用调试和运行监控

提供集成应用开发过程中的实时调试能力将有效支持以及运行过程中的监控能力。

#### Dataway 脚本语言  

DataWay 是一种轻量级的表达式语言，作为 EIS 中的表达式求值器，可用于访问和计算消息中的的消息体，参数和变量，通过对消息上的数据进行灵活控制，如过滤、映射、格式转换等，以应对复杂数据转换场景的业务需求。
