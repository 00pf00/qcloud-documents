## 什么是云API?

云 API 是腾讯云开放生态的基石。通过云 API，只需少量的代码即可快速操作云产品；在熟练的情况下，使用云 API 完成一些频繁调用的功能可以极大提高效率；除此之外，通过 API 可以组合功能，实现更高级的功能，易于自动化， 易于远程调用，  兼容性强，对系统要求低。 

## 云API应用场景

**1.功能组合**
当用户有很多重复性操作需要完成时，这时腾讯云 API  就将会是一个很好的选择。例如，在双11，618的时候，某些平台需要大量建立某些服务，结束之后又要批量删除某些服务，这个操作过程是固定的，重复性极高，如果使用传统方法来解决就会浪费大量时间和精力，而腾讯云 API，可以改变这一情况，用户只需要按照云 API  的文档，编写自己的代码，按照自己的需求组合各种功能，可以快速实现类似的操作，有效提高用户的效率，进而创造更多的价值。 
![](https://main.qcloudimg.com/raw/f38eb496a817a5745ae5f0edb1ed5a71.png)
**2.定制需求**
用户可以通过自己自由组合腾讯云 API 的各个功能，实现更高级的功能，进而实现定制化开发。此外，您还可以将这个定制化开发的结果，融合到自己的软件中，来提高自己的工作效率。

## 云API具有哪些特性？

**快速**：腾讯云 API 提供腾讯云产品各类资源的接口，您只需通过云 API 即可快速操作云产品，让您更加方便地管理您的云资源。 
**高效**：腾讯云 API 对系统要求低，且兼容性强，在熟练使用 API 的情况下，您可自行组建 API 完成常用功能的调用，极大的提高了效率。
**灵活**：腾讯云 API 易于自动化，支持远程调用，您可通过云 API 自由组合接口，实现更高级的功能，进而实现功能定制。 

## 如何使用云API

腾讯云提供如下方式进行云 API 的查询和调用：
 -  **API 中心**：您可以通过 [API中心](https://cloud.tencent.com/document/api) 快速了解腾讯云各产品 API 信息并且使用云 API。
 -  **API Explorer**：您可以使用 [API Explorer](https://console.cloud.tencent.com/api/explorer) 工具快速便捷完成 API 调用。
 -  **SDK 中心**：您可以通过 [SDK 中心 ](https://cloud.tencent.com/document/sdk) 使用对应的资源。
