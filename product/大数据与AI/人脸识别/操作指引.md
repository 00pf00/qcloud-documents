### 登录账号
注册并通过实名认证后，您可以登录腾讯云 [人脸识别控制台](https://console.cloud.tencent.com/aiface) 进行使用。如果没有账号，请参考账号 [注册教程](https://cloud.tencent.com/document/product/378/9603)。

### 创建密钥
完成注册后，您需要在 [访问管理](https://console.cloud.tencent.com/cam/capi) 创建密钥。AppID、SecretID 和 SecretKey 是您进行应用开发的唯一凭证，请妥善保管。

### 调用 API

腾讯云 API 文档提供了腾讯云所有云产品的 API 接口，您可以查询云 API 文档来调用和配置各种腾讯云资源。查看人脸识别的 [API文档>]() 

您也可以编写代码调用腾讯云 SDK 来访问腾讯云产品。腾讯云SDK将腾讯云产品的各 API 基于常见的编程语言进行了打包，包括 C++，PHP，Python，Java，JavaScript 和 Node.js 等。查看人脸识别的 [SDK 文档 >]()

您还可以通过简单配置，使用腾讯云命令行工具（Command Line Interface，CLI） 来调用腾讯云 API，管理您的腾讯云资源；
您还可以基于腾讯云 CLI 来做自动化和脚本处理，从而简化一些具有复杂 API 的服务。查看人脸识别 [命令行工具文档>]()

### 查看调用
登录腾讯云 [人脸识别控制台](https://console.cloud.tencent.com/aiface)，支持查看人脸识别的各服务的使用情况。
