### 注册登录
登录腾讯云官网。如果没有账号，请参考 [账号注册教程](h)（超链待提供）。

### 通过 API/SDK/命令行来使用服务
- 我们为您提供了丰富多样的 API 接口，您可以查看 [人脸试妆 API 文档](e) （超链待提供）。
- 建议您使用 [API 3.0 Explorer](e)（超链待提供）工具进行在线测试，该工具提供了在线调用、签名验证、SDK 代码生成和快速检索接口等能力，能显著降低使用云 API 3.0的难度。
- 建议您通过 API 3.0 Explorer 生成服务端 SDK 代码，结合腾讯云的已编写好的开发工具集（SDK），可以快速调用人脸特效服务。SDK 已支持多种语言，包括 Python、Java、PHP、Go、NodeJS、.Net 等，您可以在每个服务的文档中下载对应的 SDK，也可以通过 [SDK 中心](e) （超链待提供）获取。
- 通过 [腾讯云命令行工具](e)（超链待提供）来调用腾讯云 API ，管理您的腾讯云资源；您还可以基于腾讯云 CLI 来做自动化和脚本处理，从而简化一些具有复杂 API 的服务。

### 调用成功
当请求返回结果图片的Base64信息，表示接口调用成功，您可以查看具体的试唇色效果。
