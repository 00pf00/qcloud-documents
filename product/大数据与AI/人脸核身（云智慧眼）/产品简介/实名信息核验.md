## 产品介绍

实名信息核验是对用户身份信息真实性验证的一种基础方式，对姓名、身份证号、手机号、银行卡号等信息进行真实性和一致性的核验。适用于社交娱乐行业，在线游戏实名认证、在线主播、粉丝实名核验等场景。

## 产品功能

**身份信息认证**	

通过调用 [身份信息认证 API ](https://cloud.tencent.com/document/api/1007/33188) 接口，传入姓名和身份证号和权威库信息比对，验证信息真实性和一致性。

**身份证人像照片验真**	

通过调用 [身份证人像照片验真 API ](https://cloud.tencent.com/document/product/1007/47276) 接口，传入身份证人像面照片，识别身份证照片上的信息和权威库信息比对，验证是同一个人。

**身份证识别及信息核验**	

通过调用 [身份证识别及信息核验 API ](https://cloud.tencent.com/document/product/1007/37980) 接口，传入身份证照片和权威库信息比对，校验姓名和身份证号的真实性和一致性。

**银行卡基础信息查询**	

通过调用 [银行卡基础信息查询 API ](https://cloud.tencent.com/document/product/1007/47837) 接口，传入银行卡号查询银行卡基础信息，包括开户行、银行卡性质等

**银行卡二要素核验**	

通过调用 [银行卡二要素核验 API ](https://cloud.tencent.com/document/api/1007/35776) 接口，传入姓名和银行卡号与权威库信息比对，验证真实性和一致性

**银行卡三要素核验**	

通过调用 [银行卡三要素核验 API ](https://cloud.tencent.com/document/api/1007/33848) 接口，传入姓名、开户证件号、银行卡号与权威库信息比对，验证真实性和一致性

**银行卡四要素核验**	

通过调用 [银行卡四要素核验 API ](https://cloud.tencent.com/document/api/1007/35775) 接口，传入姓名、开户证件号、银行卡号、手机号与权威库信息比对，验证真实性和一致性

**手机号在网时长核验**	

通过调用 [手机号在网时长核验 API ](https://cloud.tencent.com/document/product/1007/40546) 接口，传入手机号查询手机号在网时长。

**手机号状态查询**	

通过调用 [ 手机号状态查询 API ](https://cloud.tencent.com/document/product/1007/40545) 接口，传入手机号验证手机号的在网状态。

**手机号三要素核验**	

通过调用 [手机号三要素核验 API ](https://cloud.tencent.com/document/product/1007/39765) 接口，校验手机号、姓名和身份证号的真实性和一致性。

**手机号二要素核验**	

通过调用 [手机号二要素核验 API ](https://cloud.tencent.com/document/product/1007/50364) 接口，校验手机号、姓名的真实性和一致性。



## 产品优势

### 海量训练数据

立足于腾讯社交数据大平台，拥有海量的数据分析和人脸、图片训练集，贴近实际业务，覆盖多样场景。

### 在线核实用户身份

根据用户的自拍照片或视频，结合腾讯优图的 OCR、活体检测和人脸识别技术，核实用户的真实有效身份。



## 应用场景 

产品适用于银行证券、视频直播、在线考试等对姓名、身份号、手机号等信息真实性有要求的实名认证服务场景，支持 API 方式接入使用。
