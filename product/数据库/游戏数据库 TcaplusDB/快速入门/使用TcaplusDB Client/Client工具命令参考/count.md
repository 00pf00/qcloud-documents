## 简介
查询表格的记录总数

## 语法
```
count [tableName];
```

## 错误
参考 [CLI] 常见错误返回

## 例子
获取表格的记录数

```
tcaplus> count game_player;
--------------------------------------------------------------------------------
| TableName                                 Count                              |
--------------------------------------------------------------------------------
| table_list                                16                                 |
--------------------------------------------------------------------------------
```
