移动解析 HTTPDNS 服务按照解析次数进行收费，为后付费按日结算的方式，当天 00:00:00 - 23:59:59 产生的解析次数，将会在第二天 8:00:00 进行计算扣费。

## 免费额度
开通移动解析 HTTPDNS 且服务正常启用中的用户，系统每个月（按30个自然日计算）均提供一定额度的免费流量包，根据用户类型不同提供的额度不同：

| 用户类型 | 免费解析次数额度 |
|---------|---------|
| DNSPod 解析域名 | 600万次/月 |
|非 DNSPod 解析域名 | 300万次/月 |


## 按量计费

### 计算方式
每个自然日以统计当前账户的 HTTP 协议的解析次数进行计算，计算方式如下：
- **1次 HTTPS 协议的解析次数 = 5次 HTTP 协议解析次数**
- **1次 AES 解析次数 = 3次 HTTP 协议解析次数**

使用移动解析 HTTPDNS，本月01日使用 HTTP 协议的解析次数为100万，本月 02 日使用HTTP协议的解析次数为80万、使用 HTTPS 协议的解析次数为20万，则本月01日统计的 HTTP 协议的解析次数数量为100万，本月 02 日统计的 HTTP 协议的解析次数数量为 80万+5*20万=180万。总按量计费费用为 0.04元/万次*（100万+180万）= 11.2元
