# 开通 HTTPDNS

## 概述
私有域解析 Private DNS 为付费型服务类产品，并采取后付费计费方式，需确认协议后开通使用。
本文档将指导您如何开通移动解析 HTTPDNS 。

## 开通移动解析 HTTPDNS 
### 未拥有授权 ID。

1. 登录移动解析 [移动解析 HTTPDNS 管理控制台](https://console.cloud.tencent.com/httpdns)。
2. 在移动解析 HTTPDNS概览页中，单击【开通服务】。如下图所示：
![](https://main.qcloudimg.com/raw/c533bcb55d13c452e4dd3bcf82cc0255.png)
3. 在弹出的开通服务窗口中，勾选【同样移动解析服务协议】。如下图所示：
![](https://main.qcloudimg.com/raw/8e142d1898c77958f007dff3502e472b.png)

4. 单击【开通服务】，即可完成开通移动解析 HTTPDNS 操作。

>? 完成开通操作后，系统将为您颁发授权ID和密钥信息。您可以在移动解析 HTTPDNS 管理控制台进行查看。

### 已拥有授权 ID。
>?如果您拥有通过线下方式申请的授权ID和密钥，可以填写相关信息进行验证，验证通过可选择将授权ID与当前账号进行绑定。

1. 登录移动解析 [移动解析 HTTPDNS 管理控制台](https://console.cloud.tencent.com/httpdns)。
2. 在移动解析 HTTPDNS概览页中，单击【开通服务】。如下图所示：
![](https://main.qcloudimg.com/raw/c533bcb55d13c452e4dd3bcf82cc0255.png)
3. 在弹出的开通服务窗口中，单击【点此绑定】。如下图所示：
![](https://main.qcloudimg.com/raw/91b09a7c60a5ae94a75474b5fb5e20ec.png)

3. 在弹出的绑定ID开通服务窗口中，填写企业ID和密钥信息。如下图所示：
![](https://main.qcloudimg.com/raw/ec62743ee4e6d5c5a7a6e7106d04fd2a.png)

4. 单击【验证】，即可完成操作。
