## 概述

本文档将指导您开通移动解析 HTTPDNS 服务后，如何在控制台提交接入 SDK 的申请。


## 前提条件
- 已 [开通移动解析 HTTPDNS ](1) 服务。
## 操作指南
1. 登录 [移动解析 HTTPDNS 管理控制台](https://console.cloud.tencent.com/httpdns)。
2. 在移动解析 HTTPDNS管理控制台中，选择【开发配置】。
3. 在开发配置页，单击【申请应用】。
![](https://main.qcloudimg.com/raw/588228555e38e82e2da6f57577e87a61.png)
4. 在弹出的申请应用 (HTTPDNS SDK) 窗口中，输入应用名称等信息。如下图所示:
![](https://main.qcloudimg.com/raw/2f5720d9a934a1670226cdddacca8f81.png)
应用名称：输入您的应用名称。
描述 (选填)：输入您的应用描述，可选。
业务类型：请根据您的具体业务情况进行选择。

7. 单击【保存】，即可成功申请。您的腾讯云安全邮箱将会收到接入 SDK 必须的 APPID（Android 和 iOS 各一个）。如下图所示：
![](https://main.qcloudimg.com/raw/3b7d8079e97a0d5e492ad60831a5c7ea.png)

或者您可以通过在【开发配置】页进行查看。如下图所示:
![](https://main.qcloudimg.com/raw/36d81973c2d57a0b0b09b5e7249ecbe6.png)
