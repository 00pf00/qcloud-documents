## 模板简介

本模板是应用于汽车服务的小程序模板，包含了服务分类，中心介绍，联系我们，热门服务，知识问答等多种功能。

### 功能模块

| 模块     | 功能名称                                    | 详细功能清单                                                 |
| -------- | ------------------------------------------- | ------------------------------------------------------------ |
| 管理后台 | 产品                                        | 产品新增/查看/编辑/删除  产品详情                            |
| 服务类型 | 服务类型新增/查看/编辑/删除                 |                                                              |
| 咨询     | 咨询详情                                    |                                                              |
| 环境     | 环境新增/查看/编辑/删除                     |                                                              |
| 特色服务 | 特色服务新增/查看/编辑/删除  特色服务详情   |                                                              |
| 介绍     | 店铺介绍、地址、联系方式新增/查看/编辑/删除 |                                                              |
| 问答     | 问答新增/查看/编辑/删除问答详情             |                                                              |
| 小程序端 | 首页                                        | 服务分类                  精品服务                    热门服务                  知识问答 |
| 中心介绍 | 中心介绍服务环境服务特色                    |                                                              |
| 联系我们 | 咨询 联系方式                               |                                                              |

### 第一步：购买和试用小程序

[微搭应用市场](https://weda.cloud.tencent.com/)选购小程序，点击【试用7天】或【立即购买】下单，下单后跳到模版管理安装模版。

![img](https://main.qcloudimg.com/raw/941652ea8ab7c3dfc41e390410d4f797.jpeg) 

### 第二步：安装模版

在模版列表点击安装，并填写自定义名称，生成可拖拽的店铺应用。

![img](https://main.qcloudimg.com/raw/48bf278dd64e092578109a2ebde8d855.jpeg) 

模版安装的应用通过【应用管理】，进入应用详情页

![img](https://main.qcloudimg.com/raw/6032e025ddfb8ca9fea1884f91b9fe9c.jpeg) 

### 第三步： 小程序店铺装修

您在装修过程中遇到任何问题，欢迎咨询模版提供客服。

装修编辑器整体介绍：

页面管理：可设置所需要的页面；编辑区：可将组件拖拽到编辑区，进行页面布局；预览区：页面实时预览；页面编辑：可设置页面样式

![img](https://main.qcloudimg.com/raw/3b566745c2ad3e2a89779ee500aee375.jpeg) 

组件：官方组件和自定义组件；组件编辑：可设置组件数据绑定、样式、事件等 

![img](https://main.qcloudimg.com/raw/d7814eb62fc2cbb44eae2f64bf4bd4a3.jpeg) 

l 【页面管理】支持切换页面，对所有页面进行个性化装修

l 【编辑区】点击选中要编辑的组件

l 【预览区】可查看页面效果

l 【组建编辑】可按需修改您的个性化内容，包含数据（静态文本、图片）、样式、事件

l 【组件】模版所带组件库，可用于新页面开发
