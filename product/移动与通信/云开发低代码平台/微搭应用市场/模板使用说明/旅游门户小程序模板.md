## 模板简介

本模板是应用于旅游门户的小程序模板，包含了公司介绍，旅游服务介绍，企业新闻等多种功能。

### 功能模块

| 模块       | 功能名称                                    | 详细功能清单                                                 |
| ---------- | ------------------------------------------- | ------------------------------------------------------------ |
| 管理后台   | 新闻                                        | 新闻新增/查看/编辑/删除     新闻详情                         |
| 新闻模块   | 新闻模块新增/查看/编辑/删除                 |                                                              |
| 旅游服务   | 旅游服务新增/查看/编辑/删除    旅游服务详情 |                                                              |
| 集团介绍   | 集团介绍新增/查看/编辑/删除    集团介绍详情 |                                                              |
| 首页轮播图 | 轮播图新增/查看/编辑/删除                   |                                                              |
| 小程序端   | 首页                                        | 轮播图                  公司简介                    旅游服务                  新闻动态 |
| 旅游服务   | 旅游服务列表             旅游服务详情       |                                                              |
| 集团新闻   | 集团新闻列表             集团新闻详情       |                                                              |
| 集团简介   | 集团简介详情                                |                                                              |

### 第一步：购买和试用小程序

[微搭应用市场](https://weda.cloud.tencent.com/)选购小程序，点击【试用7天】或【立即购买】下单，下单后跳到模版管理安装模版。

![img](https://main.qcloudimg.com/raw/400664d19408a1a00dfe96e7fbad37c5.jpeg) 

### 第二步：安装模版

在模版列表点击安装，并填写自定义名称，生成可拖拽的店铺应用。

![img](https://main.qcloudimg.com/raw/2be90e728b78ec8bb0f43502163a7c8d.jpeg) 

模版安装的应用通过【应用管理】，进入应用详情页

![img](https://main.qcloudimg.com/raw/9e783d2fb39efca23148da0b7fee85e3.jpeg) 

### 第三步： 小程序店铺装修

您在装修过程中遇到任何问题，欢迎咨询模版提供客服。

装修编辑器整体介绍：

页面管理：可设置所需要的页面；编辑区：可将组件拖拽到编辑区，进行页面布局；预览区：页面实时预览；页面编辑：可设置页面样式

![img](https://main.qcloudimg.com/raw/8c0d5c22f56e1cc40e175192935fa8d5.jpeg) 

组件：官方组件和自定义组件；组件编辑：可设置组件数据绑定、样式、事件等   

![img](https://main.qcloudimg.com/raw/b302ba7683c2f5e42e8cad5316a3fed8.jpeg) 

l 【页面管理】支持切换页面，对所有页面进行个性化装修

l 【编辑区】点击选中要编辑的组件

l 【预览区】可查看页面效果

l 【组建编辑】可按需修改您的个性化内容，包含数据（静态文本、图片）、样式、事件

l 【组件】模版所带组件库，可用于新页面开发

### 第四步： 创建模型应用

![img](https://main.qcloudimg.com/raw/d66acd07cbdffa44ca144c79f4f4c0ce.jpeg) 

点击“进入应用页面编辑”，进入编辑页面

![img](https://main.qcloudimg.com/raw/cd36e5dadece28951ebaac4394dd15e1.jpeg) 

点击“发布”，选择发布方式，点击下一步  

![img](https://main.qcloudimg.com/raw/99a3cb7903a9445315521824fddb9c35.jpeg)
