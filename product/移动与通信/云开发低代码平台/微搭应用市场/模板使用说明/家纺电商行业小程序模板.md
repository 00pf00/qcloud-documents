## 模板简介

本模板是应用于家纺行业的小程序模板，包含了商品介绍，商品购买，限时折扣，订单管理等多种功能。

### **功能模块**

| 模块         | 功能名称                                                     | 详细功能清单                                                 |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 管理后台     | 账户资产                                                     | 账户绑定提现/提现记录订单流水                                |
| 分类管理     | 新增分类编辑分类删除分类关联商品                             |                                                              |
| 交易统计     | 账户绑定提现/提现记录订单流水                                |                                                              |
| 商品管理     | 商品新增/编辑/删除商品上下架商品批量修改售价/原价/库存       |                                                              |
| 订单管理     | 订单列表/详情订单改价订单发货查看订单支付信息/顾客信息/配送信息/附加信息/优惠信息 |                                                              |
| 运费模板     | 新增运费模板编辑运费模板删除运费模板设置发货地址             |                                                              |
| 图文介绍     | 图文介绍新增/查看/编辑/删除                                  |                                                              |
| 商品轮播图   | 轮播图新增/查看/编辑/删除                                    |                                                              |
| 店铺信息     | 店铺信息新增/查看/编辑/删除                                  |                                                              |

### **第一步：购买和试用小程序**

[微搭应用市场](https://weda.cloud.tencent.com/)选购小程序，点击【试用7天】或【立即购买】下单，下单后跳到模版管理安装模版。

​                 ![img](https://main.qcloudimg.com/raw/0679793f2233af892c2bc8d684ad461a.png)        

### 步骤2：安装模版

1. 在模版列表单击**安装**，并填写自定义名称，生成可拖拽的店铺应用。 ![img](https://main.qcloudimg.com/raw/339651040fc616c5c0c1f06e28a7289d.png)

2. 模版应用安装后，会生成2个应用，一个是小程序，一个是电商管理后台。

   ​                 ![img](https://main.qcloudimg.com/raw/dc86172bb71264f27d0d9bf5bdf7b81c.png)

### 步骤4： 数据管理

#### **修改首页轮播图**

​                 ![img](https://main.qcloudimg.com/raw/2d6241806132c5ae366d9bfc1ea471f8.png)        



1. 进入**应用管理**页面，单击模板右下角二维码，单击**企业工作台访问链接**，进入店铺管理后台。

   ![img](https://main.qcloudimg.com/raw/3e5d310ce399b6d0fa704f078898484b.png)        

   

2. 输入已创建的用户名与密码，进行登录。详情请参见 [用户权限管理](https://cloud.tencent.com/document/product/1301/59396)。

   ![img](https://main.qcloudimg.com/raw/4fc6776ee1fbe3f43dc9cb53ed0b356d.png)

3. 进入**数据管理后台**，找到对应的数据管理页面。 ，可新增轮播图，编辑轮播图，删除轮播图。

​                 ![img](https://main.qcloudimg.com/raw/f4f8aa98daa89859dd767dab89fe41c9.png)        

4. 单击**编辑**，进入编辑页面。修改相应信息后单击**提交**，完成修改。

​                 ![img](https://main.qcloudimg.com/raw/4e7af381c9b309c80678e099e194b43c.png)        

> 字段说明
>
> - 图片路径：可删除图片，上传新图片。
> - 轮播图位置布局：A、如果轮播图显示在页面顶部位置，设置为top。B、如果轮播图显示在页面底部位置，设置为bottom。
> - 跳转小商店path：为商品的详情页路径地址。

#### **修改信息**

​                 ![img](https://main.qcloudimg.com/raw/0bb34b59302a82d4bcda6b6ba9ed1baf.png)        

找到对应的数据管理页面，点击编辑，打开修改页面，输入新的营业时间，地址，电话，店铺名称，保存即可。

​                 ![img](https://main.qcloudimg.com/raw/59af32b482485ddb6aa2e7cee8fc0990.png)        

​                 ![img](https://main.qcloudimg.com/raw/c5345eaebfaa56bf325bb7ea2f5ea016.png)        

#### **首页横排栏目设置**

​                 ![img](https://main.qcloudimg.com/raw/3be0671fce0ee1f3e7889e896d825bb4.png)        

找到对应的数据管理页面，可新增，编辑，删除栏目

​                 ![img](https://main.qcloudimg.com/raw/a6c7a6342d2228902b4a19d4bcc1d2e1.png)        

点击【编辑】，然后在打开的页面中，修改栏目图片，跳转类型，分类名等。

​                 ![img](https://main.qcloudimg.com/raw/246b4b690f741796a63a3bb0e0c1e255.png)                

> 字段说明
>
> - 页面链接：A、如果跳转到分类页，设置为classificationB、如果跳转到商品详情页面，设置为商品的path。
> - 图片地址：可删除图片，上传新图片。
> - 分类名：为商品所属的分类名。
> - 跳转类型：A、如果跳转到分类页，设置为page。B、如果跳转到商品详情页面，设置为detail。

#### **设置限时折扣栏**

​                 ![img](https://main.qcloudimg.com/raw/38c4dd9e7de9408af315db4e9bd787e4.png)        

找到对应的数据管理页面，点击【编辑】，然后在打开的页面中，修改商品名称，商品地址，商品展示图片。

​                 ![img](https://main.qcloudimg.com/raw/791908e1ca4a82882659ec4e76f4ca0e.png)        

​                 ![img](https://main.qcloudimg.com/raw/9904c0d3ec1581601eca1e7aa9378f81.png)        



> 字段说明
>
> - 商品地址：如果跳转到商品详情页面，设置为商品的path。
> - 图片：可删除图片，上传新图片。
> - 商品名：商品名称。
