## 模板简介

本模板是应用于美容美发行业的小程序模板，包含了产品中心，产品服务，企业资讯，关于我们，在线留言等多种功能。

### 功能模块

| 模块         | 功能名称                                | 详细功能清单                               |
| ------------ | --------------------------------------- | ------------------------------------------ |
| 管理后台     | 关于我们                                | 店铺信息新增/查看/编辑/删除                |
| 产品中心信息 | 产品中心信息新增/查看/编辑/删除         |                                            |
| 企业资讯     | 企业资讯新增/查看/编辑/删除企业资讯详情 |                                            |
| 成功案例     | 成功案例新增/查看/编辑/删除成功案例详情 |                                            |
| 咨询         | 咨询详情                                |                                            |
| 产品         | 产品新增/查看/编辑/删除产品详情         |                                            |
| 小程序端     | 首页                                    | 轮播图产品中心信息成功案例品牌介绍企业资讯 |
| 产品服务     | 产品列表产品详情                        |                                            |
| 企业资讯     | 资讯列表资讯详情                        |                                            |
| 关于我们     | 介绍，联系地址、方式                    |                                            |

### 第一步：购买和试用小程序

[微搭应用市场](https://weda.cloud.tencent.com/)选购小程序，点击【试用7天】或【立即购买】下单，下单后跳到模版管理安装模版。

![img](https://main.qcloudimg.com/raw/b2d91f82750eb773ac6acb6a320938df.jpeg) 

### 第二步：安装模版

在模版列表点击安装，并填写自定义名称，生成可拖拽的店铺应用。

![img](https://main.qcloudimg.com/raw/f8baed84a2a399bacb54d62b4b188ead.jpeg) 

模版安装的应用通过【应用管理】，进入应用详情页

![img](https://main.qcloudimg.com/raw/12bab9b0c7e7425eccac513d60e4d2cc.jpeg) 

### 第三步： 小程序店铺装修

您在装修过程中遇到任何问题，欢迎咨询模版提供客服。

装修编辑器整体介绍：

页面管理：可设置所需要的页面；编辑区：可将组件拖拽到编辑区，进行页面布局；预览区：页面实时预览；页面编辑：可设置页面样式

![img](https://main.qcloudimg.com/raw/4702ebbe5fa270650945b44bcc7d30da.jpeg) 

组件：官方组件和自定义组件；组件编辑：可设置组件数据绑定、样式、事件等   

![img](https://main.qcloudimg.com/raw/44b785fad82b2b554332ff832a94f5c5.jpeg) 

l 【页面管理】支持切换页面，对所有页面进行个性化装修

l 【编辑区】点击选中要编辑的组件

l 【预览区】可查看页面效果

l 【组建编辑】可按需修改您的个性化内容，包含数据（静态文本、图片）、样式、事件

l 【组件】模版所带组件库，可用于新页面开发

### 第四步： 创建模型应用

点击“创建空白应用”，选择“模型应用”，点击下一步  

![img](https://main.qcloudimg.com/raw/26edb3fec7b2db7c98a199b64adca125.jpeg) 

点击“进入应用页面编辑”，进入编辑页面

![img](https://main.qcloudimg.com/raw/1349f581f6983b42bf2198f84d3dbc85.jpeg) 

点击“发布”，选择发布方式，点击下一步  

![img](https://main.qcloudimg.com/raw/da5b52481a85adbef41a534697a2e727.jpeg) 

发布完成后，点击"访问链接"到管理后台

![img](https://main.qcloudimg.com/raw/e33ebcd25330af9ac719bcb6dda95443.jpeg)
