腾讯云微搭低代码 WeDa 是高效、高性能的拖拽式低代码开发平台，向上连接前端的行业业务，向下连接云计算的海量能力，助力企业垂直上云。微搭将繁琐的底层架构和基础设施抽象化为图形界面，通过行业化模板、拖放式组件和可视化配置快速构建多端应用（小程序、H5应用、PC Web 应用等），免去了代码编写工作，让您能够完全专注于业务场景。微搭以云开发作为底层支撑，云原生能力将应用搭建的全链路打通，提供高度开放的开发环境，且时刻为您的应用保驾护航。  


## 平台组成

腾讯云微搭低代码平台由控制台、编辑器、数据管理三部分组成。其中控制台由应用管理、模板中心、数据源管理和组件库管理组成，编辑器由主面板、左侧面板、右侧面板和菜单栏组成，数据管理由数据源管理、变量管理、CMS 内容管理工具组成。

微搭控制台用于管理平台物料，组成如下：

| 控制台           | 描述                                       |
| ---------------- | ------------------------------------------ |
| 应用管理     | 多应用一站式管理。支持多端、多版本的应用管理，支持多人协作。 |
| 模板中心           | 轻松利用模板创建应用。支持模板开发、发布、管理和交易。 |
| 数据源管理       | 管理数据源和应用后台数据。支持内部数据和外部数据源。 |
| 组件库管理           | 支持组件库开发、发布、版本管理和交易。 |

微搭编辑器用于可视化拖拽开发，组成如下：

| 可视化编辑器     | 描述                                                         |
| -------- | ------------------------------------------------------------ |
| 主面板   | 拖拽开发主面板区，具有编辑区和实时预览区；可切换应用端。             |
| 左侧面板 | 集成了页面管理、组件管理、资源管理和 JSON 管理能力。         |
| 右侧面板 | 可对组件进行相关配置，或对整个页面进行配置。                                 |
| 菜单栏   | 实现保存应用、预览发布应用、代码编辑、变量管理和撤销操作等功能。 |

微搭提供了完整的数据链路，用于创建、管理应用产生的动态数据内容，组成如下：

| 数据管理           | 描述                                           |
| --------------- | ---------------------------------------------- |
| 数据源管理      |可创建、管理内外部数据源。数据源模型可用于创建数据源变量。        |
| 变量管理        | 变量用于连接组件和数据，可实现页面内容绑定动态数据的效果。支持多种类型的变量。 |
| CMS内容管理工具 | 应用部署后，可在 CMS 上管理应用变量，实现动态更新应用展示内容的效果。               |


## 多端支持

微搭低代码可用于多端下的开发，拖拽一份页面，可自适部署到多端。详情如下：

| 多端支持        | 描述                                                         |
| ----------- | ------------------------------------------------------------ |
| 小程序      | 微信小程序是运行在微信客户端的应用。一个微搭账号可授权发布同主体下的所有小程序。 |
| H5 应用     | H5 是运行在移动端的网页应用。免注册提供默认域名。 |
| PC Web 应用 | PC Web 是运行在 PC 端的网页应用。免注册提供默认域名。 |
