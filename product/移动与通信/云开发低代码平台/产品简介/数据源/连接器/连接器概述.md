腾讯云微搭低代码最新上线连接器能力，可以打通外部应用数据或调用外部应用开放的服务，例如：腾讯文档、腾讯会议、腾讯地图等。微搭开发者只需要对连接器完成授权或少量填写一些配置信息，即可使用对应产品的服务接口，而无需操心这些接口的实现细节。

## 目前支持的连接器

| 连接器 | 相关产品文档 |
| ----- | ---------- |
| [腾讯文档](https://cloud.tencent.com/document/product/1301/68446) | [腾讯文档 - 开放平台](https://docs.qq.com/open/wiki/) |
| [腾讯会议](https://cloud.tencent.com/document/product/1301/68447) | [腾讯会议 - 开放平台](https://meeting.tencent.com/open-api.html) |
| [腾讯地图](https://cloud.tencent.com/document/product/1301/68448) | [腾讯位置服务 WebService API](https://lbs.qq.com/service/webService/webServiceGuide/webServiceOverview) |
