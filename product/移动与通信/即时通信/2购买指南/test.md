## 基础服务资费

即时通信 IM 分为体验版、专业版和旗舰版，创建应用后默认为体验版（免费），您可以根据实际业务需求选择不同的套餐包。


### 套餐包简介
不同套餐包功能差异如下表所示：

| 功能点       | 体验版      | 专业版    | 旗舰版           |
| ----------------- |---------- |----------------- | ------------------- |
| 全球多点覆盖                               | 支持        | 支持           | 支持                    |
| 用户数上限      | 100人       | 无限制                | 无限制                                 |
| 单个用户好友人数上限         | 20个好友   | 3000个好友          | 3000个好友            |
| 单个用户可加入群组数  | 50个群/人   | 500个群/人                | 1000个群/人            |
| 单个群成员数上限（私有群、公开群和聊天室） | 20人/群     | 200人/群        | 私有群：200人/群<br />公开群和聊天室：2000人/群      |
| 可创建音视频聊天室个数    | 10个        | 50个          | 无限制             |
| 免费历史消息存储时长    | 7天         | 7天             | 30天          |
| 免费日活跃用户数（DAU）额度  | 100/月         | 1万/月       | 1万/月           |
| 免费峰值群组数| 10万/月 | 10万/月  |10万/月   |

- **DAU**：单个用户当日登录即时通信 IM 计为1个 DAU，同一用户重复登录时，DAU 不累加。
- **峰值群组数**：单个应用（SDKAppID）下所有用户创建和加入的群组数的总和，以自然月中最高峰值进行结算。
- **音视频聊天室**：仅提供聊天室内发送文字、图片、短语音及其他自定义消息的能力，如需使用 [云直播](https://cloud.tencent.com/document/product/267)、[实时音视频](https://cloud.tencent.com/document/product/647) 等服务，您需要开通对应服务。


### 套餐包价格详情
每个套餐包均是针对单个应用（SDKAppID）生效，如果您有多个应用（SDKAppID）需要升级，您需依次为每个应用（SDKAppID）单独 [购买](https://cloud.tencent.com/document/product/269/32458) 套餐包。
具体的计费详情如下表所示：

| 计费详情         | 体验版      | 专业版         | 旗舰版             |
| ----------- | ---------- | --------------------- | ---------------------- |
| 单价 | 免费      | 999元/月            | 2999元/月            |
| 有效期     | 永久    | 按周期购买<br />有效期为`T`日 - 次月`T-1`日 | 按周期购买<br />有效期为`T`日 - 次月`T-1`日 |

### 套餐外超量费用
套餐外超量费用是指超出套餐包免费额度以外的 DAU 和峰值群组数所需支付的费用，均以该自然月中最高峰值进行结算，采用月结后付费，每月3日扣除上一个自然月所产生的费用。具体的计费详情如下表所示：

| 计费项         | 专业版         | 旗舰版             |
| ----------- |--------------------- | ------------------- |
| DAU   | 1000元/万/月，不足1万的部分按1万计算| 1000元/万/月，不足1万的部分按1万计算 |
|峰值群组数  | 1000元/10万/月，不足10万的部分按10万计算 | 1000元/10万/月，不足10万的部分按10万计算 |


## 增值服务资费
增值服务是指扩展音视频聊天室创建数量上限、单个群成员数上限、单个用户可加入群组数上限等，每一项增值服务都有其独立的计费规则，使用增值服务产生的费用即为增值服务费用。

>?单个用户好友人数上限最大仅支持3000人，不支持付费扩展。


- **扩展单个群群人数上限**、**扩展单人可加入群组数上限**：针对 SDKAppID 内所有的群组或用户生效。
- **历史消息存储时长**：支持文字、图片、短语音、短视频、文件以及自定义消息延长服务，但不同版本的 SDK 支持延长历史消息存储时长的消息类型不同，请参见 [消息存储](https://cloud.tencent.com/document/product/269/3571#MsgType)。您可以在 [控制台](https://console.cloud.tencent.com/im) 自助延长历史消息存储时长，最大支持延长至360天。
- **安全打击服务**：仅支持中文、英文自定义词库，不支持其他语种，不限制过滤次数。
