号码认证目前支持按量计费（月结后付费）的方式。

## 购买方式
号码认证提供两大功能：
- 一键登录
	- 一键登录按照成功返回号码的次数计费，未成功返回号码时不计费
- 本机校验
	- 本机校验按照认证结果计费，如果认证结果是“一致”“不一致”时计费，认证结果是“无法判断”时不计费。

一键登录和本机校验的单价相同，两项能力合并计算使用量级，并按使用量级所对应的固定价格计费。具体单价如下：

| 使用量级x（次/月）     | 单价（元/次） |
| -------------- | ------- |
| x ≤1万          | 0.04    |
| 1万< x ≤5万      | 0.038   |
| 5万< x ≤10万     | 0.035   |
| 10万< x ≤50万    | 0.03    |
| 50万< x ≤500万   | 0.026   |
| 500万< x ≤1000万 | 0.024   |
| x >1000万       | 0.022   |

## 退费说明
已购买的服务暂不支持退费服务。
