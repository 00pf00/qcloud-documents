# 自定义特效

插件支持自定义`alpha`类型特效， 可参考Clip编辑页特效组件`金粉`系列。

它的底层原理基于带透明通道的mp4视频文件，所以需要生成mp4文件才能完成特效的自定义。

你可能会觉得有点复杂，其实非常的简单，因为相关工具已经为你准备好了，你仅仅只需要准备好素材然后按照一下步骤生成mp4视频文件即可。

## 1. 准备好序列帧图片

序列帧素材示例，可参考：

https://github.com/tencentyun/weijian-sdk/tree/master/alpha-tool/caisehudie

## 2. 从仓库克隆alpha工具

工具已为你准备好，使用非常简单。

```
git clone https://github.com/tencentyun/weijian-sdk.git

cd alpha-tool
```

## 3. 素材文件创建

假设需要生成一个`彩色蝴蝶`的特效，在`alphaVideo.py`创建文件夹 `caisehudie`。

<image src="https://cdn-weijian-1258344699.cos.ap-guangzhou.myqcloud.com/others/jietu5.png" width="500">

然后将序列帧图片放入`caisehudie`目录下:

<image src="https://cdn-weijian-1258344699.cos.ap-guangzhou.myqcloud.com/others/jietu2.png" width="200">

注意： 图片名称务必为 000, 001, 002 这样的格式递增。

## 4. 环境安装

#### 1. 安装phthon

https://www.python.org/about/

#### 2. 安装imageMagick

https://imagemagick.org/index.php

#### 3. 安装ffmpeg

https://www.ffmpeg.org/

## 5. 生成alpha视频

安装好环境之后，接下来的事情就非常简单了，执行一条命令就可以生成mp4视频了。

```
python alphaVideo.py --dir ./caisehudie
```

等待一会之后，caisehudie目录下会生成一个output文件，里面就是我们要的mp4文件了。


<image src="https://cdn-weijian-1258344699.cos.ap-guangzhou.myqcloud.com/others/jietu3.png" width="200">

生成好的mp4文件如下：

<image src="https://cdn-weijian-1258344699.cos.ap-guangzhou.myqcloud.com/others/jietu4.png" width="500">

可以看到视频分为左右两部分，左边是彩色，右边是黑白的，它就是我们要的alpha视频文件了。

## 6. 上传cdn

接下来将这个`video.mp4`上传到你的cdn，然后生成一个在线链接就可以用于自定义特效啦。
