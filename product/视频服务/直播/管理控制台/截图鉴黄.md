在视频直播菜单栏内单击【截图鉴黄】即可进入启动页面，单击【编辑】即可启动或关闭截图与鉴黄。
>**注意：**
>截图设置可单独开启使用，但鉴黄设置需开启截图后才能开启，不可单独使用。

## 截图
**开通方式：**
1. 用户可通过 API 对直播频道进行截图，创建截图任务 API 使用方式请阅读 [创建截图任务](https://cloud.tencent.com/document/product/267/4726)。如要及时终止截图任务，可使用终止截图任务 API，具体请参考 [终止截图任务](https://cloud.tencent.com/document/product/267/4727)。
2. 也可在控制台上开启截图任务（推荐）。
![](//mc.qcloudimg.com/static/img/c5a1a995347bca1c46a2693e05a83772/image.png)

>**注意：**
>由于历史原因，对于部分老客户截图后文件会存放于面向对象存储 COS 平台，故用户需在使用截图功能前提前开通 COS 服务并购买相关空间和流量用于存放和下载截图文件。目前腾讯云直播服务已经优化了截图功能，客户无需自己开通 COS 服务，开通直播截图功能即可。

**规格说明：**
1. 截图视频针对直播原始码率截取，频率为每 10 秒一张，该频率固定不可修改；
2. 截图可自定义截图回调地址，保存时长默认为15天，不可更改。
3. 输出格式为 JPG；
4. 在启动任务后预计在直播时间 20 秒后，可获得对应文件；
5. **提醒：**截图后的文件会在 COS 平台产生对应费用，请根据实际情况留意相关成本支出，详细计费方式请参考 [COS 产品介绍](http://cloud.tencent.com/product/cos.html)。

## 鉴黄
在启动截图后可启动鉴黄，在设置中可自定义【鉴黄回调地址】和【操作回调地址】。
![](https://main.qcloudimg.com/raw/ab005d031fc85e30124bfb38862b3018.png)
>**注意：**
>截图与鉴黄为收费功能，启用后，截图功能按每千张￥0.1元收费；鉴黄功能按每千张￥1.3元收费。查看 [收费详细说明](https://cloud.tencent.com/document/product/267/2818#3.2-.E7.9B.B4.E6.92.AD.E6.88.AA.E5.9B.BE.E8.B4.B9.E7.94.A8)
