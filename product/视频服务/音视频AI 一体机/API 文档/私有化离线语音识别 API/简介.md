### 简介
视频云离线语音识别私有化部署接口，通过接口可以对音频文件或者音频链接进行离线语音识别。
### 统一请求包头必填字段
```
Host: videoface.avc.qcloud.com
Accept:application/json;charset=utf-8;
Content-Type:application/json;charset=utf-8;
Content-Length:实际 body 长度;
```

### 概览

|接口名称|	接口功能|
|----|------|
|语音识别请求	|用于创建语音识别请求|
|语音识别结果回调通知	|用于语音识别结果回调通知|
