## 操作场景
成功添加并解析新域名后，您便可以通过新域名访问您的视频资源。但通过 API 接口和控制台获取到的视频 URL，Host 仍然是原始域名`xxx.vod2.myqcloud.com`。您需要登录 [点播控制台](https://console.cloud.tencent.com/video/cdnlog) 将新域名设置为默认域名，才能将 URL 中的 Host 修改成新添加的域名。

## 操作步骤
1. 登录 [点播控制台](https://console.cloud.tencent.com/video/cdnlog)，在左侧菜单栏中选择【分发播放设置】>【域名设置】。
2. 单击新添加的域名所在行的【设为默认域名】。
