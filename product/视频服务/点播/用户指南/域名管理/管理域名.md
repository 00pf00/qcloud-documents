成功添加并解析域名后，您便可以通过新域名访问您的视频资源。但通过 API 接口和控制台获取到的视频 URL，Host 仍然是原始域名`xxx.vod2.myqcloud.com`。您需要登录 [点播控制台](https://console.cloud.tencent.com/video/cdnlog) 将新域名设置为默认域名，才能将 URL 中的 Host 修改成新添加的域名。
对于已有域名，您还可以执行证书管理和海外 CDN 配置操作。

## 设为默认域名
1. 登录 [点播控制台](https://console.cloud.tencent.com/video/cdnlog)，在左侧菜单栏中选择【分发播放设置】>【域名设置】。
2. 单击新添加的域名所在行的【设为默认域名】。
 ![](https://main.qcloudimg.com/raw/514fe9b822c1367fb6e0dce91afb0725.png)

## 证书管理
1. 登录 [点播控制台](https://console.cloud.tencent.com/video/cdnlog)，在左侧菜单栏中选择【分发播放设置】>【域名设置】。
2. 单击新添加的域名所在行的【设置】。
3. 单击【证书管理】右侧的【编辑】，您可以启用或关闭证书，单击【确定】保存设置。
 ![](https://main.qcloudimg.com/raw/4ecf9855e523f15daab910d0809b51bb.png)


## 海外 CDN 配置
1. 登录 [点播控制台](https://console.cloud.tencent.com/video/cdnlog)，在左侧菜单栏中选择【分发播放设置】>【域名设置】。
2. 单击新添加的域名所在行的【设置】。
3. 单击【海外CDN】右侧的【编辑】，您可以根据实际需求启用海外 CDN，单击【确定】保存设置。
 >?海外 CDN 开通后将无法关闭。
