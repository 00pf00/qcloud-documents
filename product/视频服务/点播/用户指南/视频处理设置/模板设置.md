模板设置内置了五种类型的模板，分别为**视频转码模板**、**音频转码模板**、**水印模板**、**截图模板**和**转动图模板**。每种模板都可以添加至任务流设置中，用于视频处理设置。

![模板设置](https://main.qcloudimg.com/raw/96105db0ad07f80732df6512ea1dd240.png)

## 视频转码模板

您可根据业务需要创建新的视频转码模板，并进行自定义设置。单击【创建视频转码模板】进入模板自定义设置。

![创建视频转码模板](https://main.qcloudimg.com/raw/8f0800d7f387ceab1a77133f4c2d894d.png)

+ 模板名称，自定义模板名称，仅支持 中文、英文、数字、空格、_、-和. 七种格式，长度不能超过64个字符
+ 封装格式分为 MP3、HLS、FLV 三种可选
+ 配置项默认勾选视频参数，音频参数为可选项
+ 视频参数的编码标准，分为 H.264 和 H.265 两种可选
+ 码率即视频码率，大小限制在0和[128, 35000]之间，单位为Kbps
+ 分辨率为视频宽高比例，范围限制在0和[128, 4096]之间，单位为px
+ 帧率范围限制在[0, 60]，单位为Hz
+ 常用模板，可自定义是否开启

创建好的模板可在模板列表里展示，支持视频设置为常用模板，并能对模板进行查看、编辑、删除等管理操作。

![视频转码模板](https://main.qcloudimg.com/raw/bb670eca686cd0e75913fdcc9ec3e07b.png)

## 音频转码模板


您可根据业务需要创建新的音频转码模板，并进行自定义设置。单击【创建音频转码模板】进入模板自定义设置。

![创建音频转码模板](https://main.qcloudimg.com/raw/a3239c32483b8b1bb6cc01bd932b4f90.png)

+ 模板名称，自定义模板名称，仅支持 中文、英文、数字、空格、_、-和. 七种格式，长度不能超过64个字符
+ 封装格式分为 MP3、HLS、FLV 三种可选
+ 音频参数的编码标准可选项为AAC、MP3两种
+ 采样率设置为三种默认可选
+ 码率即音频码率，大小限制在0和[26, 256]之间，单位为Kbps
+ 声道分为单声道和双声道，二选一
+ 常用模板，可自定义是否开启

创建好的模板可在模板列表里展示，支持视频设置为常用模板，并能对模板进行查看、编辑、删除等管理操作。

![音频转码模板](https://main.qcloudimg.com/raw/d4fc0939f4e8aecd158b7ab70aa6ba2a.png)

## 水印模板

您可根据业务需要通过上传图片的方式来创建水印模板，并进行自定义的默认水印配置以及水印在视频中的位置设置。

![创建水印模板](https://main.qcloudimg.com/raw/2ea843e4b6ab0e961ecab92c92a6bacf.png)

+ 模板名称为自定义，仅支持 中文、英文、数字、空格、_、-和. 七种格式，长度不能超过64个字符
+ 水印图片上传，支持图片格式为 png 格式，图片大小不能超过200KB，尺寸不能超过200 * 200px
+ 水印位置提供左上、右上、左下、右下四个常规选择，亦可通过调整垂直偏移/水平偏移进行精细化的位置配置，水印位置会在选择上与此保持同步
+ 水印尺寸有两种可筛选的方式，分别是按百分比%和像素px调整大小。单位选择%时，按原有尺寸来进行百分百比例缩放；单位选择px时，直接根据指定大小缩放水印

在创建完成的水印模板管理列表中可以查看水印模板的名称、预览水印文件、格式、类型、水印位置、尺寸等信息，还可以对该水印模板进行查看、编辑、删除及设为默认模板等操作。

![水印模板](https://main.qcloudimg.com/raw/7537b6bfc2757eeb5325227f31ce7607.png)

## 截图模板
通过截图模板，用户可以对上传的视频进行多种方式的截图，目前腾讯云控制台支持时间点截图、采样截图、雪碧图截图三种截图方式。

操作指引：

1、在【模板设置】下单击【截图模板】

![sp截图模板](https://main.qcloudimg.com/raw/c7b434d5d68d51e2ebc81be7de5d473b.png)

2、在截图模板下单击【创建截图模板】，进入创建界面

![sp创建截图](https://main.qcloudimg.com/raw/a6664c6a3f0bafe54f41929e6b1d7a60.png)

### 时间点截图
单击【创建截图模板】，输入模板名称，在截图类型中选择时间点截图，配置需要截图的图片尺寸，采样的时间点需在任务流中配置，模板仅进行类型的配置。详细配置方式查看任务流设置。

![sp时间点截图](https://main.qcloudimg.com/raw/da2aa92eeb37cb07a8414a0935c622d0.png)

### 采样截图
单击【创建截图模板】，输入模板名称，在截图类型中选择采样截图，配置需要截图的图片尺寸，采样间隔支持按照百分比截图和按时间间隔截图两种方式。

![sp采样截图](https://main.qcloudimg.com/raw/9a3af378df1d95c89cce38c12502af3f.png)

### 雪碧图截图
单击【创建截图模板】，输入模板名称，在截图类型中选择雪碧图截图，配置需要截图的图片尺寸，采样间隔支持按照百分比截图和按时间间隔截图两种方式，同时输入需要的小图行数和列数来完成配置。

![sp雪碧图截图](https://main.qcloudimg.com/raw/a8ed6199c5280d11ed9343b47b7ac84e.png)

在截图模板里面会展示模板的名称、截图类型、图片尺寸。在【操作】栏可以对该截图模板进行查看、编辑和删除操作。

![sp截图模板管理](https://main.qcloudimg.com/raw/534e7abfe47a15c63e301ed3709ea439.png)


## 转动图模板

您可根据业务需要创建转动图模板，用户可以按照时间点截取所需要的转动图。

操作指引：

1、选中【转动图模板】下的【创建转动图模板】

![sp创建转动图模板](https://main.qcloudimg.com/raw/de26419472a463c6f6b296c6ba23b5aa.png)

2、单击【创建转动图模板】，输入模板名称，选择需要输出的图片格、帧率和图片质量，配置需要截图的图片尺寸，转动图的时间点需在任务流中配置，模板仅进行类型的配置。详细配置方式查看任务流设置。

![创建转动图模板](https://main.qcloudimg.com/raw/e4ed072208ef5e03711e51b65aa2dbb3.png)
