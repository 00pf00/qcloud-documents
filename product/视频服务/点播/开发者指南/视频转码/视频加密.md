视频加密是指将视频转自适应码流并对码流进行加密的过程。云点播视频加密提供了两种级别的加密方案：
* 商业级 DRM：目前提供 FairPlay 和 Widevine 两种。
* 基础级 DRM：SimpleAES 普通加密。

>?
- 视频加密的目标规格，包含了“转哪些码率的视频流”和“转哪些码率的音频流”等参数。云点播使用转自适应码流模板表示参数集合，具体请参见 [转自适应码流模板](《超链本文的“转自适应码流模板”一章》) 单元。
- 视频加密和 [转自适应码流](《超链本文的“转自适应码流”一文》) 一样，是一种离线任务，发起方式请参见 [任务发起](《超链本文的“任务发起”一章》) 和 [结果获取](《超链本文的“结果获取”一章》) 单元。

## 转自适应码流模板
通过转自适应码流模板，可以指定以下相关参数。

| 参数 | 说明 |
| -- | -- |
| 打包类型 | 自适应码流的格式，目前支持 HLS 和 Dash 两种。 |
| DRM 类型 | 可以选择是否对视频加密保护，支持以下三种 DRM 加密方案：<li>FairPlay</li><li>Widevine</li><li>SimpleAES</li> |
| VideoTrack 列表 | 包含哪些不同码率的视频流。 |
| AudioTrack 列表 | 包含哪些不同码率的音频流。 |
| 是否过滤“低码率转高码率” | 通常来说，低码率的原始视频转码高码率无法获得画质和音质的提升。开启过滤“低码率转高码率”，可以避免不必要的转码。|

针对常见的参数组合，点播提供了 [预置转自适应码流模板](《超链“预置参数模板列表”的“预置转自适应码流模板”一章》)。目前暂不支持通过自定义转自适应码流模板。

## 任务发起
发起视频内容审核任务，有“通过服务端 API 直接发起”、“通过控制台直接发起”和“上传时指定要执行的任务”三种方式，请参照视频处理的 [发起任务](超链“视频处理任务体系”的“任务发起”一章)。

下面给出各种方式发起视频内容审核任务的说明：
* 调用服务端 API [ProcessMedia](https://cloud.tencent.com/document/product/266/33427) 发起任务：在请求中的 MediaProcessTask.AdaptiveDynamicStreamingSet.Definition 参数指定 [转自适应码流模板]() 的模板 ID。
* 调用服务端 API [ProcessMediaByUrl](/document/product/266/33426)  发起任务：在请求中的 MediaProcessTask.AdaptiveDynamicStreamingSet.Definition 参数指定 [转自适应码流模板]() 的模板 ID。
* 通过控制台对视频发起任务：在控制台 [添加任务流](/document/product/266/14058#.E8.A7.86.E9.A2.91.E5.A4.84.E7.90.86.E4.BB.BB.E5.8A.A1.E6.B5.81)，任务流中配置转自适应码流任务；在控制台使用该任务流 [发起视频处理](/document/product/266/14054#.E5.8F.91.E8.B5.B7.E8.A7.86.E9.A2.91.E5.A4.84.E7.90.86)。
* 服务端上传时指定任务：在控制台 [添加任务流](/document/product/266/14058#.E8.A7.86.E9.A2.91.E5.A4.84.E7.90.86.E4.BB.BB.E5.8A.A1.E6.B5.81)，任务流中配置转自适应码流任务； [客户端上传签名](/document/product/266/9221#.E7.AD.BE.E5.90.8D.E5.8F.82.E6.95.B0) 中的```procedure```参数指定为该任务流。
* 客户端上传时指定任务：在控制台 [添加任务流](/document/product/266/14058#.E8.A7.86.E9.A2.91.E5.A4.84.E7.90.86.E4.BB.BB.E5.8A.A1.E6.B5.81)，任务流中配置转自适应码流任务；在 [客户端上传签名](/document/product/266/9221#.E7.AD.BE.E5.90.8D.E5.8F.82.E6.95.B0) 中的```procedure```指定该任务流。
* 控制台上传：在控制台 [添加任务流](/document/product/266/14058#.E8.A7.86.E9.A2.91.E5.A4.84.E7.90.86.E4.BB.BB.E5.8A.A1.E6.B5.81)，任务流中配置转自适应码流任务；通过控制台上传视频，选择[【上传的同时对视频进行处理操作】](/document/product/266/2841#web-.E4.B8.8A.E4.BC.A0)并指定视频上传后执行该任务流。
