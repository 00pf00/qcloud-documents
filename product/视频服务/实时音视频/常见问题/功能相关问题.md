

### 什么是实时游戏音视频？

实时游戏音视频（Tencent Mobile Gaming）是一个专门针对游戏场景定制的实时游戏音视频 SDK，覆盖了休闲社交类、MOBA 类、MMORPG 等多种游戏类型，提供多种常用功能，包括多人实时语音、实时视频、语音消息、语音转文本等。功能完备，接入门槛低，一个 SDK 即可满足多样化的游戏音视频诉求。

支持，具体操作指南请参考：

- [在线直播(Android)](https://cloud.tencent.com/document/product/647/35428)

### 实时音视频最多可以同时创建多少个房间？
没有限制。

### 实时音视频的视频服务端最大支持带宽是多少？
没有限制。

### 实时音视频最多可以支持多少个人同时视频通话？
单个房间最多支持300人同时在线，最多支持50人同时开启摄像头。

### 实时音视频房间 ID 的区间值是多少？
房间号取值区间为1 - 10000000，由开发者自行维护和分配。

### 实时音视频是否支持私有化部署？
目前不支持。

### 实时音视频开通旁路直播，域名是否需要进行备案使用？
如需开通旁路直播，域名需要备案才能使用，更多详情请参见 [CDN 直播观看](https://cloud.tencent.com/document/product/647/16826)。

### 实时音视频延时大约多少？
全球端到端延时小于300ms。

### 实时音视频是否支持主动呼叫功能？
支持。

### 实时音视频双人视频通话是否支持蓝牙耳机？
支持，但在使用蓝牙耳机输入输出音频时，audioDevice 列表里将只有蓝牙耳机一个设备，无法切换为手机自身的音频输入输出。

支持的浏览器列表如下表所示。
| 操作系统 | 浏览器类型 | 最低版本要求 | 接收（播放）| 发送（上麦）|
|:-------:|:-------:|:-------:|:-------:|:-------:|
| Mac OS  | 桌面版 Safari 浏览器 |  11+ | 支持 | 支持 | 
| Mac OS  | 桌面版 Chrome 浏览器 |  47+ | 支持 | 支持 | 
| Windows  | 桌面版 Chrome 浏览器|  52+ | 支持 | 支持 | 
| Windows  | 桌面版 QQ 浏览器 |  10.2 | 支持 | 支持 | 
| iOS | 移动版 Safari 浏览器 | 11.1.2 | 支持 | 支持 | 
| iOS | 微信内嵌网页| 12.1.4 | 支持 | 不支持 | 
| Android | 移动版 QQ 浏览器| - | 不支持 | 不支持 | 
| Android | 移动版 UC 浏览器| - | 不支持 | 不支持 | 
| Android | 微信内嵌网页| - | 不支持 | 不支持 | 

### 实时音视频是否支持在国外使用？
支持。

### 实时音视频接入 PC 端是否支持屏幕分享功能？
支持，端屏幕分享接口详情请参见 [Windows（C++）API](https://cloud.tencent.com/document/product/647/32268)。

### 实时音视频是否支持在微信公众号中使用？
目前暂不支持。

### 是否支持将录制好的视频添加到实时音视频中？
目前暂不支持。

### 实时音视频能否把直播视频录制后存储在手机本地？
不支持直接存储在手机本地，录制后视频文件默认存储在云点播平台，您可以自行下载并保存到手机中，更多详情请参见 [云端录制和回放](https://cloud.tencent.com/document/product/647/16823)。


### 实时音视频是否支持单纯的实时音频？
支持纯音频推流。

### 实时语音服务组队模式开房间，是否限制房间人数？
实时语音服务开房间不限制房间人数。

### 一个房间里可以同时有多路屏幕分享吗？
目前一个 TRTC 音视频房间只能有一路屏幕分享。

### 指定窗口分享（SourceTypeWindow），当窗口大小变化时，视频流的分辨率会不会也跟着变化？
不会跟着变化，当窗口大小变化时，窗口画面会被等比例缩放到目标分辨率上。
