## 其他API对象

### TRTCParams

编码参数 TRTCParams

作为 TRTC SDK 的进房参数，只有该参数填写正确，才能顺利进入 roomId 制定的视频房间


| 属性 | 类型 | 描述 |
|-----|-----|-----|
| [sdkAppId](31511#.3Ccode.3Esdkappid.3C.2Fcode.3E) | UInt32 | 应用标识 [必填] 腾讯视频云基于 sdkAppId 完成计费统计  |
| [userId](31511#.3Ccode.3Euserid.3C.2Fcode.3E) | NSString *_Nonnull | 用户标识 [必填] 当前用户的 userid，相当于用户名  |
| [userSig](31511#.3Ccode.3Eusersig.3C.2Fcode.3E) | NSString *_Nonnull | 用户签名 [必填] 当前 userId 对应的验证签名，相当于登录密码  |
| [roomId](31511#.3Ccode.3Eroomid.3C.2Fcode.3E) | UInt32 | 房间号码 [必填] 指定房间号，在同一个房间里的用户（userId）可以彼此看到对方并进行视频通话  |
| [privateMapKey](31511#.3Ccode.3Eprivatemapkey.3C.2Fcode.3E) | NSString *_Nonnull | 房间签名 [非必选] 如果您希望某个房间（roomId）只让特定的某些用户（userId）才能进入，就需要使用 privateMapKey 进行权限保护  |
| [bussInfo](31511#.3Ccode.3Ebussinfo.3C.2Fcode.3E) | NSString *_Nonnull | 业务数据 [非必选] 某些非常用的高级特性才需要用到此字段  |
