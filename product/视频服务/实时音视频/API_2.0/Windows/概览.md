### TRTCParams

编码参数 TRTCParams

作为 TRTC SDK 的进房参数，只有该参数填写正确，才能顺利进入 roomId 制定的视频房间


| 属性 | 类型 | 描述 |
|-----|-----|-----|
| [sdkAppId](31519#.3Ccode.3Esdkappid.3C.2Fcode.3E) | uint32_t | 应用标识 - [必填] - 腾讯视频云基于 sdkAppId 完成计费统计  |
