### 退款金额及途径

- 若资源已消费金额 >（当前有效订单金额 + 未开始订单金额），则退款金额为0元，多出费用不额外收取。

购买时使用的代金券不支持退还，购买时使用的非代金券费用按支付方式（现金/赠送金）及支付比例退还到支付方腾讯云账户。

## 退款案例

### 案例背景

- 用户 A 新购实时音视频 TRTC **25万分钟**的通用套餐包。
- **25万分钟**的通用套餐包单价1588元，按量计费服务单价第一阶梯7.00元/千分钟、第二阶梯6.72元/千分钟。
- 用户 A 新购**25万分钟**的通用套餐包，并在活动期间享受0.6612折，折后总价 = 1588 × 0.6612 ≈ 1050元；其中，代金券支付300元、现金支付300元、赠送金支付450元。

### 无理由退款场景

用户 A 在新购5天内申请退款，且是该账号首次申请退款。

- 购买时使用的300元代金券不支持退还。
- **实际支付金额** = 现金支付金额 + 赠送金支付金额 = 300 + 450 = 750元。
- **应退款总金额** = 实际支付金额750元。
  - 其中 **应退还现金金额** = 现金支付金额 = 300元。
  - **应退还赠送金金额** = 赠送金支付金额 = 450元。

### 非全额退款场景

用户 A 在购买5天内申请退款，且不是该账号首次申请退款，总使用时长5万分钟。

- 购买时使用的300元代金券不支持退还。
- **实际支付金额** = 现金支付金额 + 赠送金支付金额 = 300 + 450 = 750元。
- **资源已消费金额** = 已使用通用套餐包费用 = 7.00元/千分钟 ×25千分钟  +  6.72元/千分钟 × (50千分钟 - 25千分钟 )  = 343元。
- **应退款总金额** = 实际支付金额 - 资源已消费金额 = 750 - 343 = 407元。
  -  其中，**应退还现金金额** = 应退款总额 × 现金支付金额/实际支付金额 = 407 × ( 300 / 750 )  = 162.8元。
  -  **应退还赠送金金额** = 应退款总额\*赠送金支付金额/实际支付金额 = 407 × ( 450 / 750 ) = 244.2元。
