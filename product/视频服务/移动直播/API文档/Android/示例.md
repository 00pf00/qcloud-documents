## 前言
这个开源示例 Demo 主要演示了 [移动直播 SDK](https://cloud.tencent.com/document/product/454) 部分 API 的使用示例，帮助开发者可以更好的理解 移动直播 SDK 的 API，从而快速实现一些移动直播场景的基本功能。 

## 工程地址

| 所属平台 |                         GitHub 地址                          |
| :------: | :----------------------------------------------------------: |
| Android  | [Github](https://github.com/tencentyun/MLVBSDK/tree/master/Android/MLVB-API-Example) |

## 目录说明
在这个示例项目中包含了以下场景：

- 基础功能：
  - [摄像头推流](https://github.com/tencentyun/MLVBSDK/tree/master/Android/MLVB-API-Example/Basic/LivePushCamera)
  - [录屏推流](https://github.com/tencentyun/MLVBSDK/tree/master/Android/MLVB-API-Example/Basic/LivePushScreen)
  - [直播拉流](https://github.com/tencentyun/MLVBSDK/tree/master/Android/MLVB-API-Example/Basic/LivePlay)
  - [连麦互动](https://github.com/tencentyun/MLVBSDK/tree/master/Android/MLVB-API-Example/Basic/LiveLink)
  - [连麦 PK](https://github.com/tencentyun/MLVBSDK/tree/master/Android/MLVB-API-Example/Basic/LinkPK)
- 进阶功能：
  - [动态切换渲染组件](https://github.com/tencentyun/MLVBSDK/tree/master/Android/MLVB-API-Example/Advanced/SwitchRenderView)
  - [自定义视频采集](https://github.com/tencentyun/MLVBSDK/tree/master/Android/MLVB-API-Example/Advanced/CustomVideoCapture)
  - [第三方美颜](https://github.com/tencentyun/MLVBSDK/tree/master/Android/MLVB-API-Example/Advanced/ThirdBeauty)
  - [RTC 连麦 + 超低延时播放](https://github.com/tencentyun/MLVBSDK/tree/master/Android/MLVB-API-Example/Advanced/RTCPushAndPlay)


>?目前的工程结构跟标准的 Android Studio 工程在名称大小写上可能有略微的差异，主要目的是方便大家在网页上看到此工程时，名称意义更加清晰。
