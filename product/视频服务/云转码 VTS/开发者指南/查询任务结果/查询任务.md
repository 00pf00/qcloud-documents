关键结构体说明：
WorkflowTask：



TaskNotifyConfig：
|名称|	类型|	必选|	描述|
| :--- | :---: | :---: | :--- |
| CmqModel |	String |	是 | CMQ 的模型，有 Queue 和 Topic 两种。
| CmqRegion |	String |	是 | CMQ 的园区，如 sh，bj 等。
| QueueName |	String |	否 | 当模型为 Queue 时有效，表示接收事件通知的 CMQ 的队列名。
| TopicName | String |	否 | 当模型为 Topic 时有效，表示接收事件通知的 CMQ 的主题名。
| NotifyMode | String |	否 | 工作流通知的模式，可取值有 Finish 和 Change，不填代表 Finish。

输出结构及字段定义请参考[DescribeTaskDetail API 接口文档](/document/product/1090/33530)。
