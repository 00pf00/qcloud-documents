|:---------:|:---:|:----:|
| MP4 | H.264, H.265 |  AAC |

| MOV | H.264, H.265, MPEG4 |  AAC |
| WMV | WMV1, WMV2 |  WMA1, WMA2 |
| MKV | H264, VP8, MPEG4 | AAC  |
| AVI | H264, WMV1, WMV2, MPEG4 |  AAC, WMA1. WMA2 |
| RMVB | RV系列 |  RAAC, RACP |
| TS | H.264, MP1V, MP4V | MP1, MP2, MP3, MP4A  |
| MPG | MPEG1, MPEG2 |  MP2 |
| 3GP | H263, MPEG4 |  AMR, AAC  |

| 文件格式     | 视频编码类型 | 音频编码类型  |

### 未发起转码怎么办？

未发起转码的原因及处理方法如下：

- **上传失败**：上传失败是指客户通过腾讯云COS SDK或腾讯云官网控制台上传文件返回失败，常见HTTP错误码包括4XX，5XX等。这时不会触发COS事件通知，VTS不会发起转码任务。请确认文件上传成功。
- **上传成功但未触发转码**：可能的情况是未设置工作流，工作流设置不正确等。请检查并确保工作流被正确设置。
