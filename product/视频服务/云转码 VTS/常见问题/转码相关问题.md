视频输入格式：
| 文件格式     | 视频编码类型 | 音频编码类型  |
|:---------:|:---:|:----:|
| MP4 | H.264, H.265 |  AAC |
| FLV | H.264, H.265 |  AAC |
| MOV | H.264, H.265, MPEG4 |  AAC |
| WMV | WMV1, WMV2 |  WMA1, WMA2 |
| MKV | H264, VP8, MPEG4 | AAC  |
| AVI | H264, WMV1, WMV2, MPEG4 |  AAC, WMA1. WMA2 |
| RMVB | RV系列 |  RAAC, RACP |
| TS | H.264, MP1V, MP4V | MP1, MP2, MP3, MP4A  |
| MPG | MPEG1, MPEG2 |  MP2 |
| 3GP | H263, MPEG4 |  AMR, AAC  |

| 文件格式     | 视频编码类型 | 音频编码类型  |
|:---------:|:---:|:----:|

### 未发起转码怎么办？

未发起转码的原因及处理方法如下：

- **上传失败**：上传失败是指客户通过腾讯云COS SDK或腾讯云官网控制台上传文件返回失败，常见HTTP错误码包括4XX，5XX等。这时不会触发COS事件通知，VTS不会发起转码任务。请确认文件上传成功。
- **上传成功但未触发COS事件通知**：该情况是指客户文件已上传成功，但VTS 并未收到COS事件通知，可能的原因是COS事件通知服务异常。该情况很少出现，并且客户无感知，一旦出现文件上传成功但没有收到文件转码结果回调的情况，请联系客服或售后寻求支持。

### 发起转码失败？

发起转码失败是指文件已上传成功并且收到COS事件通知但并发起转码任务失败，可能的原因及处理方法如下：

- **协议不支持**：当前支持COS v5版本的事件通知协议，同时兼容COS v4版本协议，使用其他版本协议可能会有兼容性问题，导致发起转码任务失败。该种情况下，请确认使用的COS版本为前述两种类型。

### 转码失败怎么办？

转码失败是指转码服务提供的各类型子任务（转码、截图、打水印、智能审核、智能识别、智能分析和小程序发布）出现失败的情形。

请参考[转码错误码](https://localhost)查看具体错误，如有需要，请联系客服和售后寻求支持。
