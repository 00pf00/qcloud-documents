多媒体创作引擎同时提供标准 PaaS 服务，即开发型平台，开发型平台仅支持 API 调用，不提供用户端，适用于云转推、云剪辑等 API 使用场景。本文将指导您如何在多媒体创作引擎的管理端创建开发型平台。 

[](id:step1)

## 步骤1：创建平台入口
进入 **[多媒体创作引擎](https://cme.cloud.tencent.com/)>平台管理**，进入平台管理页面。单击 **创建平台**，开始创建新的多媒体创作引擎平台。
![](https://main.qcloudimg.com/raw/57a5d23975ad46b328054558e3d1a2ac.png)

[](id:step2)
## 步骤2：平台设置
填写平台名称（15字以内），选择平台类型为**开发型**，单击 **下一步** 进入存储设置。
![](https://main.qcloudimg.com/raw/a2f36449e40004704b13f4bfca09f556.png)   
> ?
> - 开发型平台创建时需要绑定会员，但是在创建后，API调用都将产生一定费用，请及时关注 [用量](https://cme.cloud.tencent.com/)。
> - 多媒体创作引擎为您提供3个开发型平台，若需创建更多开发型平台，请 [提交工单](https://console.cloud.tencent.com/workorder/category)。

[](id:step3)
## 步骤3：存储设置
开发型平台仅支持云点播作为底层存储，您需要绑定云点播子应用，以保证后续 API 的正常使用。选择好点播子应用后，单击 **创建** 即可创建一个新的开发型平台。
> ?
>- 如果您尚未开通云点播子应用服务，请先开通 [点播子应用](https://cloud.tencent.com/document/product/266/14574#.E5.BC.80.E9.80.9A.E5.AD.90.E5.BA.94.E7.94.A8)。
>- 一个云点播子应用只能绑定一个多媒体创作引擎平台。
