云导播可在云端实现点播转直播，多个直播流间切换，画中画混流播出，录制重播，备播护航等功能。既省去了沉重的硬件设备和高额的费用，还满足了丰富的线上业务场景。 

本文将指导您创建导播台后，如何使用导播台实现在线导播功能。

[](id:limit)
## 使用限制
- 每个账号可创建**10**个云导播台实例，删除导播台实例后即可重新添加。若您需要多个导播台，请 [提交工单](https://console.cloud.tencent.com/workorder/category) 申请。
- 最多添加**100个**轮播文件。
- 最多支持转推**10路**。

[](id:step1)
## 步骤1：创建导播台
1. [访问平台](https://tcloud-doc.isd.com/document/product/1489/62230?!preview&!editLang=zh#step5) 并进入 **云直播**>**云导播台**，单击 **新增导播台**，进入导播台新增页进行如下配置：
  - 自定义录入导播台名称和描述。
  - 设定导播台运行结束时间，单击 **确定**。
3. 单击 **确定**，即可创建一个新导播台。
![](https://main.qcloudimg.com/raw/c92ddeb43afb23915aa93773d2a9294a.png)
>! 导播台到达运行结束时间时，将会执行停止运行操作，导播台的预监、输出、录制及转推等功能将会关闭。

## 步骤3：配置布局
1. 在功能区的布局组建区中单击 **添加布局**，进入布局新增页。
2. 选择布局模板，支持选择**内置布局模板**和**自定义布局**两种。
	- **内置布局模板**：目前云导播内置了5种模板，请根据需要选择。
	- **自定义布局**：单击 **添加**，生成输入源布局模块（最多可添加4路输入源）。支持排列叠放次序，大小变换，拖动位置配置。
		- 按住鼠标拖动输入源，对相应的输入源大小和位置进行调整。
		- 单击 **前面** 或 **后面** 调整各输入源叠放次序。
		- 可单击![](https://main.qcloudimg.com/raw/c36f212774fe93c1939213b52cb23027.png)，可删除当前输入源。	
3. 为布局模块选择对应的输入源，可在预览框中实时预览效果。
![](https://main.qcloudimg.com/raw/823535bd8f553d80c368d7ab3fd1990a.png)
4. 单击 **确定** 即可创建一个布局。创建成功后，您可在布局组件区中看到已添加成功的布局。
5. 单击已添加成功的布局即可推送到预监。
![](https://main.qcloudimg.com/raw/3036f2c3a609d951d286b5ca434002e8.png)

>?
>- 一个布局里面，一个输入源只能选择一次。
>- 正在 PVW 使用的布局显示绿色框，正在 PGM 使用的布局显示红色框，正在使用的布局不能被编辑或删除。
>- 如果您需要修改布局模板，您可以单击相应的布局上右下方的![img](https://main.qcloudimg.com/raw/f3894a9549d790b21c4651b05c626f31.png)进行修改操作。
>- 如果您需要删除布局模板，您可以单击布局右上方的![img](https://main.qcloudimg.com/raw/84c4c8669560299f53b5b2dd9872bf12.png)进行删除操作。

## 步骤4：启动预监
查看视频输入源区域，单击已添加的视频源或已创建的布局模板，即可成功启动预监（PVW）。
![](https://main.qcloudimg.com/raw/70e0b59962ba6af85f1b118b6453e047.png)
>!预监使用的布局出现绿色框预监状态提示。使用中的输入源或布局无法删除。如果您需要删除该布局，您可以手动关闭 PVW 或 PGM 或释放导播台后删除。

[](id:step5)
## 步骤5：音频处理
- 支持调整输出流中，每个输入流的音量。当勾选音视频同步切换时，PVW 播放的视频和音频相同。
<dx-alert infotype="explain" title="示例">
当前 PVW 正在播放1号视频源，则正在播放的音频也是1号音频，若当前 PVW 播放的视频为1号和2号的混流，则正在播放的音频也是1号和3号视频源的混音。
</dx-alert>
- 取消勾选音视频同步切换选项后，可以将音频和视频分离。
<dx-alert infotype="explain" title="示例">
若当前播放的为1号视频源，可以选择播放3号视频源的音频，若当前播放的是1号和2号视频源的混流，可以选择仅播放1号视频源的声音。
</dx-alert>

## 步骤6：添加组件

[](id:watermark)
### 添加水印
导播台支持叠加多个水印。若您需在导播画面中添加水印，具体操作步骤如下：

1. 选择水印标签，单击 **添加** 进入新增水印页。
2. 单击 **上传图片**，上传您的水印图片。
> ? 水印图片大小限制为2M，宽高不超过1024px，可上传类型为 PNG 或 GIF 格式，支持动态水印。
3. 通过在编辑画面中拖动水印图片，或单击 **开启精确像素调整**，输入精确像素值两种方式对水印图片的位置和尺寸进行调整。
![](https://main.qcloudimg.com/raw/6de46e16e54d4b0179af29c363211915.png)
> ? 若您需使用精确像素调整功能，需要先开启 PVW 预监。
4. 编辑完成后，单击 **确定** 保存水印模板。
5. 选择需要开启的水印模板的 **PVW窗口** 开关，即可在 PVW 上显示水印。
![](https://main.qcloudimg.com/raw/2fd55b957fe9ccff1b6abca009187817.png)

[](id:txt)
### 添加文本
导播台支持多文本叠加，也支持文本和水印叠加。若您需在导播画面中添加文本，具体操作如下：

1. 选择文本标签，单击 **添加** 进入新增文本页。
2. 在下方输入您需要的文本，支持调节字体字号、颜色，支持拖动调整文本位置，支持**固定显示**或**循环滚动**。
![](https://main.qcloudimg.com/raw/b507cac9eb8eb113b9b7a27dcbfe0453.png)
3. 编辑完成后，单击 **确定** 保存文本模板。
4. 单击需开启的文本模板的 **PVW窗口** 开关，即可在 PVW 上显示文本。
![](https://main.qcloudimg.com/raw/f76b74b2ea488a0b110a5164572f2499.png)

## 步骤7：设置输出
如果您需要将直播流推送到第三方云商，请先配置转推。
1. 单击右上方的设置按钮![](https://main.qcloudimg.com/raw/e2e81e20555220e8439facc0949b038c.png)，进入推流设置页。
2. 在**设置转推地址**框中，单击选择您需添加的转推地址类型（可自定义），在新增的输入栏中，填写厂商名称、转推地址和延时播放时长。
![](https://main.qcloudimg.com/raw/f17f89b565870afe2c1f5f760f79ea6a.png)
>! 
>- 转推第三方最多支持转推三路，只支持 RTMP 协议转推，平台推流地址需满足如下格式：`rtmp://domain/app/stream?arg1=xxx`。
>- 开启 PGM 之后转推状态才会更新。
3. **在设置视频输入尺寸**框中，根据实际需求配置视频宽、视频高、帧率和码率。支持选择自定义尺寸或推荐比例。
![](https://main.qcloudimg.com/raw/ce1397eb7deca2b5682e132f9daa724a.png)
> ! 
> - 推单流时，视频宽、视频高、帧率、码率的参数不填或填0，按照原始流的参数输出。
> - 使用画中画推流时，视频宽、视频高、帧率、码率的参数不填或填0，按照720p输出。
4. 单击 **确定** 即可配置完成。

[](id:step8)

## 步骤8：启动输出
如果预监效果满足您的要求，您可以单击 **推送到直播** 启动输出。启动输出后，您可以在主监区看到推送的画面。
>!
>- 在直播开始前，请先配置 [转推地址](#step7)。
>- 输出使用的输入源或布局上将出现红色框的主监状态提示。

![](https://main.qcloudimg.com/raw/9adcbc50fc650518a893b5535c0a984f.png)

## 步骤9：录制
1. 单击主监（PGM）的![](https://main.qcloudimg.com/raw/42963b40189dad3a8f6d9f68022ac9ed.png)，进入录制配置页。
2. 填写录制文件名称，选择录制存储目录位置，单击 **确定** 即可开始录制。
![](https://main.qcloudimg.com/raw/102b51001356e0b774a6a0cfeeab3132.png)
3. 录制完成后，单击 **录制中** 停止录制。
![](https://main.qcloudimg.com/raw/12a1c3f63584ae0931e261733256d692.png)
4. 系统将录制好的文件传送到云媒资系统中，您可以在云媒资中查看录制文件。
