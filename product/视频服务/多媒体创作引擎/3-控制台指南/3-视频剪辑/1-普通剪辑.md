普通剪辑是集视频剪辑、智能媒资和特效合成于一体的非线形剪辑工具，适用于剪辑本地视频及云媒资中的视频资源。

普通剪辑编辑面板一共分为五个模块，如下图所示：

## 步骤1：选择项目素材

1. [访问平台](https://tcloud-doc.isd.com/document/product/1489/62230?!preview&!editLang=zh#step5) 选择 **视频剪辑>创作视频>普通剪辑**，创建普通剪辑项目并进入剪辑器。
2. 打开项目资源库导入普通剪辑项目中使用的项目素材。支持通过 [导入](#upload)、[素材市场添加](#public)、[配音生成](#dubbing) 和 [文字转语音](trans) 方式进行添加素材。
   ![](https://main.qcloudimg.com/raw/df16e8bd086be0277c3f3707bd7089b9.png)
3. 素材添加成功后，将自动根据素材的类型分为视频、音频、图片、语音包四种。
>? 若您需删除已添加到项目资源库中的素材，可将鼠标悬停在素材上方时，单击右上角的![](https://main.qcloudimg.com/raw/552f842912dfbcd6938f6e80b28691dd.png)，即可删除项目资源库中的素材（**使用中的素材不可删除**)。

### 导入素材
进入 **项目资源库** 对剪辑所需使用的项目素材进行管理。

3. 单击 **确定**，即可将素材导入到项目资源库中。

素材市场中提供海量图片、视频、音频的版权素材。

### 配音生成
1. 单击 **配音** 进入音频录制页面，单击![](https://main.qcloudimg.com/raw/f201b993a44127a7cd6584347ad481e2.png)开始录制音频，单击![](https://main.qcloudimg.com/raw/b8385d03967db2a730f439cc0f36d22f.png)即可暂停录制。

## 步骤2：添加到轨道
素材添加到项目资源库后，可通过下述几种方式将素材添加到轨道进行编辑操作。

### 截取片段添加
在预览框中，对视频素材进行**预览**和**片段截取**。

1. 进入  **项目资源库**，单击视频素材上方的![](https://main.qcloudimg.com/raw/ebf3120de5fe789754c24bde729c07f2.png)，进入视频预览弹窗。
![](https://main.qcloudimg.com/raw/30eec76de1fc34dbdd26031f5d0a3891.png)
2. 在时间轴中选择开始位置，单击上方的![](https://main.qcloudimg.com/raw/634445281a6cee24d7a80fb70209b30d.png)设置入点 。
3. 在时间轴中选择结束位置，单击上方的![](https://main.qcloudimg.com/raw/aa2492c7b2e71678f92da39443355de0.png)设置出点。
4. 单击 **导入到时间轴编辑**，即可将选择的片段添加到轨道中。
![](https://main.qcloudimg.com/raw/afed548cc9a0b335097c609f99e2385d.png)        

### 拖拽添加
进入 **项目资源库** 中选择项目素材，单击拖至下方的编辑轨道，即可将素材添加到轨道中。
![](https://main.qcloudimg.com/raw/9cd175dfbab64eaea7878e44ae89543b.png)

### 直接添加
进入 **项目资源库** 中选择项目素材，单击视频素材上方的![](https://main.qcloudimg.com/raw/9d7502b4704b15fdbbd94c8d153ad531.png)，即可将素材添加到轨道中。
<img src="https://main.qcloudimg.com/raw/53cecaf7dda37bf89836f31ee8009a04.png" width=700 />

### 更换素材
若您对当前加入轨道的素材不满意，可以在轨道中选择需替换的素材，单击其左上方的![](https://main.qcloudimg.com/raw/5dd0414e819a8b22c43657097078e752.png)，在打开的项目资源库中单击选择替换素材即可。

![](https://main.qcloudimg.com/raw/2947b0e19d50a47e5fffd51525a7aa22.png)

### 删除素材
若您需去除已加入轨道的素材，可在轨道中选中需删除的素材，单击上方的删除按钮即可。
![](https://main.qcloudimg.com/raw/037889025658a03fe76919464168738b.png)

## 步骤3：编辑项目素材
项目素材素材添加到轨道后，您可以在轨道或预览框中单击目标素材，打开素材的编辑页，进行二次编辑。

### 基本设置/图片编辑

| 功能             | 说明                                                         |
| ---------------- | ------------------------------------------------------------ |
| 设置音量大小     | 拖拽进度条，即可改变视频素材的音量（仅视频素材支持）         |
| 设置透明度       | 拖拽进度条，即可改变素材的透明度                             |
| 开启分离视频音轨 | 单击 **分离视频音轨**，可将视频素材的音频和画面（仅视频素材支持） |
| 开启画中画       | 单击 **开启画中画**，可对素材进行放大、缩小、挪动等调整      |
| 设置裁剪         | 单击 **设置裁剪**，可对素材进行**自由裁剪**或**按推荐比例裁剪** |
| 设置播放速度     | 单击下拉框选择视频素材的播放速度，可调节速度为 0.25倍 至 4.0倍 （仅视频素材支持） |
| 位置调整         | 支持**逆时针旋转**、**顺时针旋转**、**水平翻转**和**垂直翻转**四种旋转方式 |

![](https://main.qcloudimg.com/raw/5421ed508f1b1c179b27330fdb24d788.png)        

### 添加滤镜
普通剪辑提供了多种滤镜素材，可覆盖多种场景。在 **滤镜** 中单击目标滤镜，为所选的视频/图片素材段增加滤镜。
![](https://main.qcloudimg.com/raw/e0ef98f80335623fb2405d065d3b6f4e.png)        

### 添加动画
在 **动画** 中，可对目标素材段添加入场、运动、出场动画。
![](https://main.qcloudimg.com/raw/af9aea7b95bdc08412242de027f60eaf.png)        

### 添加调色
在 **调色** 中，可对项目素材的亮度、对比度、饱和度与锐化进行调整。支持将调整结果**重置**和**应用到全部**项目素材。
![](https://main.qcloudimg.com/raw/efede0f8c7106a0650fd35674c78702b.png)        

### 添加背景
在 **背景** 中，可选择背景填充方式为**无填充**、**纯色填充**、**图片填充**或**模糊填充**，改变素材背景的显示效果。同时，支持将背景效果应用到全部项目素材。
![](https://main.qcloudimg.com/raw/80e2c7aff3d2cf6af1fedeb09ec97580.png)

## 步骤4：添加组件

[](id:txt)

### 添加文本

普通剪辑数提供了多种文字素材，若您需使用自定义的文字素材，可前往 [创作工具>素材制作](https://cloud.tencent.com/document/product/1489/62219?!editLang=zh&!preview) 或 [创作资源>文字](https://cloud.tencent.com/document/product/1489/62249?!editLang=zh&!preview#part2)  中创建文字素材模板，导出后即可到剪辑器里面使用。

1. 单击左侧的 **文本** 进入资源库，选择文字类型为 **花字**、**气泡文字** 或 **自定义文字**。
![](https://main.qcloudimg.com/raw/6d22f3ef68d3c76f657ce7bc1819e2f3.png)
1. 将目标文字模板拖动到轨道中，调整出现时长。        
2. 单击轨道上的文本模板，打开文本基本设置界面，可调整文本内容及其样式。
3. 在右侧的预览框中拖动文本，可调整其在视频界面的显示位置。
![](https://main.qcloudimg.com/raw/714cb3863a8aac0f29b940b4c7853ea8.png)

### 添加音乐

云剪辑提供了多个不同的背景音乐和音效模板。

1. 单击左侧的 **音乐** 进入资源库，选择音乐类型为 **背景音乐** 或 **音效**。
![](https://main.qcloudimg.com/raw/53afe8523a4734f276c709eaed615a0d.png)
2. 将目标文字模板拖动到轨道中，调整音乐时长。
3. 单击轨道上的音乐，打开音频编辑界面，可调整音量大小、播放速度、淡入/淡出时间。
![](https://main.qcloudimg.com/raw/2d0f4b1e88d65df43019f0ea03ea0d60.png)

### 添加贴片

云剪辑数十种在线贴片素材供您使用，若您需使用自定义的贴片素材，可前往 [创作工具>素材制作](https://cloud.tencent.com/document/product/1489/62219?!editLang=zh&!preview) 或 [创作资源>贴片](https://cloud.tencent.com/document/product/1489/62249?!editLang=zh&!preview#part2)  中创建贴片素材，导出后即可到剪辑器里面使用。

1. 单击左侧的 **贴片** 进入资源库，选择贴片类型为 **表情包**、**通用**、**心情**、**爱心**、**涂鸦**、**生活**、**马赛克** 或 **自定义贴片**。
![](https://main.qcloudimg.com/raw/2f437754c9cd90a400ea4eff5755e44d.png)
2. 将目标贴片拖动到轨道中，调整贴片显示时长。
3. 单击轨道上的贴片，打开贴片编辑界面，可调整贴片的透明度、位置进行调整，并为贴片增加入场和出场动画。
4. 在右侧的预览框中拖动贴片，可调整其在视频界面的显示位置。
![](https://main.qcloudimg.com/raw/a9776532bbcaf3d5cad20dad3b731aa0.png)

### 添加转场

云剪辑目前提供了上十种转场动画样式，鼠标悬停在动画上即可预览转场动画。
>! 只有在视频分割、拼接处才可成功添加转场动画。

1. 单击左侧的 **转场** 进入资源库，将目标转场动画拖至轨道的**视频分割处**。
2. 单击轨道上的转场动画，即在弹框中修改特效持续时间和切换特效样式。

![](https://main.qcloudimg.com/raw/6fee0e73bcbc7bddd89b509fbf0d02fb.png)

2. 将目标特效拖动到轨道中，调整特效显示时长即可在预览框中查看效果。

## 步骤5：轨道剪辑
视频素材整理完成后，可在轨道中完成位置切割、素材任意裁剪、删除、锁定、撤销、重做及调整刻度表等操作，让视频画面更加流畅，完成视频剪辑。

### 切割
可以将视频按照指针所在的地方进行分割，将一段视频或图片素材切割为两段素材。
![](https://main.qcloudimg.com/raw/fde5090fac44889f177230c23fc6a62e.png)

### 裁剪
支持视频与图片按照自由比例裁剪与固定比例裁剪，裁剪完成后的比例将直接自适应当前视频尺寸。
![](https://main.qcloudimg.com/raw/0671d03a2a33b4f48b06278ed184ab13.png)

### 锁定
分为**素材锁定功能**（锁定单个素材）与**轨道锁定功能**（锁定整条轨道），被锁定的素材无论是在轨道模块还是在素材模块都无法进行任何的操作。
![](https://main.qcloudimg.com/raw/703236767e355969acb52aa3432ed7da.png)

## 步骤6：预览成片
视频剪辑完成后，可在右上角的预览区中查看视频处理后的效果，支持暂停，切至片头或片尾以及快进或快退（每次5s）的操作。
![](https://main.qcloudimg.com/raw/338b95fe8f7702967e72199826088f5c.png)

## 步骤7：导出视频
1. 视频剪辑成片预览没问题后，单击预览框右下角的 **导出视频**，进入视频导出与发布页。
2. 选择导出视频的分辨率、视频封面及所需发布的平台，单击 **导出视频**。
![](https://main.qcloudimg.com/raw/b42d024ac274694a449a2cc89a158294.png)
3. 视频导出成功后，可前往 **视频剪辑>我的成片** 进行查看。
