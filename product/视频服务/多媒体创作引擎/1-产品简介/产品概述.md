多媒体创作引擎（Cloud Media Engine，CME）是一站式的音视频内容创作平台，为内容创作者提供在线剪辑、云端导播和转推、云媒资存储和管理、AI音视频处理等能力。多媒体创作引擎在产品架构上分为管理端与用户端两端，[管理端]() 提供资源管理、域名管理、用量统计等服务，通常由企业管理人员使用；用户端提供具体的剪辑、直播、媒资等服务，通常由企业具体的业务人员使用。

> ? 在管理端创建平台即开通多媒体创作引擎服务，开通指南请参见 [快速入门]()。


## 产品架构

![](https://main.qcloudimg.com/raw/11340c1ee3a83ff8b22980dcd525b25c.png)

> ? 管理端需要使用腾讯云账号登录，后续用于资源购买及账单推送；用户端则使用多媒体创作引擎的自有登录体系，目前支持手机号、微信、QQ 和微博登录。

多媒体创作引擎的使用均需要先在管理端上创建平台，创建平台后方可使用后续服务。其中，平台有标准型和开发型两种类型，以适配用户不同的应用场景：

- 标准型平台：提供标准 SaaS 服务，即开即用，同时提供 API 调用方式。创建成功后可直接分享用户端域名，企业成员直接访问用户端域名，即可享受多媒体创作引擎的全部服务。
- 开发型平台：提供标准 PaaS 服务，不提供用户端页面，仅提供 API 调用的使用方式。创建成功后可获取平台 ID，用于后续的 API 使用。

## 产品定价

多媒体创作引擎目前支持会员（预付费）、资源包（预付费）和按量日结（后付费）三种计费方式。使用多媒体创作引擎服务会产生存储、导出或者流费用，相关计费文档参见 [计费概述]()。
