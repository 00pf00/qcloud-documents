## 腾讯会议 SaaS 服务退款说明
>?
- 腾讯会议 SaaS 服务属于消耗性服务产品，不满足5天内无理由全额退款，请仔细阅读以下说明并发起工单退款流程。
- 如您在退款时遇到问题，可 [ 提交工单](https://console.cloud.tencent.com/workorder/category) 处理。
### 退款说明
1. 单价和折扣按照最终生成的订单说明为准。
2. 参与活动购买的产品，如若退款规则与活动规则冲突，以活动规则为准，活动规则中若说明不支持退款，则无法申请退款。
3. 您同意，如出现异常/恶意退款情况，腾讯云有权否决退款申请。您认可腾讯云有权根据一般的、合理的认识对异常/恶意退款进行判断和识别。

### 退款金额
1. **退款金额计算：退款金额 = 已生效订单金额 + 未生效订单金额 - 已消费金额。**
2. **已生效订单金额**：对于您退款时间处于订单中服务生效周期内的订单，您实际支付的金额。
3. **未生效订单金额**：对于您退款时间处于订单中服务生效开始日期前的订单，您实际支付的金额。
4. **已消费金额**：您发起退费的当天，已满整月的，按已使用整月的包年包月价格及对应的官网折扣进行扣除；不满整月的，按已使用的天数折算当月总天数的比例乘以包月价格进行退款。
5. 若按包年价格购买的订单，其退费时包月价格按照包年价除以12计算。
6. 您理解并同意，腾讯云退款金额中不包含您在购买时使用折扣或代金券抵扣掉的现金额度。

### 发起工单退款流程
1. **发起工单退款前，请先确定是否满足退款说明的条件，并确保基于腾讯会议 SaaS 服务产生的一切数据、资料已完成备份、迁移。**若您未进行备份、迁移，在退款后产生的任何损失由您自行承担，与腾讯云无关。
2. 登录腾讯云官网，进入 [提交工单](https://console.cloud.tencent.com/workorder/category) 页面，找到需要退款的产品分类，填写退还原因和问题描述后提交工单。
3. 您提交退款工单后，工单将进入审核阶段，腾讯云客服人员将会在2个工作日内处理您提交的申请。
4. 您可在 [订单管理](https://console.cloud.tencent.com/deal) 页面查看退货订单，订单状态为“已退款”，可在 [费用中心](https://console.cloud.tencent.com/account) 页面查看款项。若审核不通过，您可在工单里查看审核结果。

### 参考案例
**1. 背景：**
- 用户 A 在2020年09月10日新购为期1年腾讯会议企业版 SaaS 服务，而后又续购为期1年的腾讯会议企业版 SaaS 服务。
- 新购1年腾讯会议企业版 SaaS 服务及续费1年价格均为7213元。
- 用户新购1年腾讯会议企业版 SaaS 服务的订单享受官网100元代金券折扣。

**2. 场景：**
- 用户 A 在2020年12月11日申请订单退款，退款订单包含新购订单和续费订单。
- 购买时使用的100元代金券不支持退还。
- **实际支付金额** = 订单原价金额 - 代金券抵扣金额 = 7213 * 2 - 100 = 14326元。
- **资源已消费金额** = 已使用服务费用 = (7213 - 100) / 12(12 - 3) + (7213 - 100) / 12 / 31(11 - 10) ≈ 5353.87元。
- **其中**：(7213 - 1000) / 12(12 - 3)为减去代金券后您实际的支付的服务费用折算成3个月（9月至12月）的已消费金额；
 (7213 - 1000) / 12 / 31(11 - 10)为减去代金券后您实际支付的服务费用折算成当月（12月）1天（10号至11号）的已消费金额。
- **应退款总金额** = 实际支付金额 - 资源已消费金额 = 14326 - 5353.87 = 8972.13元。
