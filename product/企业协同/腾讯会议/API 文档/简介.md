腾讯会议（Tencent Meeting，TM）Rest API 是为参与腾讯会议生态系统建设的合作方开发者接入并访问腾讯会议资源提供的一组工具，是访问腾讯会议 SaaS 服务的入口。合作伙伴可以通过腾讯会议 API 进行二次开发，来使用腾讯会议 SaaS 的音视频会议资源，例如创建一个会议，修改会议，查询会议信息等等。
## 申请安全凭证
点击 [腾讯API接入申请](https://cloud.tencent.com/apply/p/xf1vhv5lq79)，填写企业资料，我们将在三个工作日内完成审核，并通过邮件发送接入API的权限密钥和客户端证书。

## 腾讯会议 REST API网关
版本 ：1.0.3
Host : api.meeting.qq.com
Protocols : https
Accepts : application/json
Responds With : application/json

所有 API 都是基于 Restful 规范通过 HTTPS 协议访问特定的 API 网关来完成，例如访问 URL:
```
https://api.meeting.qq.com/v1
```


完整的资源访问请求 URL 格式取决于要访问的资源，例如，查询一个会议的信息，需要通过 GET 方法：
```
GET https://api.wemeet.tencent.com/v1/meetings/{meetingId}
```


在 API 的参考指引页面，可以查阅 API 的具体说明和请求/回复的参数说明。
