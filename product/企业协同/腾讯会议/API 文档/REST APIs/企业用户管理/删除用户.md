## 接口描述

描述：删除企业用户。
接口请求域名：`https://api.meeting.qq.com/v1/users/{userid}`







## 输入参数

| 参数名称 | 必选 | 参数类型 | 参数描述 |
| -------- | ---- | -------- | -------- |
| userid   | 是   | String   | 用户 ID   |



## 输出参数
无输出参数，成功返回空消息体，失败返回错误码和错误信息。



## 示例

#### 输入示例
```
DELETE https://api.meeting.qq.com/v1/users/9527
```


#### 输出示例
删除用户成功，返回 Body 为空。
