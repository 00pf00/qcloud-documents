## 接口描述
描述：会议拥有者获取某指定会议的参与人员列表。只有会议的拥有者即创建者可以查询参会成员的列表。其他用户的调用会被拒绝。如果会议还未开始，调用此接口查询会返回空列表。
调用方式：GET
接口请求域名：`https://api.meeting.qq.com/v1/meetings/{meetingId}/participants?userid={userId}`
