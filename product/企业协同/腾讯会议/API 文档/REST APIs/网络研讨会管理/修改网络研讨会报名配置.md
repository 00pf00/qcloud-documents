
## 基本信息
**描述：**修改网络研讨会的报名配置和报名问题，仅会议创建者可修改，且需要会议已开启报名。企业 secret 鉴权用户可修改任何该企业该用户创建的有效会议，OAuth2.0 鉴权用户只能修改通过 OAuth2.0 鉴权创建的有效会议。用户必须是注册用户，请求头部 X-TC-Registered 字段必须传入为1。
**调用方式：** PUT
**接口请求域名：**
```Plaintext
/v1/webinars/{meeting_id}/enroll/config
```



## **输入参数**

| **名称**            | **类型**  | **是否必须** | **默认值** | **备注**                                             | **其他信息**      |
| ------------------- | --------- | ------------ | ---------- | ---------------------------------------------------- | ----------------- |
| approve_type        | integer   | 非必须       |     -       |   审批类型：<br>1：自动审批<br>2：手动审批，默认自动审批      |       -            |
| instanceid          | integer   | 非必须       |        -    |   设备类型。                                               |            -       |
| is_collect_question | integer   | 非必须       |    -        | 是否收集问题：<br>1：不收集，默认值为不收集<br>2：收集       |        -           |
| question_list       | object [] | 非必须       |        -    |   报名问题列表，按传入的顺序排序，仅开启收集问题时有效。 | item 类型：object  |
| userid              | string    | 非必须       |     -       | 用户 ID。                                              |                   |

**报名问题对象**

| **名称**       | **类型**  | **是否必须** | **默认值** | **备注**                                                     | **其他信息**      |
| -------------- | --------- | ------------ |---------- | ------------------------------------------------------------ | ----------------- |
| is_required    | integer   | 非必须       |     -       | 是否必填：<br>1：否<br>2：是                                         |           -        |
| option_list    | object []| 非必须      |     -       | 问题选项列表，按传入的顺序排序，仅单选/多选时有效，最多8个选项，每个选项限40个汉字。 | item 类型：object |
| question_title | string    | 非必须       |      -       |       问题标题，限制40个汉字。                                       |     -              |
| question_type  | integer   | 非必须       |      -       |   问题类型：<br>1：单选<br>2：多选<br>3：简答                             |    -               |

**问题选项对象**

| **名称** | **类型** | **是否必须** | **默认值** | **备注** | **其他信息** |
| -------- | -------- | ------------ | ---------- | -------- | ------------ |
| content  | string   | 非必须       |  -          |      -    |         -     |


## **输出参数**

| **名称**       | **类型** | **备注**                              | 
| -------------- | -------- |  ------------------------------------- | 
| meeting_id     | string   |   网络研讨会唯一标识。                    |   
| question_count | integer  |    报名问题数量，不收集问题时该字段返回0。 |
