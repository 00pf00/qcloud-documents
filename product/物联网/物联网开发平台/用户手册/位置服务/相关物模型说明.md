

## 物模型说明：地理位置（属性）

- **属性标识符**：GPS_Ext 
- **类型**：字符串
- **参数说明**

| 字段名称       | 描述                                                     | 是否必选 |
| -------------- | -------------------------------------------------------- | -------- |
| lat            | GPS纬度，单位度，WGS84格式，6位小数点                    | 是       |
| lon            | GPS经度，单位度，WGS84格式，6位小数点                    | 是       |
| altitude       | 海拔，数值型；数值范围：-5000-99999  初始值：0  单位：m  | 否       |
| gps_speed      | GPS速度，整形，数值范围：0-1000 初始值：0  单位：km/h    | 否       |
| direction      | 方向角，数值范围：0-360初始值：0单位：度                 | 否       |
| location_state | 定位状态，整形  0 - 无效 1 - 有效                        | 否       |
| satellites     | 卫星数，整形                                             | 否       |
| gps_time       | GPS时间  时间型 ； 时间截精度到秒，从卫星上采集的时间    | 否       |
| collect_time   | 采集时间  时间型 ； 时间截精度到秒，备采集到车信息的时间 | 否       |

- **示例**：
```
{  
	"clientToken": "***",  
	"method": "report",
	"params": {"GPS_Ext": "lat:22.****97;lon:113.****61"  } 
}
```

>?
>- 若您需要自定义设置以上参数时，经度、纬度必须定义，其他参数可以根据实际情况进行添加或删除。
>- 如果以上四个参数您都定义了，则上报数据里时就必须包含这四个参数的数据。



## 蜂窝定位物模型说明

- 目标：对于2G,4G等设备，可上报单个基站信息到云端定位

- 属性标识符：LBS_BS

- 类型：字符串（String）

- 参数说明：


  | 字段名称     | 描述                                    | 是否必选 |
  | ------------ | --------------------------------------- | -------- |
  | mcc          | 基站国家码（460）                       | 是       |
  | mnc          | 基站网络码（00）                        | 是       |
  | lac          | 基站小区号（5位十进制数）               | 是       |
  | cid          | 基站ID（5位十进制数）                   | 是       |
  | rss          | 基站信号强度，单位dbm                   | 是       |
  | networkType  | 1：GSM，2：CDMA,3:WCDMA,4:TD_CDMA,5:LTE | 是       |
  | collect_time | 设备采集到基站信息的时间                | 否       |

- 示例：
```
{  
	"clientToken": "***",  
	"method": "report",
	"params": {"LBS_BS":"mcc:460;mnc:13824;lac:3;cid:33:rss:-85;networktype:1"} 
}
```
