## 1. 接口描述

本接口 (GetConnectorStatus) 用于查询数据同步任务状态。

接口请求域名：<font style="color:red">ckafka.api.qcloud.com</font>

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数，其它参数见[公共请求参数](https://cloud.tencent.com/doc/api/431/5883)页面。

| 参数名称 | 必选 | 类型 | 描述 |
| --- | --- | --- | --- |
| connectorId | 是 | string | connectorId，根据该参数指定connector过滤。|


## 3. 输出参数

| 参数名称 | 类型 | 描述 |
| --- | --- | --- |
| status| string| connector状态 ,状态说明：UNASSIGNED:任务还未分配; RUNNING:代表connector正在运行; PAUSED:代表connector已经暂停; FAILED:代表任务失败; DESTROYED:代表任务销毁。  |
| type| string | connector 类型，source ， sink 两种类型。 |

## 4. 示例

输入：

```
 https://domain/v2/index.php?Action=GetConnectorStatus&<公共请求参数>
```

输出：

```
{
    "code":0,
    "codeDesc":"Success",
    "message":"ok",
    "data":{
        "state":"RUNNING",
        "type":"source"
    }
}

```
