## 请求
| 属性        | 类型     | 约束   | 说明                   |
| --------- | ------ | ---- | -------------------- |
| appId     | String | 必选   | 应用 ID                |
| callId    | String | 必选   | 呼叫 ID                |
| caller    | String | 必选   | 主叫号码（手机、固话）          |
| called    | String | 必选   | 被叫号码（400、固话）         |
| event     | String | 必选   | 通知事件类型（incomingcall） |
| timeStamp | String | 可选   | 时间戳            |
